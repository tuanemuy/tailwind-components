# Stacked Footer Layouts

ランディングページやシンプルなWebサイト向けのフッターレイアウトコンポーネント集。

## MiniStackedFooterWithLinks

説明: ミニマルな1行フッター。コピーライト表示、リンクリスト（Terms、Privacy、Your Privacy Choices）、ダークモード切り替えボタンを中央揃えで配置。ボーダー付きの背景とコンパクトな高さが特徴。

主なユースケース: ランディングページやシンプルなWebサイトのフッター。フッターに多くの情報を載せず、必要最低限のリンクとダークモード切り替えだけを表示したい場合。

## SimpleStackedFooterWithLinksAndModal

説明: シンプルなレイアウトのフッター。左側にコピーライト、右側にFAQ・Licenseリンクとキーボードショートカットモーダル呼び出しボタンを配置。ページ下部に固定配置され、サイドバーなしのレイアウトに最適化されている。

主なユースケース: SaaSアプリケーションやツール系Webサービスのフッター。キーボードショートカット一覧をモーダルで表示したい場合。シンプルながらも機能的なフッターが必要な場合。
