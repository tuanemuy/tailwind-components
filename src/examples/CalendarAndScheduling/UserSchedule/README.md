# User Schedule

Navigate user schedules with ease using a weekly calendar, organized list groups, and revealing admin modals with even more content.

## CalendarWithListGroupAndModals

説明
週間カレンダーとイベントリストを組み合わせたスケジュール管理コンポーネント。ヘッダーに日付表示とイベント追加ボタン、横並びの週日ボタン（選択日はハイライト）、時間帯別にグループ化されたイベントリストを配置。各イベントはクリックで詳細確認可能。イベント追加・編集用のモーダルを含む。ダークモード対応。

主なユースケース
ダッシュボードのスケジュールウィジェット、1日の予定確認画面、タスク管理アプリのイベント表示、サイドパネル型のスケジュール管理

## VideoCallUserSchedule

説明
ビデオ会議スケジュールに特化した予定表示コンポーネント。デートピッカー、今日ボタン、前後日移動ボタンを備えたヘッダーと、カルーセルで複数の会議カードを横スクロール表示。各カードにはタイトル、時間、主催者、参加者アバター、ライブインジケーター、Joinボタン、アクションドロップダウン（会議開始、招待コピー、編集、削除）を配置。

主なユースケース
ビデオ会議アプリのスケジュール画面、Zoom/Google Meet連携のダッシュボード、チームミーティング一覧表示、オンライン会議の参加管理UI

## ListingBooking

説明
単一の予約情報をカード形式で表示するコンポーネント。左側に日付（曜日・日・月）を大きく表示し、右側に予約タイトル、詳細説明（折りたたみ可能）、時間、参加者アバターグループを配置。参加者アバターはホバーでツールチップ表示（名前、メール、参加ステータス）。

主なユースケース
予約確認カード、ミーティング詳細表示、カレンダーアプリのイベントプレビュー、スケジュール通知カード
