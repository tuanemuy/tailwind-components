# Calendar Months

Clear and simple monthly calendar layouts with ample space to write in appointments, deadlines, tasks, and reminders.

## SingleCalendarMonth

説明
コンパクトな単月カレンダーコンポーネント。月名表示、前後月へのナビゲーションボタン、曜日ヘッダー、日付選択ボタンを備える。選択中の日付は青色でハイライト表示され、イベントがある日にはドットマーカーが表示される。前月・次月の日付はグレーアウトで区別される。

主なユースケース
日付ピッカーのドロップダウン内カレンダー、フォームの日付入力補助、サイドバーやウィジェットでのミニカレンダー表示、予約システムでの日付選択UI

## YearlyCalendarMonth

説明
12ヶ月分のカレンダーをグリッドレイアウトで一覧表示するコンポーネント。各月は月名ヘッダーと日付ボタンで構成され、イベントドットマーカーも表示可能。スクロール可能なコンテナ内で全月を俯瞰できる。

主なユースケース
年間スケジュールの俯瞰表示、長期プロジェクトの計画立案、年間の予定確認ビュー、ダッシュボードでの年間カレンダーウィジェット
