# Calendar Sidebars

Boost productivity and organization with calendar template featuring an overlay sidebar for seamless information access.

## CalendarCreateEventSidebar

説明
カレンダーイベント新規作成用のオーバーレイサイドバーコンポーネント。画面右側からスライドインする形式で、イベントタイトル、開始・終了時刻の15分刻みセレクト、日付ピッカー、終日イベントトグル、繰り返し設定、参加者追加、場所入力、説明・URL・添付ファイルの追加フィールドを備える。ダークモード対応。

主なユースケース
カレンダーアプリでの新規イベント作成画面、会議やタスクのスケジュール登録フォーム、Google Calendar風のイベント作成UI

## CalendarEditEventSidebar

説明
既存カレンダーイベントを編集するためのオーバーレイサイドバーコンポーネント。CalendarCreateEventSidebarと同様のフィールド構成で、既存データがプリセットされた状態で表示される。「Save changes」ボタンで変更を保存できる。

主なユースケース
カレンダーイベントの詳細変更、会議時間の調整、参加者の追加・削除、既存予定の修正UI
