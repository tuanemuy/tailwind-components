# Calendars

Discover user-friendly calendar UIs designed for both dashboard and admin interfaces.

## DayViewFullCalendar

説明
1日の時間軸を縦にスクロール表示するフルカレンダーレイアウト。左サイドバーにミニカレンダーとカレンダー種類フィルター（アカウント別、タスク種類別）を配置。ヘッダーにはDay/Week/Month/Year切り替えボタン、今日ボタン、前日・翌日移動ボタン、イベント追加ボタンを備える。メインエリアはタイムスロットグリッドで時間ごとにイベントを配置可能。ダークモード対応。

主なユースケース
1日のスケジュール詳細表示、会議室予約システムの日別ビュー、タスク管理アプリの日表示、Googleカレンダー風の日表示UI

## WeekViewFullCalendar

説明
週間の時間軸を7列グリッドで表示するフルカレンダーレイアウト。曜日ヘッダーに日付表示、現在日はハイライト表示。左側に時間軸、All-day（終日）イベント行を備える。サイドバー、ヘッダー構成はDayViewと同様。

主なユースケース
週間スケジュール管理、チームの週間予定確認、会議スケジュールの週間表示、勤怠管理カレンダー

## MonthViewFullCalendar

説明
月全体を曜日×週のグリッドで表示するフルカレンダーレイアウト。各日のセルにイベントを表示可能。サイドバー、ヘッダー構成はDayView/WeekViewと同様で、Day/Week/Month/Year間をシームレスに切り替え可能。

主なユースケース
月間スケジュール一覧、イベントカレンダーの月表示、プロジェクトマイルストーン管理、予約状況の月間俯瞰

## YearViewFullCalendar

説明
12ヶ月分のミニカレンダーをグリッドで一覧表示するフルカレンダーレイアウト。各月のカレンダーは日付ボタンで構成され、イベントドットマーカー表示可能。年間の予定を俯瞰しながら特定の日付に素早くアクセスできる。

主なユースケース
年間スケジュール俯瞰、長期プロジェクトの計画確認、休日・祝日カレンダー表示、年間イベント計画
