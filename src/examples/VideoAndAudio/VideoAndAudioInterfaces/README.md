# Video & Audio Interfaces

Explore modern video and audio interface examples, including call layouts, media controls, and interactive UI elements.

## VideoCallSettings

説明: ビデオ通話前の設定パネル。カメラ、スピーカー、マイクのデバイス選択ドロップダウンと、スピーカー・マイクのテストボタン、音量レベルインジケーターを備えた包括的なデバイス設定UI。

主なユースケース: ビデオ会議アプリの通話前設定画面、オンライン面接システムのデバイス確認画面、ウェビナー参加前のオーディオ・ビデオチェック機能、リモートワークツールのAV設定パネル
