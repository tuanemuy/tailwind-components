# Video Call Layouts

Discover cutting-edge video and audio call layouts, media controls, and interactive UI elements for modern web apps.

## VideoCallLayoutTwoGrid

説明: 2人の参加者を表示するビデオ通話レイアウト。大画面では横並び、小画面では縦並びのレスポンシブグリッド。各参加者に名前ラベルとマイク状態インジケーターを表示。

主なユースケース: 1対1のビデオ通話画面、オンライン面接の面接官と候補者表示、カスタマーサポートのビデオチャット、リモート相談サービスの通話UI

## ParticipantPreviewOverlay

説明: メイン発言者を全画面で大きく表示し、右端に他の参加者のサムネイルをスクロール可能なリストでオーバーレイ表示するレイアウト。発言者フォーカス型のビデオ通話UI。

主なユースケース: プレゼンテーション中のビデオ会議、ウェビナーの発言者ビュー、オンライン授業の講師画面、ライブ配信中のゲスト表示

## VideoCallLayoutFourGrid

説明: 4人の参加者を2x2のグリッドで均等に表示するビデオ通話レイアウト。各参加者に名前ラベルとマイク状態インジケーターを配置。

主なユースケース: 小規模チームミーティング、グループ面接、オンライン勉強会、4人以下のディスカッション

## VideoCallLayoutSixGrid

説明: 6人の参加者を表示するビデオ通話レイアウト。大画面では3x2、小画面では2x3のレスポンシブグリッド。各参加者に名前ラベルとマイク状態インジケーターを配置。

主なユースケース: 中規模チームミーティング、部門会議、オンラインワークショップ、複数人でのブレインストーミング
