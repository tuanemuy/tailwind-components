# Voice Messages

Voice Messages demo showcases a collection of modern, interactive audio components for web applications, featuring CRM and chat interfaces with customizable voice message players and playback controls.

## CRMVoiceMessage

説明: CRMシステム向けのコンパクトな音声メッセージプレーヤー。再生ボタン、波形ビジュアライザー、再生時間をインラインで表示するミニマルなデザイン。ボーダー付きカードスタイル。

主なユースケース: CRMの顧客対応履歴での音声メモ表示、営業管理システムの通話録音プレビュー、カスタマーサポートのボイスメッセージ一覧、業務システムの音声添付ファイル表示

## ChatVoiceMessageNotPlayed

説明: 未再生状態を示すチャット用音声メッセージバブル。青い背景色で強調され、再生ボタン、波形、再生時間に加えて未再生を示すドットインジケーターを表示。

主なユースケース: メッセージアプリの受信音声メッセージ、チャットアプリの未読ボイスメッセージ表示、SNSのDM音声メッセージ、チーム連絡ツールの音声通知

## ChatVoiceMessage

説明: 再生済み状態のチャット用音声メッセージバブル。白背景にシャドウ付きのニュートラルなデザインで、再生ボタン、波形ビジュアライザー、再生時間を表示。

主なユースケース: メッセージアプリの既読音声メッセージ、チャット履歴内の音声メッセージ表示、会話ログの音声コンテンツ、アーカイブされたボイスメッセージ
