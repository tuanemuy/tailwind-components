# Empty States

Browse Empty States UI blocks that guide users when content is missing—ideal for onboarding, search results, errors, and no-data screens.

## Components

### TitleBarAndButtonGroupEmptyState

説明: タイトルバー、カレンダードロップダウン、アクションボタンを備えたページヘッダーと、データがない状態を表示する空状態コンポーネント。バーチャート風のプレースホルダーグラフィック、「No data」メッセージ、説明文、CTAボタン（Link data）を含む。

主なユースケース: ダッシュボードの概要ページでデータがない場合の表示、分析・レポート画面の初期状態、プロジェクト管理ツールでの新規ユーザー向けオンボーディング

### PieChartPlaceholder

説明: ブラウザ統計を表示するカード型のプレースホルダーコンポーネント。大きな数値（0）の表示エリア、Chrome・Firefox・Safariのブラウザ別統計カード（破線ボーダー）、プッシュ通知設定のアラート（トグルスイッチ付き）を含む。データがない状態でも構造化された表示を維持。

主なユースケース: アクセス解析ダッシュボードの初期状態、ブラウザ統計ウィジェットのプレースホルダー、データ収集前のアナリティクス画面
