# Reviews Forms

Explore user-friendly forms designed for collecting and managing reviews, enhancing feedback efficiency and organization.

## Components

### CardsInTabsFeedback

説明: 星評価別にタブでフィルタリング可能なレビューセクション。上部にサマリー（平均評価、レビュー数、認証済みバッジ）とピル型タブナビゲーション（5 star、4 star、3 star、2 and below）を配置。各タブ内には2列グリッドでレビューカードを表示し、カードには投稿者名、投稿日時、星評価、商品画像（オプション）、レビューテキスト、Read moreリンクが含まれる。

主なユースケース: ECサイトの商品レビューセクション、デジタル製品のユーザーレビュー、サービス評価ページ

### IllustrationBadgesStartsAndProgrssBarsFeedback

説明: レビュー統計サマリーを表示するカード型コンポーネント。左側に星のイラスト（SVG）、平均評価スコア（4.24）、レビュー総数、週間増加バッジを配置。右側に各星評価（5つ星から1つ星）の分布をプログレスバーで視覚化し、件数を表示。下部にOverall performanceのステータスバッジ（Average）を含む。

主なユースケース: レビューダッシュボードの統計サマリー、商品詳細ページのレビュー概要、カスタマーフィードバック分析画面

### ClothingReviewSection

説明: 衣類・靴向けの完全なレビューセクション（3カラムレイアウト）。左サイドバーに平均評価（Verified by Shopバッジ付き）、Rating snapshot（星評価別プログレスバー）、星評価フィルター（チェックボックス式）、Customer rating（Quality/Fit/Comfortのスライダー）を配置。右側に複数のレビューカード（星評価、推奨ラベル、タイトル、投稿日時、商品画像、レビューテキスト、投稿者名、Verified customerバッジ、いいね/悪いねボタン、Reportリンク）をリスト表示。ソート機能とView all reviewsボタンも含む。

主なユースケース: アパレルECサイトの商品レビューページ、靴・アクセサリーのレビューセクション、フィット感や快適さが重要な商品のフィードバック表示
