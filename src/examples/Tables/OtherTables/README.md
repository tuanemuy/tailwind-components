# Other Tables

Manage tables with search inputs, review table contents and paginations.

## AnalyticsByRegionTable

説明: 地域別のアナリティクスデータを表示するテーブルコンポーネント。各国の国旗アイコンと国名、訪問数（Visits）、購入金額（Purchases）、変動率（Change）を表示する。変動率は上昇/下降を示すトレンドアイコン付きで視覚的に分かりやすい。

主なユースケース: ダッシュボードの地域別アクセス分析、ECサイトの国別売上レポート、マーケティング分析の地域別パフォーマンス表示、グローバルビジネスの地域別統計

## ExportTable

説明: データエクスポートの履歴と進捗を一覧表示するテーブル。Type（エクスポート形式：CSV等）、Date（実行日）、Status（進行中のパーセンテージまたは完了）、ダウンロードボタンを表示する。レスポンシブ対応でモバイルではラベル付きレイアウト。

主なユースケース: データエクスポート履歴の管理画面、バッチ処理の進捗状況確認、エクスポートファイルのダウンロード管理、データ出力ジョブの一覧表示

## PostsTable

説明: ブログ記事やニュースレター投稿の一覧を表示するテーブル。サムネイル画像付きのタイトル、著者名、投稿日、送信数（Sends）、開封率（Opens）、ステータスバッジ（Published等）を表示する。詳細リンク付きで個別ページへ遷移可能。

主なユースケース: CMS管理画面の記事一覧、ニュースレター配信履歴の確認、コンテンツ管理ダッシュボード、メールマーケティングのパフォーマンス分析

## ReviewsTable

説明: レビューを検索・フィルタリングできるテーブルコンポーネント。検索バーとフィルタードロップダウン（All/Published等のステータス別絞り込み）を備え、レビュー内容を効率的に管理できるUI。フィルター適用数のバッジ表示も含む。

主なユースケース: ECサイトのレビュー管理画面、商品レビューの承認・却下管理、カスタマーフィードバックの一覧表示、レビューのモデレーション機能
