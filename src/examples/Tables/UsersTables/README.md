# Users Tables

Clean and simple admin table for managing users content and status.

## BorderedTable

説明: タブナビゲーション（All/Valid accounts/Fake accounts）と検索バーを備えたユーザー管理テーブル。アカウントの有効性でフィルタリングでき、ボーダーで区切られたカードスタイルのデザイン。

主なユースケース: ユーザーアカウント管理画面、アカウント検証・審査ダッシュボード、不正アカウントの検出・管理、ユーザー一覧の分類表示

## ContactsReports

説明: タブナビゲーション（All/Assigned/Unassigned）でコンタクトを分類するテーブル。Message、Platform、Assignedの列を持ち、各タブには件数が表示される。コンタクトの割り当て状況を管理するUI。

主なユースケース: カスタマーサポートの問い合わせ管理、コンタクト割り当てダッシュボード、サポートチケットの担当者振り分け、マルチプラットフォームの問い合わせ一覧

## ContactsTable

説明: ビュー切り替え（Board/Table/Timeline）とソート機能（Newest first/Oldest first/A to Z）を備えた高機能なコンタクト管理テーブル。複数の表示形式とソートオプションで柔軟なデータ閲覧が可能。

主なユースケース: CRM管理画面、顧客コンタクト管理、営業リード管理、複数ビュー対応のデータ一覧表示

## FiltersAvatersImagesBadgesTable

説明: 検索バーとフィルタードロップダウンを備えたリファラルユーザーテーブル。アバター画像付きのユーザー情報とカラフルなバッジでステータスを表示。フィルター適用数をバッジで表示。

主なユースケース: リファラルプログラム管理、紹介ユーザーの一覧表示、アフィリエイト管理画面、ユーザー紹介状況の追跡

## MembersTable

説明: メンバー一覧をシンプルに表示するテーブル。アバター画像付きのメンバー情報（名前、メール、加入日）、ステータス（Free member等）、メール開封数、イベント、詳細リンクを表示。

主なユースケース: メンバーシップ管理画面、ニュースレター購読者一覧、メール配信のパフォーマンス確認、会員ステータス管理

## SimpleWithBadgeTable

説明: メンバー一覧をシンプルなテーブル形式で表示するコンポーネント。Member（アバター付き）、Role、Email、Last activity、Status（Active/Inactive等のバッジ）の列を持つ基本的なテーブル。

主なユースケース: チームメンバー管理、ユーザー権限の一覧表示、アクティビティステータスの確認、シンプルなユーザー管理画面

## SmallImagesTable

説明: 新規顧客一覧を表示するテーブル。ヘッダーに検索・ソート・フィルターのアイコンボタンと追加ボタンを配置。コンパクトなアバター画像を使用した顧客リスト表示。

主なユースケース: 新規顧客の一覧表示、顧客管理ダッシュボード、最近登録したユーザーの確認、顧客データのクイック操作

## SurveyDataWithUserTable

説明: サーベイ（アンケート）の回答データをユーザー情報と共に表示するテーブル。質問文、回答タイプ、回答数をヘッダーに表示し、User、Response、Timeの列で各回答を一覧表示。

主なユースケース: アンケート結果の分析画面、ユーザーフィードバックの一覧表示、サーベイ回答データの管理、ユーザー調査結果のレポート

## TeamMembersReports

説明: チームメンバー一覧を表示するテーブル。複数選択可能なチームフィルターとステータスフィルター（Online/Busy/Away/Offline）、検索バーを備える。チームと状態で絞り込み可能な高機能UI。

主なユースケース: チーム管理ダッシュボード、チームメンバーのオンライン状況確認、部署・チーム別のメンバー表示、リモートチームの状態管理
