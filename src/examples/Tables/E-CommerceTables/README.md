# E-Commerce Tables

Organize essential e-commerce data with optimized tables for products and discounts, including product comparisons and item sizes.

## CompareShopProducts

説明: 複数の商品を横並びで比較できるテーブルコンポーネント。商品画像・価格・レビュー評価をヘッダーに表示し、在庫状況、サイズ（EU/UK/US）、商品プロパティ（対象グループ、スポーツウェア対応、色、パターン、フィット、丈、ディテール、洗濯方法）、タイプなどの属性を行ごとに比較できる。削除ボタンやローディング状態のプレースホルダーも含む。

主なユースケース: ECサイトの商品比較ページ、ウィッシュリストからの商品比較機能、類似商品の仕様比較、購入検討時の複数商品の詳細比較

## SizeGuide

説明: アパレル商品のサイズガイドを表形式で表示するコンポーネント。XXS〜XLの各サイズに対して、胸囲（Chest）、ウエスト（Waist）、ヒップ（Hip）、身長（Height）の数値範囲を一覧表示する。交互に背景色が変わるストライプデザインで視認性を確保。

主なユースケース: 衣料品ECサイトのサイズガイド表示、商品詳細ページのサイズ選択支援、アパレル商品のフィッティング情報提供、サイズ選びに迷う顧客への参考情報表示
