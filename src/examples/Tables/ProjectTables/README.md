# Project Tables

Transform your tables into a command center! Searchable filters, vibrant badges, avatars and eye-catching progress bars unlock instant insights.

## BorderedTableWithSearchableFilterBar

説明: プロジェクト追加ボタン、検索バー、フィルター機能を備えたプロジェクト管理テーブル。ボーダーで区切られたカードスタイルのUIで、プロジェクトの検索と新規追加を効率的に行える。

主なユースケース: プロジェクト管理ダッシュボード、チームのプロジェクト一覧画面、プロジェクト検索・フィルタリング、新規プロジェクト作成フロー

## BrandingLogosAndProgressBars

説明: プロジェクト名のヘッダーにソートドロップダウン（昇順/降順）を備えたテーブル。各列でソート機能が利用可能で、進捗バーやブランドロゴを表示するレイアウト。ボーダー付きのテーブルデザイン。

主なユースケース: プロジェクト進捗管理、ブランド別プロジェクト一覧、ソート可能なプロジェクトリスト、進捗状況の可視化

## Details

説明: タブナビゲーション（Tasks/Team/Invoices）とカレンダードロップダウンを備えたプロジェクト詳細テーブル。日付を選択してタスクやチームメンバー、請求書を切り替えて表示できる統合的なプロジェクト詳細UI。

主なユースケース: プロジェクト詳細ページ、タスク・チーム・請求の統合管理、日付別プロジェクト進捗確認、プロジェクトダッシュボード

## Incidents

説明: ヘッダーにエクスポートボタンとフィルターボタン、検索バーを備えたテーブル。インシデントやイベントログの一覧表示に適した構成で、データのエクスポートとフィルタリングが可能。

主なユースケース: インシデント管理画面、イベントログの一覧表示、システム監視ダッシュボード、障害・エラーの追跡管理

## Visitors

説明: トップソース（流入元）を表示するアナリティクステーブル。Source、Visitors、Bounce Rate、Visit Durationの列を持ち、フィルター追加や比較追加のボタンを備える。ウェブ解析向けのUI。

主なユースケース: ウェブアナリティクスダッシュボード、トラフィック流入元分析、訪問者データの比較、マーケティング効果測定
