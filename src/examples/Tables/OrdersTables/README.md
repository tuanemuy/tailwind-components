# Orders Tables

Click tabs, explore modals, set dates in date pickers, filter with dropdowns - all featuring in Orders Admin Table examples.

## AdvancedWithSearchableFilterBar

説明: タブナビゲーション（All/Archived/Publish/Unpublish）、検索バー、日付範囲選択のドロップダウンカレンダーを備えた高機能な注文管理テーブル。注文のフィルタリング、検索、期間指定による絞り込みが可能な管理画面向けのUIを提供する。

主なユースケース: 管理画面の注文一覧ページ、EC管理ダッシュボードの注文管理、注文ステータス別のフィルタリング機能、期間指定での注文検索

## PaymentTable

説明: 支払い履歴を一覧表示するテーブルコンポーネント。日付（PDFアイコン付き）、支払い方法（銀行口座マスク表示）、ステータス、金額を列として表示する。グリッドレイアウトでレスポンシブ対応し、モバイルではラベル付きで表示される。

主なユースケース: アカウント設定の支払い履歴、サブスクリプション請求履歴の表示、支払い明細の一覧表示、請求書ダウンロード機能

## SimpleWithModals

説明: 請求書一覧をシンプルなテーブル形式で表示するコンポーネント。Invoice（請求書番号）、Billing date（請求日）、Amount（金額）、Plan（プラン）の列を持ち、各行にViewボタン（モーダル表示用）とダウンロードボタンを配置。

主なユースケース: アカウント設定の請求書一覧、サブスクリプション請求履歴ページ、請求書のプレビューとダウンロード機能、プラン別の支払い履歴確認

## SimpleWithIconsAndBadges

説明: 日付、支払い方法、ステータス、金額を表示する支払いテーブル。銀行アイコンやPDFアイコンを活用し、ステータスはバッジ形式で表示する。視覚的に分かりやすいシンプルなデザイン。

主なユースケース: 支払い履歴の一覧表示、トランザクション履歴ページ、ステータス付きの決済履歴確認、請求書管理画面
