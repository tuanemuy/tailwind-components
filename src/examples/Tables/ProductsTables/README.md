# Products Tables

Discover Products Tables featuring admin switches, checkboxes, vibrant badges, quick action dropdowns, and filters.

## SearchableFilterBarWithColorfulBadgesInTable

説明: 検索バー、エクスポートボタン、フィルタードロップダウンを備えた製品/割引管理テーブル。カラフルなバッジでステータスを視覚的に区別し、フィルター適用数をバッジで表示する。複数条件でのフィルタリングに対応。

主なユースケース: EC管理画面の割引クーポン管理、製品カタログの検索・フィルタリング、プロモーション管理ダッシュボード、ステータス別商品一覧

## SimpleTableWithSorting

説明: 上部にクイックアクションカード（Product/Discount/Collection/Get paid）を横スクロールで配置し、その下にソート機能付きテーブルを表示するコンポーネント。新規作成への導線とデータ一覧を統合したUI。

主なユースケース: 製品管理ダッシュボードのメイン画面、ECストア管理のホーム、製品・割引・コレクション・決済の統合管理、管理機能へのクイックアクセス

## TableWithSwitchesInTabs

説明: タブナビゲーション（All/Archived/Publish/Unpublish）、検索バー、インポート/エクスポートドロップダウンを備えた高機能な製品管理テーブル。各行にスイッチコントロールを配置し、ステータスの即時切り替えが可能。

主なユースケース: 製品の公開/非公開管理、在庫商品のステータス一括管理、アーカイブ商品の管理、製品データのインポート・エクスポート機能
