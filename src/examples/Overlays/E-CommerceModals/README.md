# E-Commerce Modals

ECサイト向けの商品表示・購入関連モーダルコンポーネント集。商品詳細、クイックビュー、サイズガイドなどの機能を提供する。

## コンポーネント

### ProductDetail

説明: 商品の詳細情報と購入オプションを表示するモーダル。2カラムレイアウトで左に大きな商品画像、右に商品名・カテゴリ・価格（セール価格対応）・カラー選択（円形のカラーパレット）・サイズ選択を配置。カートに追加ボタンを含む。

主なユースケース: 商品一覧ページからのクイック詳細表示、カートへの追加フロー、商品オプション選択

### QuickProductView

説明: ProductDetailと同様の構成を持つクイックビュー用モーダル。商品画像とオプション選択（カラー・サイズ）を2カラムで表示し、素早く購入判断ができるよう最小限の情報に絞った構成。

主なユースケース: 商品グリッドからのホバー時クイックビュー、ウィッシュリストからの素早い確認、商品比較時の参照

### ViewEntireLook

説明: コーディネート全体を表示するためのモーダル。左側にカルーセル形式の複数画像（前後ナビゲーション付き）、右側にコーディネートに含まれる各商品の情報（画像サムネイル・名前・価格・「Add to bag」ボタン）をリスト表示。

主なユースケース: ファッションECでのスタイリング提案表示、セット商品の一括確認、「この商品を使ったコーディネート」表示

### SizeGuide

説明: 商品のサイズガイドを表示するモーダル。カテゴリ（Men/Women/Boysなど）、サイズタイプ（EU/US/UK）、フィットタイプの3つのセレクトボックスでフィルタリングし、サイズ対応表をテーブル形式で表示。

主なユースケース: 衣料品・靴のサイズ確認、海外サイズ換算表示、フィット感の参考情報提供
