# Drawers (Offcanvas)

画面端からスライドして表示されるドロワー（オフキャンバス）コンポーネント集。詳細情報の表示、アクティビティログ、プロファイル表示などに使用する。

## コンポーネント

### TaskDetails

説明: タスクの詳細情報を表示するドロワー。タスク名、担当者、期限、ステータスなどを表示し、リンクコピー・複製・削除などの操作メニューをドロップダウンで提供。右端からスライドインする形式で、2XL以上の画面ではレイアウトに影響を与える。

主なユースケース: プロジェクト管理アプリでのタスク詳細表示、カンバンボードからのタスク情報確認、タスク編集画面

### Activity

説明: ユーザーやシステムのアクティビティログを時系列で表示するドロワー。ユーザーアバター、アクション内容、関連ファイル・フォルダへのリンク、タイムスタンプを縦型ステップ形式で表示。ファイル添付のプレビューも含む。

主なユースケース: チーム活動の履歴確認、プロジェクトの変更履歴表示、監査ログの閲覧

### UserDetails

説明: ユーザーの詳細プロフィール情報を表示するドロワー。アバター、名前、作成日に加え、ユーザーID、メール、電話番号、住所などの情報を3カラムグリッドのリスト形式で表示。

主なユースケース: CRMでの顧客詳細表示、ユーザー管理画面での情報確認、連絡先の詳細ビュー

### ChatUserDetails

説明: チャットの相手やグループの詳細情報を表示するサイドバー型ドロワー。グループ名、メンバー数、プロフィール表示・ミュートなどのアクションボタンを配置。2XL画面では自動的に表示される。

主なユースケース: チャットアプリでのグループ情報表示、メッセージ相手のプロフィール確認、チャット設定の変更

### FilesInfo

説明: ファイルの詳細情報と操作を提供するドロワー。PDF編集、ダウンロード、共有などのアクションボタンを上部に配置し、ファイルのメタデータ（サイズ、作成日、更新日など）を表示。

主なユースケース: ファイル管理アプリでのファイル詳細表示、ドキュメントのプロパティ確認、ファイル共有・操作画面

### UserProfile

説明: ユーザープロフィールを左側から表示するサイドバー型ドロワー。会社名、所在地、参加日時などの「About」情報をアイコン付きリストで表示。XL画面以上では常時表示のサイドバーとして機能。

主なユースケース: ダッシュボードのプロフィールサイドバー、ユーザー設定ページのナビゲーション、アカウント情報の概要表示
