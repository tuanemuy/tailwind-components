# Overlay Sidebar Filters

画面端からスライドインするフィルターサイドバーコンポーネント集。商品フィルタリング、データ絞り込みなどに使用する。

## コンポーネント

### CheckboxFilterForm

説明: チェックボックスベースのフィルターフォームを右からスライドインで表示するオフキャンバス。商品ベンダー、在庫状況などのフィルター条件を2カラムのチェックボックスグリッドで表示。フッターに「Apply」「Reset」ボタンを配置。

主なユースケース: 管理画面での商品絞り込み、データテーブルのフィルタリング、在庫管理システムの検索条件設定

### ShopOffcanvasFilter

説明: ECサイト向けのフィルターサイドバー。上部に適用済みフィルターをタグ表示し、削除可能。カテゴリ、サイズ、カラー（円形カラースウォッチ）、価格帯などのフィルターセクションをアコーディオンで折りたたみ表示。「Show results」ボタンで結果を表示。

主なユースケース: ECサイトの商品一覧フィルター、ファッションサイトでのサイズ・カラー絞り込み、価格帯での商品検索
