# Feedback Modals

ユーザーへの確認・警告メッセージを表示するモーダルコンポーネント集。削除確認、キャンセル確認などの重要なアクション前の確認に使用する。

## コンポーネント

### CancelMessage

説明: 取り消しできない重要なアクション（カード終了など）の確認モーダル。警告メッセージと説明文を表示し、「Cancel」「Confirm」の2つのボタンで選択を促す。代替手段（カードの凍結など）の説明を含む。

主なユースケース: カードの解約確認、サブスクリプションのキャンセル確認、アカウント削除前の警告

### DeleteMessage

説明: メッセージやデータの削除確認モーダル。「Are you sure?」という質問形式の見出しと具体的な説明文、「Don't show me this again」チェックボックス、「Cancel」「Yes, I'm sure」ボタンを配置。削除ボタンは赤色で危険なアクションであることを視覚的に表現。

主なユースケース: チャットメッセージの削除確認、ファイル削除の確認、データ消去前の最終確認
