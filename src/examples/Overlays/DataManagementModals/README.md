# Data Management Modals

データのインポート・エクスポートや設定管理を行うモーダルコンポーネント集。外部サービス連携、データ項目選択、セキュリティ情報表示などの機能を提供する。

## コンポーネント

### PaymentMethodsWithSwitches

説明: 支払い方法の有効/無効を切り替えるためのモーダル。ATM引き出し、オンライン支払いなどの各支払い方法をリスト表示し、トグルスイッチで個別にON/OFFを制御できる。各項目にはアイコンと説明テキストを配置。

主なユースケース: カード設定での支払い方法の管理、セキュリティ設定での取引制限、アカウント設定での機能トグル

### ImportContacts

説明: 外部サービスから連絡先をインポートするためのモーダル。Gmail、Notionなどの外部アカウントとの連携オプションを、サービスロゴ・説明・「Connect」ボタンと共にリスト表示。各サービスごとに接続ステータスを管理。

主なユースケース: CRMアプリへの連絡先一括インポート、メールサービスとのアドレス帳同期、外部サービス連携の初期設定

### ExportContacts

説明: 連絡先データをエクスポートするためのモーダル。エクスポートする項目（名前、ユーザーID、メール、住所、電話番号など）をチェックボックスで選択可能。2カラムのグリッドレイアウトで項目を整理して表示。

主なユースケース: 顧客リストのCSVエクスポート、連絡先のバックアップ作成、データ移行時の項目選択

### ShowPIN

説明: セキュリティPINを一時的に表示するためのモーダル。4桁のPINコードを大きく中央に表示し、自動消去までのカウントダウンタイマー（00:56など）を併記。セキュリティを考慮した時間制限付き表示。

主なユースケース: カードのPIN確認、セキュリティコードの一時表示、認証情報の安全な閲覧
