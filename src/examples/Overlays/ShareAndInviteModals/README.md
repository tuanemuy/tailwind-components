# Share & Invite Modals

ファイル共有・ユーザー招待機能を提供するモーダルコンポーネント集。チームメンバーの招待、ファイル共有、権限設定などの機能を含む。

## コンポーネント

### ShareFile

説明: ファイルを他のユーザーと共有するためのモーダル。チームや個人の検索入力欄、メッセージ追加用テキストエリア、権限設定（Can view/Can edit/Admin）のセレクト、共有リンクのコピー機能を含む。

主なユースケース: ファイル管理アプリでのファイル共有、ドキュメントの共同編集者追加、チーム内でのリソース共有

### SimpleInviteUsers

説明: シンプルなユーザー招待モーダル。「To」入力欄と「Invite」ボタンを上部に配置し、下部にサジェストされた連絡先をアバター・名前・メールアドレス付きでリスト表示。ワンクリックで招待可能なシンプルな構成。

主なユースケース: チャットグループへのメンバー追加、シンプルな招待フロー、連絡先からの素早い招待

### InviteUsers

説明: 詳細な権限設定付きのユーザー招待モーダル。名前またはメール入力欄と権限セレクト（Can view/Can edit/Admin）を横並びに配置。招待リンクのコピー機能、既存メンバーの権限変更・削除機能を含む。アコーディオンで「Everyone at ○○」などのグループ権限も設定可能。

主なユースケース: プロジェクト管理ツールでのメンバー招待、詳細な権限管理が必要なチーム招待、ワークスペースへのユーザー追加
