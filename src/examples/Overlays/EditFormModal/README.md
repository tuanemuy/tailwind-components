# Edit Form Modals

データの編集・作成を行うフォーム付きモーダルコンポーネント集。ユーザー情報、プロジェクト、イベントなどの編集機能を提供する。

## コンポーネント

### CardLabel

説明: カードのラベル（名前）を編集するためのシンプルなモーダル。テキスト入力フィールド1つと「Save changes」ボタンのみのミニマルな構成。中央揃えのタイトルで質問形式の見出しを表示。

主なユースケース: 決済カードの名称変更、ラベル付けカードのリネーム、シンプルな名前編集

### WorkspaceCreateKanbanProject

説明: ワークスペース内でカンバンプロジェクトを作成・編集するためのモーダル。プロジェクト名、説明、担当者、期限などの入力フィールドを含む大型フォーム。コピーリンク・複製・削除などの操作メニューをドロップダウンで提供。

主なユースケース: カンバンボードでの新規プロジェクト作成、プロジェクト詳細の編集、ワークスペース内のタスク管理

### CalendarEditEvent

説明: カレンダーイベントを編集するためのモーダル。タイトル、日時、繰り返し設定、参加者などの項目をアイコン付きの12カラムグリッドで整理。自動高さ調整のテキストエリアを含む。

主なユースケース: スケジュール管理アプリでのイベント編集、ミーティング情報の更新、繰り返しイベントの設定変更

### DashboardCreateKanbanProject

説明: ダッシュボードからカンバンプロジェクトを作成するためのモーダル。WorkspaceCreateKanbanProjectと同様の構成で、プロジェクト詳細（名前、説明、メンバー、期限）を入力。操作メニュー付きのヘッダーを持つ。

主なユースケース: ダッシュボードからの新規プロジェクト作成、クイックプロジェクトセットアップ、タスクボードの初期設定

### EditUser

説明: ユーザー情報を編集するための包括的なモーダル。アバター画像のアップロード（プレビュー・削除機能付き）、名前、メール、役職などの基本情報フィールドを2カラムのラベル付きグリッドレイアウトで配置。

主なユースケース: 管理画面でのユーザー情報編集、プロフィール設定の更新、チームメンバー情報の管理
