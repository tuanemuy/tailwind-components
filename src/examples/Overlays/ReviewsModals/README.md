# Reviews Modals

レビュー・フィードバック投稿用のモーダルコンポーネント集。商品レビュー、サイトフィードバックなどの入力フォームを提供する。

## コンポーネント

### ProductReview

説明: 商品レビューを作成するためのモーダル。レビュー対象商品の画像・名前を上部に表示し、5段階の星評価（インタラクティブなラジオボタン式）、レビュータイトル、詳細コメントのテキストエリア、画像アップロード機能を含む。

主なユースケース: ECサイトでの購入後レビュー投稿、商品詳細ページからのレビュー作成、カスタマーフィードバック収集

### SiteFeedback

説明: サイト全般へのフィードバックを送信するためのモーダル。「何についてのフィードバックか」を3つのカテゴリ（Website/Shop/Preline Proなど）からラジオボタンで選択し、詳細コメントをテキストエリアで入力。アイコン付きのカテゴリ選択カードで視覚的にわかりやすく表示。

主なユースケース: サイト改善のためのユーザーフィードバック収集、バグ報告、機能リクエストの受付
