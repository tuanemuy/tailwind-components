# Billing & Payments Modals

課金・支払い関連の操作を行うモーダルコンポーネント集。決済情報の追加・管理、プラン選択、チームアカウント作成などの機能を提供する。

## コンポーネント

### GiftPro

説明: 有料プランをギフトとして他のユーザーに送るためのモーダル。名前またはメールアドレスの入力欄、月額/年額の期間切り替えタブ、Startup/Pro/Enterpriseのプランをラジオボタンで選択できる。年額プランには割引バッジを表示。

主なユースケース: 有料サブスクリプションを他のユーザーにプレゼントする機能、法人での一括購入時のギフトコード発行

### AddPayment

説明: 新しい支払い方法を追加するためのモーダル。Card/PayPal/Klarnaの支払い方法選択（Klarnaは「Soon」バッジで無効化）、カード番号・有効期限・CVC・名義の入力フォーム、デフォルト設定チェックボックスを含む。Visa/MasterCard/JCB/Discoverのカードブランドアイコンを表示。

主なユースケース: 新規支払い方法の登録、サブスクリプション開始時のカード登録、購入前の決済情報追加

### AddTeamAccountForm

説明: チームアカウントを新規作成するためのモーダル。チーム名の入力欄とサブスクリプションプラン（Pro/Free）の選択ドロップダウンを含むシンプルなフォーム構成。

主なユースケース: 組織・チーム単位でのアカウント作成、ワークスペースの新規追加、複数人での共同利用開始時

### ManageCardsWithSavedPaymentOptions

説明: 登録済みの支払いカードを一覧表示・管理するためのモーダル。Visa/MasterCardのブランドロゴ、下4桁のカード番号、有効期限（期限切れは赤字で表示）を各カードごとに表示。デフォルト設定ボタンと編集ボタンを配置。

主なユースケース: 登録済みカードの確認・編集・削除、デフォルト支払い方法の変更、有効期限切れカードの更新

### UpgradeForm

説明: 無料プランから有料プラン（PRO）へのアップグレードを促すモーダル。カード決済/PayPal切り替えトグル、クレジットカード情報入力フォームを含む2カラムレイアウト。プランの機能比較や価格情報を右側に表示。

主なユースケース: フリープランからの有料プランへのアップグレード、機能制限解除時の課金誘導、トライアル終了後のプラン選択

### AddCardForm

説明: クレジットカード情報を詳細に入力するためのモーダル。Visa/MasterCard/JCBのカードブランドアイコン表示、カード番号・有効期限・CVCの入力フィールドを備えた標準的なカード登録フォーム。

主なユースケース: 初回カード登録、追加カードの登録、カード情報の更新
