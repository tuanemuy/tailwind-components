# General Modals

汎用的な用途に使用できるモーダルコンポーネント集。ステータス設定、タグ管理、請求書表示、キーボードショートカットなど多様な機能を提供する。

## コンポーネント

### Status

説明: ユーザーのステータスメッセージを設定するモーダル。絵文字アイコン付きのステータス入力欄と、「Focus」「Away」「Do not disturb」などのプリセットステータスをラジオボタンで選択。各プリセットには時間情報（〜何分後に自動解除など）を表示。

主なユースケース: チャット・コラボレーションアプリでのオンラインステータス設定、勤怠管理システムでの在席状況更新

### Tags

説明: タグを検索・選択するためのモーダル。検索入力欄と既存タグのチェックボックスリスト（色付きドット表示）を配置。「Create」ボタンで新規タグ作成モーダルへ遷移可能。

主なユースケース: メール・タスクへのタグ付け、コンテンツのカテゴリ分類、ラベル管理

### CreateTag

説明: 新しいタグを作成するためのモーダル。タグ名入力欄と、10色程度のカラーパレットからラジオボタンで色を選択。「Save」ボタンで保存。

主なユースケース: カスタムタグの新規作成、ラベルのカラー設定、分類カテゴリの追加

### InvoiceWithBackgroundImage

説明: カラフルな背景グラフィック付きの請求書表示モーダル。ヘッダーにSVGで描画された抽象的なカラフルパターンを配置し、請求書の詳細（会社情報、品目、金額など）を表示。

主なユースケース: 請求書のプレビュー表示、支払い完了後のレシート表示、ブランディングを強調した請求書

### ListOfPaymentMethod

説明: 支払い方法を選択するためのモーダル。「How would you like to pay?」という質問見出しと、最近使用した支払い方法のリスト、その他の支払いオプション（Apple Pay、銀行口座など）をアイコン付きで表示。

主なユースケース: チェックアウト時の支払い方法選択、送金アプリでの支払い手段選択、定期支払いの支払い方法変更

### ChooseLocation

説明: 配送先住所を選択するためのモーダル。登録済み住所をラジオボタン形式のカードで一覧表示。各カードには名前、住所、デフォルトバッジを含む。「Manage addresses」リンクで住所管理画面へ遷移。

主なユースケース: ECサイトでの配送先選択、複数住所からの選択、配送オプションの確認

### KeyboardShortcuts

説明: アプリのキーボードショートカット一覧を表示するモーダル。検索入力欄（Cmd+Kで開くことを表示）と、「Formatting」「Editing」などのカテゴリ別にショートカットをリスト表示。各ショートカットはキーコンボをバッジで視覚化。

主なユースケース: テキストエディタ・開発ツールのショートカットヘルプ、アプリのキーバインド確認、パワーユーザー向けリファレンス

### PaymentCardDetails

説明: 登録済みカードの詳細情報を表示するモーダル。カード名義、カード番号（マスク表示）、有効期限などの情報をリスト形式で表示。各項目にはコピーボタンを配置し、クリップボードへのコピーが可能。

主なユースケース: 決済カード情報の確認、カード番号のコピー、支払い情報の詳細表示
