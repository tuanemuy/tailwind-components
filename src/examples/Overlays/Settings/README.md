# Settings Modals

アプリケーション設定やロケーション設定を行うモーダルコンポーネント集。一般設定、地域設定、通知設定などの機能を提供する。

## コンポーネント

### Settings

説明: アプリの包括的な設定を行うためのタブ付きモーダル。左側にアイコン付きのナビゲーション（General/Navigation/Video & Audio/Notifications）、右側に各設定項目のフォームを配置した2カラムレイアウト。モバイルではセレクトボックスでナビゲーションを切り替え。背景ぼかし効果付き。

主なユースケース: ビデオ通話アプリの設定画面、ダッシュボードの環境設定、ユーザー個人設定

### Location

説明: ユーザーの現在地を確認・変更するためのモーダル。国旗アイコンと共に現在設定されている国を表示し、通貨や配送オプションの説明をリスト表示。「Continue」で確定、「Change」で地域設定変更へ遷移。

主なユースケース: ECサイトでの配送地域確認、地域別価格表示の確認、初回訪問時のロケーション設定

### RegionalSettings

説明: 言語と国・通貨を選択するためのモーダル。国旗アイコン付きのセレクトボックスで言語と国を選択可能。言語選択はEnglish (US)/English (UK)/Deutschなど、国選択はUnited Kingdom/United States/Germanyなどをサポート。

主なユースケース: 多言語サイトでの言語切り替え、国際ECサイトでの地域・通貨設定、ローカライゼーション設定
