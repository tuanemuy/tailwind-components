# Custom Date & Time Modals

日付・時間の選択やスケジューリングを行うモーダルコンポーネント集。カレンダーピッカー、時間選択、繰り返し設定などの機能を提供する。

## コンポーネント

### ScheduleTransfer

説明: 予約送金や繰り返し送金をスケジュールするためのモーダル。日付選択用のカレンダードロップダウン（月・年の選択機能付き）と、繰り返し頻度（繰り返しなし/毎日/毎週/毎年）を選択できるセレクトボックスを含む。

主なユースケース: 銀行・決済アプリでの予約送金設定、定期送金のスケジュール作成、支払い予約機能

### CustomSnoozeDateAndTime

説明: スヌーズの日時をカスタム設定するためのモーダル。時間帯（01:00 AM〜12:00 PMなど）をラジオボタンで選択できるグリッドレイアウト。Snoozeモーダルから遷移して使用する詳細設定画面。

主なユースケース: 通知のリマインダー時刻のカスタム設定、メールのスヌーズ機能、タスク管理アプリでの延期時刻指定

### CustomReccurenceEvent

説明: イベントや予定の繰り返しルールをカスタマイズするためのモーダル。繰り返し間隔（数値入力）と頻度（毎日/毎週/毎月/毎年）を組み合わせて詳細な繰り返しパターンを設定できる。

主なユースケース: カレンダーアプリでの繰り返しイベント作成、定期ミーティングの設定、習慣トラッカーの繰り返し設定

### Snooze

説明: 通知やタスクをスヌーズ（一時保留）するためのモーダル。「今日」「明日」「今週」「次週」などのプリセット選択肢をラジオボタンで提供し、具体的な日時（Wed 9:00 AMなど）を併記。「Custom」ボタンから詳細設定画面へ遷移可能。

主なユースケース: メール・通知のスヌーズ機能、タスクの延期設定、リマインダーの再通知時刻選択
