# Features

Tailwind CSS feature sections with images, icons, headings, and descriptions to highlight key offerings.

## ContainerOriginFullWidthSlider

説明: レスポンシブなカルーセル/スライダーコンポーネント。見出しと左右ナビゲーションボタンを上部に配置し、ドラッグ操作可能なカードカルーセルを表示。画面サイズに応じて表示枚数が変化する。

主なユースケース: 製品一覧のスライダー表示、フィーチャー紹介のカルーセル、関連コンテンツのスクロール表示、ギャラリーセクション。

## GridViewCaseStudies

説明: ダークテーマのケーススタディグリッド。各カードにクライアントロゴ、大きな統計数値、タイトル、説明、リンクを配置。ホバー時に下部からグラデーションが出現するエフェクト付き。

主なユースケース: 導入事例一覧、成果を数値で示すケーススタディ、パートナー企業との成功事例紹介。

## HorizontalMiniImageCardGridView

説明: 横並びのミニカードグリッド。各カードは左に円形サムネイル画像、右にタイトル・説明・補足テキストを配置した水平レイアウト。

主なユースケース: サービス・機能一覧、チュートリアルや学習コンテンツの一覧、アプリやツールのリスト表示。

## ListingViewCaseStudies

説明: ダークテーマのケーススタディリスト表示。各アイテムはタイトル・説明・サムネイル画像を横並びで配置し、区切り線で分離された縦リスト形式。

主なユースケース: ケーススタディ一覧ページ、導入事例のアーカイブ、プロジェクト実績リスト。

## MediaCardGridView

説明: メディアカードのグリッドレイアウト。上部にタイトル・説明・補足テキスト、下部に大きなサムネイル画像を配置したカード形式。

主なユースケース: コンテンツ・アプリ一覧、サービスカタログ、学習リソースの一覧表示。

## NavTabsWithScroller

説明: 水平スクロール対応のタブナビゲーション。アイコン付きのピル型タブボタンが横に並び、画面幅に応じてスクロール可能。タブ切り替えでコンテンツを表示。

主なユースケース: 機能紹介セクションのタブ切り替え、ダッシュボードのビュー切り替え、カテゴリーフィルター。

## ProductDetails

説明: 製品詳細情報の2カラムレイアウト。大きな製品画像と説明文、仕様リスト（ラベル+値の2カラム表）を交互に配置。背景色付きセクション。

主なユースケース: 製品詳細ページ、商品紹介セクション、サービス仕様の説明、ストーリー+スペック表示。

## ProjectsFeatures

説明: ダークテーマのプロジェクトカードグリッド。各カードにアイコン、タイトル、説明、リンクを配置。ホバー時にカードが回転しボーダーがアクセントカラーに変化するアニメーション付き。

主なユースケース: プロジェクトポートフォリオ、サービス一覧、機能紹介セクション。

## PublishedPost

説明: レスポンシブな記事/投稿カード。モバイルでは画像が上・テキストが下の縦並び、デスクトップでは左に画像・右にテキストの横並びレイアウト。

主なユースケース: ブログ記事のフィーチャー表示、ニュースのハイライト、おすすめコンテンツカード。

## SourceLink

説明: ウェブサイトへのソースリンクカード。タイトル、説明文（2行制限）、ファビコンアイコン+URL表示を含むコンパクトなリンクブロック。

主なユースケース: 参考文献・引用元の表示、検索結果のリンク表示、外部リソースへの誘導、AIチャットの出典表示。

## SplitGradientGridFeatures

説明: 左に画像プレビュー、右にタイトル+説明を配置した2カラムのフィーチャーカードグリッド。ライト/ダークモード対応の画像切り替え付き。

主なユースケース: 機能紹介セクション、製品機能のビジュアル説明、UIコンポーネントのショーケース。

## TestimonialsAndStatsSection

説明: 左右2カラムの統計+説明セクション。見出し・説明文と、統計数値（上昇アイコン付き）+説明の組み合わせを縦に並べたレイアウト。

主なユースケース: 成果・実績の訴求セクション、導入効果の数値表示、コンバージョン率や改善率のアピール。

## TestimonialsWithAccordion

説明: 左右2カラムで、左側に見出し、右側にアコーディオン形式の機能説明を配置。各アコーディオンは展開時にリスト項目を表示。

主なユースケース: 機能詳細の説明セクション、FAQ形式の機能紹介、製品特徴の展開型説明。
