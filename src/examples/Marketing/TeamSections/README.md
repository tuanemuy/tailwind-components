# Team Sections

Tailwind CSS team sections with avatars, names, titles, and social links to highlight your team.

## TableFormat

説明: テーブル形式のチームメンバー一覧セクション。上部にセクション見出しと「We're hiring」リンク、2カラムグリッドで各メンバーの円形アバター・名前・役職を横並びで表示。名前と役職の間に点線の区切り線を配置したミニマルなデザイン。

主なユースケース: Aboutページのチーム紹介、企業の経営陣・メンバー一覧、スタートアップのチームセクション、採用ページへの導線付きチーム表示。
