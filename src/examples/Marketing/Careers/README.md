# Careers

Browse Careers UI blocks including job listings, filters, and layouts. Explore reusable design components crafted for recruitment and hiring pages.

## CareersListing

説明: ダークテーマの求人一覧ページセクション。部門・勤務地・雇用形態の3つのセレクトボックスによるフィルター機能と、部門ごとにグループ化された求人カードリストを表示。各求人カードにはポジション名・勤務地・雇用形態が表示され、ホバー時に「Apply」リンクが出現するインタラクティブなデザイン。

主なユースケース: 企業の採用ページ、求人一覧セクション、キャリアサイトでの募集ポジション表示、部門別の採用情報ページ。
