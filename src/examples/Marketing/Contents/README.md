# Contents

Tailwind CSS content sections with text, images, and layouts for structured and engaging presentations.

## NewsDetails

説明: ダークテーマのニュース・ブログ記事詳細ページレイアウト。横幅いっぱいのヒーロー画像、大きな見出し、複数段落の本文、そしてタグ（ピル型ラベル）のセクションを含むシンプルな記事コンテンツ構成。

主なユースケース: ニュース記事詳細ページ、ブログ投稿の表示、お知らせ詳細、プレスリリースページ。

## StartupAboutPageContents

説明: 会社概要・Aboutページ向けのコンテンツセクション。中央揃えのヒーローエリア（タイトル・説明・フルワイド画像）と、左右2カラムの見出し+説明文レイアウトを組み合わせた構成。企業のストーリーや理念を伝えるのに適している。

主なユースケース: 会社概要ページ、サービス紹介ページ、企業理念・ミッションの説明セクション、スタートアップのAboutページ。

## WorkDetails

説明: ポートフォリオ・作品詳細ページ向けのコンテンツレイアウト。タイトル・説明文・大きな画像・2カラムのテキストレイアウト・画像グリッド・リスト項目（横線付き）などを組み合わせた縦長のリッチコンテンツ構成。

主なユースケース: ポートフォリオの作品詳細ページ、プロジェクト紹介、ケーススタディの詳細表示、クリエイティブ作品のショーケース。
