# Hero Sections

Create impactful hero sections and banners to captivate visitors with stunning visuals, compelling messages, and clear calls to action.

## AgencyAboutPageHero

説明: ダークテーマのAboutページ向けヒーロー。ドットパターン背景に中央揃えの大きな見出し（アクセントカラー付き）を配置し、周囲に異なるサイズの画像カードが浮かぶ装飾的なレイアウト。

主なユースケース: エージェンシーや企業のAboutページ、ミッションステートメント表示、ブランドストーリーの訴求セクション。

## AgencyVideoHero

説明: ダークテーマのビデオヒーロー。上部にパートナー/連携サービスのロゴバッジ、大きな見出し、説明文、CTAボタン、ビデオ再生ボタンを含む構成。

主なユースケース: エージェンシーやSaaSのランディングページ、製品紹介動画への誘導、パートナーシップをアピールしつつ動画で訴求したい場合。

## SearcInputWithBackgroundImageHero

説明: 背景画像をフルに使った検索フォームヒーロー。中央に見出しと大きな検索入力フィールドを配置したシンプルな構成。

主なユースケース: ヘルプセンター・FAQページ、ナレッジベースの検索入口、サポートページのトップセクション。

## ShopMainPageSlider

説明: ECサイト向けのシンプルなヒーロースライダー。フルサイズ背景画像に中央揃えの見出し・説明・リンクを配置。無限ループ対応。

主なユースケース: ECサイトのメインビジュアル、季節キャンペーンの表示、ファッション・アパレルのトップページ。

## ShopMarketplaceMainPageSlider

説明: マーケットプレイス向けのヒーロースライダー。背景画像+オーバーレイテキスト+CTAボタンのスライドで、ドットインジケーター付き。複数のプロモーションを表示可能。

主なユースケース: オンラインマーケットプレイスのトップページ、複数カテゴリーの製品プロモーション、セール・キャンペーンバナー。

## StartupFeaturePageHero

説明: スタートアップの機能紹介ページ用ヒーロー。ドットパターン背景に中央揃えのテキストを配置し、周囲にUIカード（タスク、アバター、バッジなど）が浮かぶダイナミックなレイアウト。

主なユースケース: 機能紹介ページ、製品ツアーのイントロセクション、UIの特徴を視覚的に訴求したい場合。

## StartupMainPageHero

説明: スタートアップ向けメインヒーロー。左側にキャッチコピー・説明文・CTAボタン群、右側に補足情報を配置した2カラムレイアウト。背景画像対応、ライト/ダークモード切り替え。

主なユースケース: SaaSやスタートアップのランディングページ、メインサービスの訴求、製品トップページ。

## StaticHero

説明: シンプルな静的ヒーローバナー。フルワイド背景画像に中央揃えのテキスト（サブタイトル・メインタイトル）とテキストリンクを配置した最小限の構成。

主なユースケース: ECサイトのセールバナー、シーズンプロモーション、シンプルなキャンペーン告知。
