# Stats

Stats cards with avatars and badges that provide a clear overview of key metrics.

## AgencyAboutPageStats

説明: ダークテーマのエージェンシー向け統計セクション。上部にインパクトのあるキャッチコピー、下部に黄色（アクセントカラー）の大きな数値+ラベルを3カラムで配置。斜めの区切り線で視覚的に分離。

主なユースケース: エージェンシーのAboutページ、実績数値の訴求セクション、パートナー数や顧客満足度の表示。

## RatingWithDivider

説明: 評価・レーティング表示の統計セクション。3カラムで数値+星アイコン（5段階評価）+ラベルを中央揃えで配置。斜めの区切り線で各項目を視覚的に分離。

主なユースケース: 製品・サービスの評価表示、アプリストアのレーティング、顧客満足度スコア、受賞歴の表示。

## StatsWithFontMonoTitle

説明: モノスペースフォントのラベルを持つ統計グリッド。上にラベル（モノスペース）、下に大きな数値を縦に並べた2〜3カラムのシンプルなグリッドレイアウト。

主なユースケース: KPI・主要指標の一覧表示、サービス実績の数値、ダウンロード数やユーザー数の表示、技術的な統計情報。

## TwoColsStatsSectionWithDivider

説明: 2カラム構成の統計セクション。左側に数値+上昇アイコン+ラベル、右側に縦の区切り線と詳細説明を配置。改善率や成果を視覚的に示す。

主なユースケース: 導入効果・改善率の訴求、Before/Afterの成果表示、ROI・パフォーマンス指標の説明付き表示。
