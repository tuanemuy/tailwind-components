# Testimonials

Tailwind CSS testimonial sections with avatars, names, ratings, and quotes to showcase customer feedback.

## CenterAlignedMiniTestimonialWithLogoInDivider

説明: 中央揃えのコンパクトなテスティモニアル。横線の間に企業ロゴを配置した装飾的な区切り、引用文、名前と役職を縦に配置したシンプルなレイアウト。

主なユースケース: ランディングページの信頼性訴求、単一の重要な顧客の声の強調表示、ブランドパートナーからの推薦コメント。

## OnTopOfImagesWithGradientBackgroundTestimonial

説明: 背景画像上にグラデーションオーバーレイを配置したテスティモニアルカード。下部からの青いグラデーションの上に、企業ロゴ、引用文、名前、役職を白文字で表示。

主なユースケース: ビジュアル重視のテスティモニアル、インパクトのある顧客の声の表示、ギャラリースタイルのレビューセクション。

## PersonalTestimonial

説明: パーソナルなレビュー・テスティモニアルページ。見出し、説明文、総合評価（数値+星アイコン）、複数のレビューを縦に並べた詳細なレビュー一覧形式。

主なユースケース: フリーランサーやコンサルタントのレビューページ、サービスの顧客レビュー詳細表示、ポートフォリオサイトの推薦の声。

## SlidingTestimonials

説明: ダークテーマの自動スクロール（マーキー）テスティモニアル。複数のレビューカードが横に流れ続けるアニメーション。各カードにアバター、名前、引用文を配置。左右に半透明グラデーションのフェードエフェクト。

主なユースケース: ランディングページでの社会的証明、多数の顧客レビューをコンパクトに表示、ページに動きを加えたい場合。

## StackedImageTestimonialCard

説明: 画像と重なるテスティモニアルカード。背景に大きな人物画像、その下にオフセットして配置された白いカードに引用文、名前、役職を表示。視覚的にインパクトのあるデザイン。

主なユースケース: フィーチャードテスティモニアル、重要な顧客の声のハイライト、ビジュアル重視のレビュー表示。

## SuccessStories

説明: グラデーション背景のサクセスストーリーセクション。見出し+説明の上部エリアと、2カラムのカードグリッド。各カードにクライアントロゴ、大きな成果数値、説明、詳細ページへのリンクを配置。

主なユースケース: 導入事例の概要表示、クライアントの成功事例紹介、ROIや成果を数値で示すカスタマーストーリーセクション。
