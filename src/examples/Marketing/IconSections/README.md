# Icon Sections

Explore visually engaging icon sections that highlight key features and services, enhancing user experience and navigation.

## CareersIconBlocks

説明: ダークテーマの採用ページ向けアイコンブロック。黄色（アクセントカラー）のアイコン+タイトル+説明を横並びで配置した2カラムグリッドレイアウト。企業の価値観や文化を伝える。

主なユースケース: 採用ページの企業価値観・カルチャー紹介、会社の理念・行動指針の説明セクション。

## ContactLinks

説明: お問い合わせ方法を示すアイコンリンクブロック。3カラムで、アイコン+ラベル+アクションリンク（チャット、電話、メールなど）を配置したコンパクトなレイアウト。

主なユースケース: お問い合わせページの連絡先一覧、フッター近くのサポートリンク、ヘルプセンターの連絡手段表示。

## SimpleCenterAlignedIconSection

説明: 中央揃えのアイコンブロックグリッド。アイコン→タイトル→説明を縦に並べ、テキストをすべて中央揃えで配置した3カラムレイアウト。

主なユースケース: ECサイトの安心ポイント（セキュア決済、送料無料など）、サービスの特徴紹介、メリット訴求セクション。

## SimpleLeftAlignedIconSection

説明: 左揃えのシンプルなアイコンブロックグリッド。アイコン+タイトル+説明を横並びで配置した3カラムレイアウト。コンパクトで情報密度が高い。

主なユースケース: ECサイトの特典バー（セキュア決済、無料配送、返品保証など）、サービス特徴の簡潔な表示。

## TreeGridColsIconsWithLinks

説明: 3カラムのアイコンブロックグリッドで、各ブロックに「Learn more」リンクを含む。アイコン+タイトル+説明+リンクの構成で詳細ページへ誘導可能。

主なユースケース: 機能紹介セクション、サービス一覧（詳細ページへのリンク付き）、製品特徴の概要表示。

## TwoGridColsWithPrimaryColorIcons

説明: 2カラムグリッドのアイコンブロック。プライマリカラー（青）のアイコン+タイトル+説明を横並びで配置。視覚的なアクセントが効いたデザイン。

主なユースケース: サービスの強み・特徴紹介、企業の価値提案、機能のハイライト表示。

## ValuesIconBlocks

説明: ダークテーマの企業価値紹介セクション。上部にセクションタイトルと説明、下部に黄色アイコン+タイトル+説明のカードを3カラムグリッドで配置。

主なユースケース: 企業価値・ミッションの紹介、Aboutページの理念セクション、カンパニーカルチャーの説明。
