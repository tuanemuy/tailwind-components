# Clients & Customers

Tailwind CSS client and customers with logos and grids to showcase trusted brands and partnerships.

## CardStyleLinkCustomers

説明: シンプルなカードスタイルのロゴグリッドレイアウト。固定高さのカード内にクライアント/パートナー企業のロゴSVGを配置し、各カードはクリック可能なリンクとして機能。ホバー時の背景色変化とシャドウエフェクト付き。

主なユースケース: パートナー企業一覧、導入企業ロゴの表示、信頼性アピールのためのクライアントロゴセクション、サービス連携先の紹介。

## CustomersCardWithDottedBackground

説明: ドットパターン背景を持つ装飾的なカードグリッド。各カードにクライアントのロゴと短い説明文を配置し、ホバー時にカードが上方向にスライドするアニメーション付き。詳細ページへのリンク機能を持つ。

主なユースケース: 導入事例・カスタマーストーリーへの導線、主要クライアントの詳細紹介、ケーススタディ一覧ページ、パートナーシップ紹介セクション。

## SlidingCustomers

説明: 自動スクロール（マーキー）アニメーションで横に流れるロゴバー。左右両端に半透明グラデーションのフェードエフェクトを持ち、無限スクロールの視覚効果を実現。シンプルなロゴのみの表示。

主なユースケース: ヒーローセクション下部のクライアントロゴ表示、ランディングページでの社会的証明、多数のパートナー企業をコンパクトに表示したい場合、ページに動きを加えたいとき。
