# Call To Actions (CTA) Sections

Tailwind CSS call-to-action sections with headings, text, and buttons to boost conversions.

## StartupCTA

説明: グラデーション背景を持つシンプルなCTAセクション。中央揃えの見出し・サブテキストと、プライマリボタン（塗りつぶし）・セカンダリボタン（テキストリンク+矢印アイコン）の2つのアクションボタンを配置したレイアウト。

主なユースケース: ランディングページやサービス紹介ページでの無料トライアル訴求、デモ申し込みへの誘導、ページ下部でのコンバージョン促進セクション。
