# Sidebar Search

Explore quick actions, user lists, and contacts with Tailwind CSS Sidebar Search. A user-friendly sidebar for efficient navigation using keyboard or mouse.

## ChatUserDetails

説明: サイドバー型の検索コンポーネント。検索ボックスにオートコンプリート機能を備え、Quick Action（Photos、Links、Documents）とContacts（連絡先一覧）のグループで構成。連絡先はアバター画像またはイニシャルで表示。検索結果が無い場合のEmpty State表示にも対応。

主なユースケース: チャットアプリのユーザー検索、メッセージアプリの連絡先検索、チームメンバー検索、メディアファイルへのクイックアクセスなど、サイドバーから連絡先やリソースを素早く検索したい場合に使用。
