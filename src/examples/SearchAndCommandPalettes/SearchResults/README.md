# Search Results

The Search Results displays relevant information and links, organized for quick access and easy navigation, helping users find what they need efficiently.

## EventsSearchResult

説明: カレンダーイベント形式の検索結果表示。各イベントにはカラーバー（カテゴリ識別用）、イベント名、開催時間、参加者アバターグループ、ビデオ会議アイコン（Google Meet等）を表示。

主なユースケース: カレンダーアプリのイベント検索、スケジュール管理システムの予定検索、会議検索機能など、日時と参加者情報を含むイベントを検索・表示したい場合に使用。

## IconsWithSubcontentsSearchResult

説明: アイコンとサブコンテンツ（カテゴリ情報）を持つ検索結果リスト。Recent（最近の操作）、Categories（カテゴリ一覧）、Resources（リソース）などのセクションに分類。キーボードショートカット表示付き。

主なユースケース: 管理画面のクイックアクション検索、ストア管理ダッシュボードの操作検索など、アクションとその所属カテゴリを明示したい場合に使用。

## BrandingIconsAndAvatarsSearchResult

説明: 外部サービスのブランドアイコン（Gmail、Slack、Notion）とユーザーアバターを組み合わせた検索結果。Recent（最近のアクション）、Users（ユーザー一覧）、Files（ファイル一覧）などのセクション構成。

主なユースケース: 複数の外部サービスと連携するアプリ、統合ワークスペース、ファイル・ユーザー・連携サービスを横断検索したい場合に使用。

## E-CommerceSearchResult

説明: ECサイト向けのタブ切り替え式検索結果ページ。All、Settings、Articles、Ordersなどのタブで検索結果をフィルタリング。商品にはサムネイル画像、在庫数、バリアント数、価格を表示。ページネーション付き。

主なユースケース: ECサイト管理画面の商品・注文・設定検索、オンラインストアの統合検索機能など、複数種類のコンテンツを一括検索・フィルタリングしたい場合に使用。

## ListingSearchResultWithHighlight

説明: 検索キーワードをハイライト表示するリスト形式の検索結果。マッチした部分が黄色背景で強調表示される。結果件数をヘッダーに表示。

主なユースケース: ドキュメント検索、ヘルプ記事検索、FAQ検索など、検索キーワードがどこにマッチしたかをユーザーに視覚的に示したい場合に使用。

## SimpleListingSearchResult

説明: アイコンとパンくずリスト付きのシンプルな検索結果リスト。各結果にはタイトル、カテゴリ階層（パンくずリスト）、説明文を表示。ホバー時に背景がハイライト。

主なユースケース: ブログ記事検索、ヘルプセンター検索、ナレッジベース検索など、コンテンツの階層構造とともに検索結果を表示したい場合に使用。

## GrayBackgroundOnHoverSearchResult

説明: ホバー時にグレー背景が表示されるシンプルなリスト形式の検索結果。各項目はタイトルのみで、右側に矢印アイコン付き。

主なユースケース: FAQページの検索、ヘルプトピック一覧、シンプルな質問リストの表示など、簡潔な項目リストを表示したい場合に使用。

## SearchListingResult

説明: カテゴリ別にグループ化された検索結果表示。検索サジェスト、Tags（タグ一覧）、Courses（コース一覧）などのセクションで構成。コース項目にはサムネイル画像付き。「more results」リンクで追加結果を表示。

主なユースケース: 教育プラットフォームのコース検索、ドキュメントサイトの検索、タグとコンテンツを組み合わせた検索機能など、複数種類のコンテンツをグループ化して表示したい場合に使用。

## ListingSearchResult

説明: フルページ型のタブ切り替え式検索結果ページ。All、Posts、Projects、Reviewsなどのタブで検索結果をフィルタリング。各結果にはアイコン、パンくずリスト、タイトル、説明文、更新日時、タグ、アバターグループを表示。ローディングインジケーター付き。

主なユースケース: プロジェクト管理ツールの検索、社内ポータルの統合検索、コミュニティサイトの投稿検索など、多様なコンテンツタイプを横断検索する機能に使用。
