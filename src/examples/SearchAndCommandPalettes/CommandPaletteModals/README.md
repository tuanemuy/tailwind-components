# Command Palette Modals

Examples of commands with Tailwind CSS Command Palettes. A user-friendly, vertical menu for efficient search and navigation using keyboard or mouse.

## SimpleListingWithHighlightModal

説明: 検索キーワードをハイライト表示するシンプルなリスト形式のコマンドパレットモーダル。検索結果のタイトルと説明文の中で、マッチしたキーワードが黄色で強調表示される。フッターにはキーボードショートカットのヒントを表示。

主なユースケース: ドキュメント検索、ヘルプ記事検索、FAQ検索など、検索キーワードがどこにマッチしたかをユーザーに視覚的に示したい場合に使用。

## ListingModal

説明: アイコン付きのカード形式で検索結果を表示するコマンドパレットモーダル。各結果にはアイコン、タイトル、パンくずリスト（カテゴリ階層）、説明文を表示。「More results」リンクで追加結果への導線を提供。

主なユースケース: ブログ記事検索、ヘルプセンターの検索、ナレッジベース検索など、コンテンツの種類と階層構造を示しながら検索結果を表示したい場合に使用。

## SearchWithBrandingIconsModal

説明: 外部サービス連携機能を備えた多機能コマンドパレット。Gmail、Slack、Notionなどのブランドアイコンとキーボードショートカットを表示。Topics（タグ）、Recent（最近のアクション）、Users（ユーザー一覧）、Events（イベント）、Files（ファイル）など複数のセクションで構成。

主なユースケース: 複数の外部サービスと連携するダッシュボード、プロジェクト管理ツール、生産性アプリなど、様々なアクションやリソースへの統合的なアクセスを提供したい場合に使用。

## ShopSearchModal

説明: ECサイト向けの商品検索モーダル。検索サジェストをカテゴリ別に結果件数とともに表示し、トレンド商品のカルーセルスライダーを含む。商品カードには画像、商品名、カテゴリ、価格（セール価格対応）を表示。

主なユースケース: ECサイトの商品検索、オンラインショップの検索機能など、商品をビジュアルに検索・ブラウズさせたい場合に使用。

## GroupedModal

説明: フィルタータブとカテゴリ別グループ化機能を備えた検索モーダル。ヘッダーに「All」「Tags」「Article」などのフィルターボタンを配置。検索結果をTags、Coursesなどのカテゴリごとにグループ化して表示。コース項目にはサムネイル画像付き。

主なユースケース: 教育プラットフォームのコース検索、ドキュメントサイトの検索、ブログの記事検索など、コンテンツを種類別にフィルタリング・グループ化して表示したい場合に使用。

## SearchWithIconsModal

説明: アイコン付きのカテゴリ別検索モーダル。Topics（タグ）、Recent（最近の操作）、Categories（カテゴリ）、Resources（リソース）などのセクションで構成。各アクションにキーボードショートカットを表示し、キーボード操作に対応。

主なユースケース: 管理画面のクイックアクション、ストア管理ダッシュボードの検索、SaaSアプリケーションのコマンドパレットなど、様々な操作やリソースへの素早いアクセスを提供したい場合に使用。
