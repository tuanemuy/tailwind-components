# Links

Access essential contact information and resources, including phone numbers, email addresses, and locations for easy reference.

## IconWithGrayBackgroundOnHoverLink

説明: グレー背景のアイコン付き戻るリンク。ホバー時に背景色が変化する丸型デザイン。

主なユースケース: 前のページへ戻るリンク、パンくずナビの戻る操作、モーダル内の戻るリンク。

## SimpleContactInfoLink

説明: お問い合わせ情報セクション。営業時間の説明とアンダーライン付きの問い合わせリンク。

主なユースケース: フッターのお問い合わせセクション、ヘルプページへの誘導、カスタマーサポート情報。

## StaticListWithTitleAndGrayBackgroundIconLink

説明: アイコン付きの静的リストアイテム。タイトル、説明、編集リンクを含む情報カード。

主なユースケース: 設定済み情報の表示と編集リンク、配送先住所の表示、アカウント情報の確認画面。
