# Buttons

Click, toggle, switch, spin - command your dashboard with colorful admin buttons.

## PrimaryButton

説明: 青色背景のプライマリボタン。アイコンとテキストを組み合わせた5種類のサイズバリエーション。

主なユースケース: メインアクション（保存、送信、作成）、フォームの確定ボタン、強調したいアクション。

## WhiteButton

説明: 白背景にボーダー付きのセカンダリボタン。アイコンとテキストを組み合わせた5種類のサイズバリエーション。

主なユースケース: キャンセル、戻る、補助的なアクション、プライマリボタンと組み合わせて使用。

## GhostButton

説明: 背景なしの透明なゴーストボタン。ホバー時にグレー背景が表示される。5種類のサイズバリエーション。

主なユースケース: 控えめなアクション、テキストリンク的な使用、ツールバー内のボタン、目立たせたくないアクション。

## DashedBorderButton

説明: 点線ボーダーの丸型ボタン。追加アクションを示唆するデザイン。

主なユースケース: フィルター追加、比較追加、新規項目の追加、任意の追加アクション。

## SwitchLabelButton

説明: クリックでラベルが切り替わるトグルボタン。チェックマーク付きの接続/切断状態表示。

主なユースケース: 接続/切断のトグル、フォロー/フォロー解除、お気に入り登録、状態切り替え。

## PrimaryIconButton

説明: アイコンのみのプライマリボタン。正方形で5種類のサイズバリエーション。

主なユースケース: ツールバーのアクションボタン、追加ボタン、コンパクトなアクション、モバイル向けボタン。

## WhiteIconButton

説明: アイコンのみの白背景ボタン。ボーダー付きで5種類のサイズバリエーション。

主なユースケース: セカンダリアクション、編集・削除などのアクション、リスト項目の操作ボタン。

## GhostIconButton

説明: アイコンのみの透明なゴーストボタン。5種類のサイズバリエーション。

主なユースケース: ツールバーの控えめなアクション、サイドバーのアイコンボタン、閉じるボタン。

## LabelButtons

説明: アイコンとラベル付きの丸型ナビゲーションボタン。リンク形式のボタングループ。

主なユースケース: セクションナビゲーション（受信箱、アクティビティ、ファイル）、カテゴリフィルター、タブ的なナビゲーション。
