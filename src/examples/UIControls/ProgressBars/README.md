# Progress Bars

Utilize dynamic progress bars to showcase product reviews and shopping process indicators, enhancing user engagement and clarity.

## MultipleBarsProgress

説明: 複数色の横並びプログレスバーとカラー凡例付きリスト。トラフィック元の比率などを表示。

主なユースケース: アクセス元の分布表示、カテゴリ別の統計表示、ストレージ使用量の内訳。

## ClothingRating

説明: 衣類評価用のスライダー型プログレスバー。品質、フィット感、快適さの各指標を表示。

主なユースケース: ECサイトの商品レビュー詳細、フィットネスアプリの評価指標、サービス品質の多軸評価。

## RatingSnapshot

説明: 星評価分布を表示するプログレスバーグループ。5段階評価の各星ごとのレビュー数を表示。

主なユースケース: 商品レビューの評価分布、アプリストアの評価表示、サービス評価の内訳。

## OrderStatusProgress

説明: 注文ステータスを表示するステップ型プログレスバー。現在のステータスをアニメーションで強調。

主なユースケース: ECサイトの注文追跡、配送状況の表示、ワークフローの進捗表示。

## MultipleProgressBarsWithLegendIndicators

説明: 複数の分離したプログレスバーとストライプパターン付き。凡例インジケーターで色分け。

主なユースケース: タスク優先度の分布表示、ステータス別の件数表示、リスクレベルの可視化。

## SquareShapedSegmentedProgressSection

説明: 四角形のセグメント型プログレスバー。10分割で進捗率を視覚的に表示。

主なユースケース: ストレージ使用量の表示、目標達成率の表示、処理進捗の段階表示。

## ColumnProgressBarWithIndicators

説明: 縦型のカラムプログレスバー。NPS（顧客推奨度）などの評価分布を表示。

主なユースケース: NPS調査結果の表示、満足度調査の回答分布、投票結果の可視化。

## WithGradientBackground

説明: グラデーション背景のプログレスバー。現在位置をドットインジケーターで表示。

主なユースケース: パフォーマンススコアの表示、健康指標の可視化、評価レベルの表示。

## ListOfHorizontalProgressGroup

説明: 技術スタックの人気度などを表示する横型プログレスバーリスト。パーセンテージと回答数付き。

主なユースケース: 技術アンケート結果の表示、スキル習熟度の表示、選択肢ごとの投票率表示。
