# Button Groups

Both horizontally and vertically grouped responsive button groups.

## BasicWhiteButtonGroup

説明: 曜日選択用のボタングループ。白背景にボーダー付きで、レスポンシブ対応（モバイルで縦並び、デスクトップで横並び）。

主なユースケース: 曜日選択、期間選択、複数オプションからの単一選択。

## SegmentStyleButtonGroup

説明: セグメントスタイルのアイコンボタングループ。白背景でコンパクトなデザイン、ツールチップ付き。

主なユースケース: 通知の既読/アーカイブ操作、項目に対するクイックアクション、コンパクトなツールバー。

## ButtonWithDropdown

説明: メインボタンとドロップダウンメニューを組み合わせたボタングループ。追加アクションをドロップダウンで表示。

主なユースケース: 返信と追加オプション、公開/非公開/削除のアクション、プライマリとセカンダリアクションの組み合わせ。

## InlineWithTitleButtonGroup

説明: タイトルラベルと前後ナビゲーションボタンの組み合わせ。矢印アイコン付き。

主なユースケース: 商品や項目の前後移動、ページネーション、リスト内アイテムのナビゲーション。

## WasArticleHelpfulButtonGroup

説明: 記事の有用性を問うフィードバック用ボタングループ。Yes/Noボタンとフィードバック統計表示。

主なユースケース: ヘルプ記事のフィードバック収集、FAQ評価、コンテンツ有用性調査。

## InboxCommentButtonGroup

説明: 丸型のアイコンボタングループ。リアクション追加、返信、編集、削除のアクション用。ツールチップ付き。

主なユースケース: メッセージへのクイックアクション、コメント操作ツールバー、SNS風のインタラクション。

## LabelButtonGroup

説明: アイコンとラベル付きの丸型ボタングループ。アップロード、デザイン、テンプレート作成などのアクション用。

主なユースケース: コンテンツ作成のクイックアクション、空状態からの新規作成オプション、ダッシュボードのアクションメニュー。

## CommentButtonGroup

説明: コメント操作用のコンパクトなボタングループ。返信テキストボタンと編集・削除アイコンボタンの組み合わせ。

主なユースケース: コメントへの返信と管理、チャットメッセージの操作、フィードバック項目の編集・削除。

## VideoCallMainPageButtonGroup

説明: ビデオ通話開始ページ用の大型ボタングループ。参加、新規ミーティング、スケジュール、画面共有のオプション。

主なユースケース: ビデオ会議アプリのメイン画面、会議開始オプション、リモートコラボレーションツール。

## ButtonGroupWithIconsAndTooltips

説明: アイコンボタンとツールチップを組み合わせたボタングループ。ボーダーで区切られたデザイン。

主なユースケース: プロジェクトのコピーとアーカイブ、ファイル操作、ツールバーアクション。

## IconAndLinkButtonGroup

説明: アイコンとテキストリンク形式のボタンリスト。矢印付きでモーダルなどへの遷移を示唆。

主なユースケース: 設定項目へのリンク、地域・言語選択、フィードバック送信リンク。

## VideoCallInMeetingButtonGroup

説明: ビデオ通話中の操作パネル用ボタングループ。カメラ/マイクのトグル、参加者、チャット、共有、録画、リアクション、終了ボタン。

主なユースケース: ビデオ会議中のコントロールパネル、オンラインミーティングツールバー、ライブ配信コントロール。

## CommentTextareaButtonGroup

説明: テキストエリア付随のコンパクトなアイコンボタングループ。ファイル添付、絵文字、音声メッセージ、送信ボタン。

主なユースケース: チャット入力欄のツールバー、コメント投稿フォームの付属ボタン、メッセージ送信インターフェース。
