# Selects

Vibrant tags, filter, searchable inputs, and customizable admin select menus that are as beautiful as they are functional.

## SimpleSelect

説明: シンプルなドロップダウンセレクト。ボーダー付きのベーシックなデザインで、選択時にチェックマークが表示される。

主なユースケース: 基本的なフォームの選択入力、設定画面のオプション選択、カテゴリ選択。

## MiniSelect

説明: コンパクトな権限選択用セレクト。小さいサイズでインライン使用に最適化。

主なユースケース: 共有権限の選択（閲覧可能、編集可能、管理者）、ファイルアクセス権限の設定、ユーザー権限の切り替え。

## CurrencySelect

説明: 国旗アイコン付きの通貨選択セレクト。検索機能付きで多くの通貨から選択可能。

主なユースケース: 決済フォームの通貨選択、国際取引の通貨設定、為替計算の通貨ペア選択。

## StatusSelect

説明: カラーインジケーター付きのステータス選択セレクト。タスクの状態を色分けして視覚的に表示。

主なユースケース: タスク管理のステータス変更（完了、保留、進行中など）、チケットステータスの更新、ワークフロー状態の選択。

## ViewSelect

説明: 表示形式切り替え用のセレクト。アイコンとテキストでボード、テーブル、タイムラインなどのビューを選択。

主なユースケース: プロジェクト管理ツールのビュー切り替え、データ表示形式の選択、ダッシュボードのレイアウト変更。

## WithIconsSelect

説明: アイコン付きのファイルタイプフィルターセレクト。ファイル種別（ドキュメント、画像、スニペットなど）を選択。

主なユースケース: ファイル管理システムのフィルタリング、メディアライブラリの種類別表示、コンテンツタイプの絞り込み。

## CustomTemplateWithAvatarSelect

説明: 国旗アイコン付きの言語・国選択セレクト。各選択肢にカスタムテンプレートを適用。

主なユースケース: 言語設定の切り替え、国・地域の選択、ローカライズ設定。

## AssigneeSelect

説明: アバター付きの担当者複数選択セレクト。タグモードで選択済みユーザーをチップ表示。検索と削除が可能。

主なユースケース: タスクの担当者割り当て、プロジェクトメンバーの選択、レビュアーの指定。

## ItemListWithDescriptionSelect

説明: アイコンと説明文付きの公開範囲選択セレクト。各選択肢に詳細な説明を表示。

主なユースケース: コンテンツの公開範囲設定、プライバシーレベルの選択、共有先の指定。

## TagsSelect

説明: アバター付きのユーザー複数選択セレクト。丸型タグスタイルで選択済みユーザーを表示。

主なユースケース: メンション先のユーザー選択、チームメンバーのタグ付け、コラボレーター追加。

## TagStyleSelect

説明: 課題タイプのタグ選択セレクト。絵文字付きのラベル（Bug、Support、Feature、Docs）を複数選択可能。

主なユースケース: GitHub Issue のラベル選択、タスクカテゴリの分類、バグ・機能要望の種類指定。

## ThreadsTagSelect

説明: ピルスタイルの通貨選択セレクト。国旗付きで検索機能を搭載。コンパクトな丸角デザイン。

主なユースケース: 送金・決済の通貨選択、価格表示の通貨切り替え、取引通貨の選択。

## TimeSelect

説明: 時間選択用のセレクト。12時間形式（AM/PM）で1時間単位の時刻を選択。

主なユースケース: 予約時間の選択、スケジュール設定、リマインダー時刻の指定。

## SearchInsideDropdown

説明: 検索機能内蔵の国選択セレクト。国旗アイコン付きで多数の国から検索・選択可能。

主なユースケース: 住所フォームの国選択、電話番号の国コード選択、配送先国の指定。

## InlineWithLabelSelect

説明: ラベル付きのインラインソートセレクト。「Sort:」などのプレフィックスと組み合わせたコンパクトなデザイン。

主なユースケース: リストの並び替え（新しい順、古い順、A-Z）、検索結果のソート、ファイル一覧の表示順設定。

## WithinInputSelect

説明: テキスト入力フィールド内に配置された権限選択セレクト。入力とセレクトを組み合わせたUI。

主なユースケース: 招待メール入力と権限選択の同時操作、共有リンク作成時の権限設定、ユーザー追加フォーム。
