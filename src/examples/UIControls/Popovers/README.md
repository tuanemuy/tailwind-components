# Popovers

Use Popovers to display additional information or actions in a compact overlay triggered by user interaction.

## AvatarGroupWithUserInfoPopover

説明: アバターにホバーすると表示されるユーザー情報ポップオーバー。名前、メール、ステータスを表示。

主なユースケース: 参加者リストのユーザー詳細表示、メンション先のプレビュー、チームメンバーの情報確認。

## ContactProfilePopover

説明: クリックで開く連絡先プロフィールポップオーバー。アバター、名前、メール、フラグ・連絡先追加ボタン付き。

主なユースケース: メール送信者の詳細表示、CRMの連絡先プレビュー、ユーザー名クリック時のプロフィール表示。
