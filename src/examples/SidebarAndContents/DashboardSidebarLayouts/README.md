# Dashboard Sidebar Layouts

Switch between Classic, Compact, Mini admin layouts and colorful background options for sidebar haven.

## ClassicDashboardSidebarLayout

説明: 標準幅(260px)のライト背景サイドバーレイアウト。テキストナビゲーションとアコーディオンによる階層メニューを備え、プロジェクト切り替えドロップダウンやフッターのアカウント情報表示を含む完全なダッシュボードサイドバー構成

主なユースケース: 多機能な管理画面やSaaSダッシュボードのメインナビゲーション、複数プロジェクトを扱うワークスペース、一般的なWebアプリケーションのサイドバー

## ClassicInvertedSidebarLayout

説明: 標準幅(260px)のダーク背景(blue-950)サイドバーレイアウト。ClassicDashboardSidebarLayoutと同じ構造で、反転カラースキームを採用。テキストナビゲーション、アコーディオンメニュー、プロジェクト切り替え機能を備える

主なユースケース: ブランドカラーを活かした管理画面、視覚的な差別化が必要なダッシュボード、ダークテーマを基調としたアプリケーション

## CompactSidebarLayout

説明: 狭幅(64px)のライト背景サイドバーレイアウト。アイコンとテキストラベルを縦に配置したコンパクトなナビゲーション。通知バッジやツールチップ付きで、限られたスペースでも機能性を維持

主なユースケース: コンテンツ領域を最大化したいダッシュボード、シンプルなナビゲーション構造のアプリ、タブレット向けの省スペースUI

## CompactInvertedSidebarLayout

説明: 狭幅(64px)のダーク背景(blue-950)サイドバーレイアウト。CompactSidebarLayoutと同じコンパクト構造で反転カラースキームを採用。アイコン中心のナビゲーションで視認性を確保

主なユースケース: モダンなダークテーマのダッシュボード、ブランドカラーを強調した管理画面、コンテンツ重視のアプリケーション

## MiniSidebarLayout

説明: デスクトップではアイコンのみ(64px)、モバイルでは展開時にテキストナビゲーション(352px)を表示するレスポンシブサイドバー。ライト背景でホバー時にドロップダウンメニューを表示し、サブメニューへのアクセスを提供

主なユースケース: コンテンツ領域を最大限確保したい大規模ダッシュボード、階層の深いナビゲーション構造、デスクトップとモバイルで異なるUXを提供したい場合

## MiniInvertedSidebarLayout

説明: デスクトップではアイコンのみ(64px)、モバイルでは展開時にテキストナビゲーション(260px)を表示するレスポンシブサイドバー。ダーク背景(blue-950)の反転カラースキームで、ホバー時のドロップダウンメニュー機能を備える

主なユースケース: ダークテーマでコンテンツ領域を最大化したいダッシュボード、視覚的インパクトのある管理画面、プロフェッショナルな印象のWebアプリ
