# Demo Dashboard Sidebars & Contents Layouts

Find the perfect admin workspace setup by experiencing Payments, Files, CRM, Analytics demo layouts.

## AIChatDetailDemoLayout

説明: AIチャット機能に特化したダッシュボードレイアウト。折りたたみ可能なサイドバーにチャット履歴リストを表示し、メインエリアにチャットインターフェースを配置。サイアン系のブランドカラーを採用

主なユースケース: AIチャットボット管理画面、カスタマーサポートチャットシステム、対話履歴を一覧表示する会話型AIアプリケーション

## AnalyticsDemoLayout

説明: データ分析・統計情報表示に特化したダッシュボードレイアウト。アカウントドロップダウン、設定メニュー、ダークモード切り替えを備えたヘッダーを持つ。インディゴ系のブランドカラーを採用

主なユースケース: ビジネスインテリジェンスダッシュボード、KPI監視画面、データ分析レポートシステム、マーケティング分析ツール

## DashboardDemoLayout

説明: 汎用的なダッシュボードレイアウト。グローバル検索バー、ヘルプサポートドロップダウン、通知システムを備えた完全なヘッダー構成。標準的なサイドバーナビゲーションとコンテンツエリアを提供

主なユースケース: 一般的な管理画面のベーステンプレート、SaaSアプリケーションのメインダッシュボード、企業向け業務システム

## FilesDemoLayout

説明: ファイル管理・ドキュメント管理に特化したダッシュボードレイアウト。プロジェクト・ファイル・ユーザー検索機能を備えたヘッダー、通知タブ付きドロップダウンを提供。ピンク系のブランドカラーを採用

主なユースケース: ファイルストレージ管理画面、ドキュメント管理システム、チームコラボレーションツールのファイルセクション

## PaymentDemoLayout

説明: 決済・支払い管理に特化したダッシュボードレイアウト。中央揃えのコンテンツ領域、ダークモード切り替え、通知タブ機能を持つコンパクトなヘッダー。パープル系のブランドカラーを採用

主なユースケース: 決済管理ダッシュボード、サブスクリプション管理画面、請求書・取引履歴管理システム、ECサイトの注文管理
