# Multi Column Layouts

Choose from Mini, Dual-Column, or Asymmetric admin dashobard layouts to conquer the admin workflow.

## ContentBetweenTwoSidebarsLayout

説明: 左右両側にサイドバーを配置した3カラムレイアウト。左に標準ナビゲーションサイドバー、右にセカンダリサイドバーを設け、中央にコンテンツを表示。各サイドバーは独立してトグル可能で、プロジェクト切り替えドロップダウンを備える

主なユースケース: 詳細パネルを必要とする管理画面、メール/チャット型アプリケーション(左にリスト、右に詳細)、複雑な情報構造を持つダッシュボード

## DualColumnsLayout

説明: 左に標準サイドバー、右にセカンダリサイドバーを持つデュアルカラムレイアウト。両サイドバーは独立したオーバーレイとしてモバイルで表示可能。プロジェクト切り替え機能付きヘッダーを備える

主なユースケース: リスト・詳細ビューを持つアプリケーション、サイドパネルで追加情報を表示する管理画面、マスター・ディテール形式のデータ管理

## DualColumnsLayoutWithMiniOnRight

説明: 左に標準サイドバー、右にミニサイドバー(アイコンのみ)を配置したレイアウト。右サイドバーは補助的なクイックアクションやツールを提供。両サイドバーは独立してトグル可能

主なユースケース: ツールパレットを右側に配置したいアプリケーション、クイックアクションボタンを常時表示したいダッシュボード、デザインツール風のUI

## GrayMiniWithSidebarMultiColumnLayout

説明: グレー背景(gray-200)のミニサイドバー(64px)と白背景の詳細サイドバーを組み合わせたマルチカラムレイアウト。ミニサイドバーにはアイコンとラベル付きナビゲーション、詳細サイドバーにはリストやコンテンツを表示

主なユースケース: チャット・メッセージングアプリのインターフェース、カスタマーサポートシステム、インボックス形式のコミュニケーションツール

## MiniInvertedMultiColumnLayout

説明: ダーク背景のミニサイドバー(64px)と標準サイドバー(260px)を組み合わせた2段階ナビゲーションレイアウト。ヘッダーには検索バーと通知ドロップダウンを配置。合計幅352pxのサイドバー領域を持つ

主なユースケース: 大規模な階層ナビゲーションを持つ管理画面、カテゴリとサブカテゴリを視覚的に分離したいダッシュボード、エンタープライズ向けアプリケーション
