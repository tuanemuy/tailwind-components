# Detached Sidebar

Minimize distractions, and maximize insights, never stealing the spotlight with detached admin dashboard sidebars.

## ContainerDetachedSidebar

説明: グレー背景(gray-50)上にサイドバーとコンテンツを白いカード形式で配置した分離型レイアウト。最大幅コンテナ(90rem)内でサイドバーが独立したスクロール領域を持ち、ヘッダーには検索バーと通知ドロップダウンを配置

主なユースケース: モダンなカード型UIを採用した管理画面、視覚的にサイドバーを際立たせたいダッシュボード、余白を活かしたクリーンなデザインのアプリケーション

## OverlayDetachedSidebar

説明: 白背景に固定ヘッダーを配置し、サイドバーがコンテンツと視覚的に分離したレイアウト。中央揃えコンテナ(85rem)で3カラムグリッドヘッダーを採用し、検索バーとアカウントドロップダウンを備える。モバイルではサイドバーがオーバーレイ表示

主なユースケース: ヘッダー中心の情報設計を持つダッシュボード、グローバル検索を重視する管理画面、コンテンツ幅を制限したいアプリケーション
