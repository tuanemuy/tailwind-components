# E-Commerce Order Details

View complete order specifics, tracking information, and transaction history for seamless order management and updates.

## ExchangeOfReturnCardUnchecked

説明: 返品・交換対象として選択可能な商品カード。商品画像、商品名、カラー、サイズ、数量、価格を表示。右上に円形チェックボックスを配置（未選択状態）。グレー背景の外枠と白い内側カードの2層構造。

主なユースケース: 返品・交換フローでの商品選択画面、複数商品から返品対象を選ぶUI

## OrderDetails

説明: 注文詳細の包括的なカード。ヘッダーにステータス（緑のドットインジケータ付き）、注文番号、注文日、合計金額を4列で表示。内側カードには配送先住所、電話番号、編集ボタンを含む。

主なユースケース: ユーザーの注文履歴詳細ページ、マイページでの注文情報確認、配送先変更が可能な注文管理画面

## ActiveOrder

説明: 進行中の注文ステータスカード。ヘッダーにステータス、注文番号、注文日、合計金額。本体に配送予定日（トラックアイコン付き）と追跡情報を表示。注文が進行中であることを視覚的に示す。

主なユースケース: 注文確認後の進行状況表示、配送追跡ページ、ダッシュボードでの最新注文ステータス

## ExchangeOfReturnCardExchangeOptions

説明: 交換・返品オプション付きの商品カード。商品情報とチェックボックス（選択済み状態）を表示し、下部に「Exchange」「Return」のタブ切り替えUIを配置。返品期限切れの場合の注意メッセージも表示。サイズやカラーの交換オプションを選択可能。

主なユースケース: 商品の交換・返品申請フロー、サイズ交換やカラー交換の詳細オプション選択、返品ポリシーに基づいた交換/返品の切り替え
