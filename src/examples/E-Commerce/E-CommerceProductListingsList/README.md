# E-Commerce Product Listings: List Layout

Efficiently browse multiple products with a clean and organized list layout featuring clear visuals with essential details.

## MarketplaceListingListViewInSlider

説明: マーケットプレイス向けの新着商品スライダー。「New arrivals」タイトルと前後ナビゲーションボタンをヘッダーに配置。ドラッグ可能なカルーセルで商品カードを横スクロール表示。各カードに商品画像（ホバーズーム）、商品名、価格を表示。

主なユースケース: マーケットプレイストップページの新着商品セクション、カテゴリページでのおすすめ商品表示、スペースを節約した商品一覧

## ShopListingWithSlider

説明: 「おすすめ商品」スライダー。「You may also like」タイトル付き。スナップスクロールとカスタムスクロールバー、ドットインジケータ対応。「Trending」バッジ（炎絵文字付き）、お気に入りボタン、商品名、カテゴリ、価格を表示。正方形画像を使用。

主なユースケース: 商品詳細ページのレコメンドセクション、カート画面の「こちらもおすすめ」、ユーザーの閲覧履歴に基づくおすすめ商品表示
