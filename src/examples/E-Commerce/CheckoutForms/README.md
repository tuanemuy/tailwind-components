# Checkout Forms

Simplified checkout process with intuitive forms to ensure a smooth and quick transaction completion.

## CheckoutLoggedIn

説明: ログイン済みユーザー向けのチェックアウトフォーム。アカウント情報を表示しつつ、連絡先情報と配送先住所の入力・編集が可能。国選択には検索機能付きドロップダウンを使用。

主なユースケース: 会員制ECサイトでログインユーザーが購入手続きを行う画面、保存済み情報を活用した迅速なチェックアウトフロー

## CheckoutAsGuest

説明: ゲスト（未ログイン）ユーザー向けのチェックアウトフォーム。アカウント登録不要で購入可能。ログインページへのリンクも提供し、連絡先と配送先住所を入力する構成。

主なユースケース: 会員登録なしで購入したいユーザー向けのチェックアウト、初回購入時の離脱率を下げるためのゲスト購入オプション

## ReviewAndPay

説明: チェックアウトの最終確認・支払いステップ。連絡先、配送先住所、配送方法を確認カードとして表示し、それぞれ編集可能。クレジットカード、PayPal、Klarna等の支払い方法選択とカード番号入力フォームを含む。

主なユースケース: 購入前の最終確認画面、複数の支払い方法を提供するECサイトの決済ページ

## OrderConfirmation

説明: 注文完了後の確認ページ。注文番号、確認メール送信先を表示。配送ステータスをタイムライン形式（注文受付→発送→配達中→配達完了）で視覚化。印刷ボタンとショッピング継続リンクを含む。

主なユースケース: 注文完了後のサンキューページ、注文状況の追跡表示、購入完了の証明画面

## ShopOrderConfirmation

説明: シンプルなショップ向け注文完了ページ。中央配置のサクセスメッセージ、注文金額サマリー、商品リストを表示。印刷と継続ショッピングボタンを配置。

主なユースケース: コーヒーショップなど専門店向けの注文確認、簡潔な注文完了表示が必要な場合

## ShopBag

説明: ショッピングバッグ（カート）ページ。商品画像、名前、オプション（重量、挽き方など）、数量選択、単価、在庫状況を表示。商品の編集・削除機能付き。

主なユースケース: コーヒーショップなど専門店のカートページ、商品オプションが多い場合のカート表示

## ShoppingBad

説明: ファッションEC向けショッピングバッグ。商品画像（お気に入りボタン付き）、価格、色・サイズ・数量の選択UI、在庫警告（Low in stock）、削除機能を備えた構成。

主なユースケース: アパレルECのカートページ、サイズ・カラー選択が必要な商品のカート、在庫状況を表示したいカート画面

## OrderDetails

説明: 管理者・ユーザー向けの注文詳細カード。注文日時、支払い状況（Paid）、履行状況（Unfulfilled）をバッジで表示。商品画像、アイテム名、カラー、サイズ、価格、数量、配達予定日を一覧表示。複製・キャンセル・アーカイブ・編集・印刷・削除のアクションメニュー付き。

主なユースケース: 管理画面での注文詳細表示、ユーザーの注文履歴詳細ページ、注文管理機能

## OrderSummary

説明: 注文内容のサマリーカード。商品一覧（画像、価格、商品名、カラー、サイズ、数量バッジ）、プロモーションコード入力欄、小計・配送料・税金・合計金額の明細を表示。カート編集と次へ進むボタンを配置。

主なユースケース: チェックアウトフローのサイドバーに表示する注文サマリー、支払い前の金額確認、プロモーションコード適用

## ShopCheckout

説明: ショップ向けワンページチェックアウトフォーム。配送先住所（氏名、住所、市区町村、都道府県、郵便番号、電話番号）と支払い方法（クレジットカード）を一画面で入力。複数のカードブランドアイコンを表示。

主なユースケース: シンプルなワンページチェックアウト、コーヒーショップなど専門店のスピーディな決済フロー
