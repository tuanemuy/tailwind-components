# Chat Widgets

Interactive Chat Widgets. Perfect for chat bots, feedback submission, FAQs, and more. Features user listing, forms, chat bubbles, and different layouts.

## Components

### WelcomeChatWidget
説明: チャットウィジェットのウェルカム画面コンポーネント。ヘッダーにサポートチームのアバターグループ、挨拶メッセージ、CTAボタンを表示。人気のヘルプトピックリンク一覧とフッターのタブナビゲーション（Home/Messages/Help）を備える。ドロップダウン型チャットウィジェットの初期表示画面として機能する。
主なユースケース: カスタマーサポートチャットボットの初期画面、ヘルプデスクウィジェットのランディング画面、FAQ導線を含むチャット開始画面

### MessagesChatWidget
説明: チャットウィジェット内のメッセージ一覧画面。各会話のプレビュー（送信者アバター、名前、最新メッセージ抜粋、タイムスタンプ）をリスト表示する。未読メッセージにはバッジが表示され、会話アイテムはクリッカブル。新規メッセージ作成ボタンとタブナビゲーションを含む。
主なユースケース: サポートチャットの会話履歴一覧、複数の問い合わせスレッド管理、過去のチャット履歴へのアクセス画面

### LeaveMessageChatWidget
説明: オフライン時やチャット不可時のお問い合わせフォーム画面。名前、メールアドレス、件名、メッセージ本文の入力フィールドとファイル添付機能を備える。送信ボタンと戻るナビゲーションを含むシンプルなコンタクトフォーム。
主なユースケース: サポート担当者不在時のメッセージ送信、問い合わせフォーム、フィードバック収集画面

### ConversationChatWidget
説明: リアルタイムチャット会話画面。ヘッダーに担当者のアバター、名前、オンラインステータスを表示。チャットエリアにメッセージバブル（送受信両方）を時系列で表示し、フッターにメッセージ入力欄と送信ボタンを配置。絵文字ピッカーとファイル添付ボタンも備える。
主なユースケース: カスタマーサポートとのリアルタイムチャット、1対1のメッセージング画面、ライブチャットサポート

### HelpTopicsChatWidget
説明: ヘルプトピック選択画面。ボットアイコンとともにヘルプカテゴリ（Getting Started、Billing、など）をボタン形式で一覧表示。ユーザーがトピックを選択してFAQや関連情報にアクセスするための導線画面。戻るボタンとタブナビゲーションを含む。
主なユースケース: FAQカテゴリ選択、セルフサービスヘルプの入口、チャットボットのトピック選択画面

### SuccessMessageChatWidget
説明: メッセージ送信完了後の確認画面。成功アイコン、完了メッセージ、追加アクション（新規メッセージ作成、ホームに戻る）へのリンクを表示。ユーザーに送信成功をフィードバックするシンプルな確認画面。
主なユースケース: 問い合わせ送信完了確認、フィードバック送信後のサンキュー画面、フォーム送信成功通知

### ChatWidgetSidebarExample
説明: サイドバー形式のフルハイトチャットウィジェット。ページ右端にオーバーレイとして表示され、ウェルカム画面、メッセージ一覧、会話画面などを含む完全なチャット体験を提供。ドロップダウン型ではなくサイドバー型のレイアウトバリエーション。
主なユースケース: EC サイトのサイドバーチャット、ダッシュボードのヘルプサイドパネル、画面端固定のカスタマーサポートウィジェット
