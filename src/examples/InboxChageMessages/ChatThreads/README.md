# Chat Threads

Chat Threads display organized conversations, allowing users to follow and manage messages in a structured layout

## Components

### InboxThread
説明: メールスレッド形式のアコーディオン型メッセージ表示コンポーネント。送信者名、宛先、日付、メッセージプレビューをヘッダーに表示し、クリックで本文を展開できる。送信者・受信者のポップオーバーで連絡先詳細を確認可能。返信、全員に返信、転送、再送信、未読/既読マーク、アーカイブ、削除、スパム報告、印刷、会話リンクのコピーなど多彩なアクションメニュー付き。
主なユースケース: メールクライアントのスレッド表示、サポートチケットの会話履歴、ビジネスコミュニケーションの受信トレイ

### InboxComposeThread
説明: メール返信・作成フォームのコンポーネント。From、To、Cc、Bcc、Subjectフィールドを備え、宛先はタグ形式で複数選択可能。Tiptapエディタを使用したリッチテキスト入力（太字、イタリック、下線、取り消し線、リスト、引用、コード）、ファイル添付、リンク挿入、絵文字、音声メッセージの追加に対応。送信、送信＆アーカイブ、予約送信のオプション付き。
主なユースケース: メール返信フォーム、新規メッセージ作成、チームメンバーへの一斉送信
