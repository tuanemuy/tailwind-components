# Chat Bubbles

Versatile Chat Bubbles for dynamic conversations. Features include customizable styles, timestamp displays, user avatars, file attachments, and more.

## Components

### MessageChatBubble
説明: 基本的なテキストメッセージバブル。送信者側（右寄せ）のスタイルで、コピーと編集のアクションボタンがホバー時に表示される。シンプルなテキストメッセージの表示に最適。
主なユースケース: 1対1のチャット画面でのテキストメッセージ送信表示、シンプルなメッセージUI

### ReplyToMessage
説明: 引用付きの返信メッセージバブル。元のメッセージを引用ブロックとして表示し、その下に返信内容を表示する形式。タイムスタンプと既読マーク付き。
主なユースケース: メッセージへの返信機能、スレッド形式の会話、Slackやメッセンジャーのような引用返信

### Reply
説明: AIチャットボットや詳細な回答を表示するためのメッセージバブル。見出し、リスト、段落などのリッチテキスト形式に対応。再生成、コピー、Good/Bad評価、共有、ソースリンクなどの多彩なアクションボタン付き。
主なユースケース: AIアシスタントの回答表示、チャットボットの応答UI、詳細な説明やドキュメント形式の返信

### CommentWithReaction
説明: 絵文字リアクション機能付きのコメントバブル。メンション（@ユーザー名）に対応し、リアクションの追加・表示ができる。アクションボタン群（リアクション追加、返信、編集、削除）がホバー時に表示される。
主なユースケース: Slackスタイルのメッセージリアクション、チーム内コミュニケーション、コメント機能付きのチャット

### PreviewLinkChatBubble
説明: URLを共有した際にリンクプレビュー（OGP）を表示するメッセージバブル。サイト名、タイトル、説明文、プレビュー画像を含むリッチなリンクカードを表示。
主なユースケース: URLシェア時のリンクプレビュー、SNSのようなリンク共有機能、記事やサイトの共有

### ImageWithCommentChatBubble
説明: 画像とテキストコメントを組み合わせたメッセージバブル。画像の下にコメントを表示する形式で、連続した複数メッセージにも対応。
主なユースケース: スクリーンショットと説明文の送信、バグ報告、画像付きの質問や説明

### ImageGroupChatBubble
説明: 複数の画像をグリッドレイアウトで表示するメッセージバブル。左側に大きな画像、右側に縦に2枚の小さな画像を配置するギャラリー形式。
主なユースケース: 複数画像の一括共有、写真ギャラリーの送信、メディアリッチなメッセージ

### DownloadAttachedFileChatBubble
説明: ファイル添付付きのメッセージバブル。ダウンロードボタン、ファイル名、ファイルサイズを表示。ファイルのダウンロードリンク付き。
主なユースケース: ファイル共有、ZIPやドキュメントの送信、添付ファイル付きメッセージ

### AttachedImageWithCommentChatBubble
説明: 画像サムネイルとテキストコメントを横並びで表示するメッセージバブル。画像のファイル名とサイズ、ダウンロードリンクを含む。
主なユースケース: 画像ファイルの共有と説明、スクリーンショット付きの問い合わせ、ファイル添付と説明文の組み合わせ

### MessageWithLinks
説明: テキスト内にクリック可能なリンクを含むメッセージバブル。メールアドレスやURLを自動的にリンク化して表示。
主なユースケース: メールアドレスやURLを含むメッセージ、リンク付きの説明文、連絡先情報の共有

### VoiceMessage
説明: 音声メッセージを表示するバブル（受信者側）。再生ボタンと波形ビジュアライザー、再生時間を表示。
主なユースケース: 音声メッセージの受信表示、ボイスチャット機能、音声メモの共有

### VoiceMessageNotPlayed
説明: 未再生状態の音声メッセージバブル（送信者側）。再生ボタン、波形ビジュアライザー、再生時間に加え、未再生を示すインジケーター（青いドット）と既読マークを表示。
主なユースケース: 送信した音声メッセージの表示、未再生状態の視覚的表示、既読・未読ステータスの表示
