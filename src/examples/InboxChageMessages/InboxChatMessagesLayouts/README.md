# Inbox, Chat & Messages Layouts

Comprehensive Inbox, Chat & Messages UI layouts featuring user listings, detailed profiles, search functionality, empty states, and more.

## Components

### ChatLayout
説明: チャットアプリケーションのメインレイアウト。左サイドバーにチャットスレッド一覧（検索、ソート、作成機能付き）、メインエリアにチャット会話画面を配置。ユーザーのアバター、最新メッセージプレビュー、タイムスタンプ、未読バッジなどを含む完全なチャットUI。レスポンシブ対応でモバイルではオーバーレイサイドバー。
主なユースケース: カスタマーサポートチャットアプリ、社内メッセージングシステム、リアルタイムチャットダッシュボード

### InboxLayout
説明: メールクライアント風のインボックスレイアウト。ヘッダーにロゴ、検索バー、ユーザーアクションボタン、左サイドバーにナビゲーションメニュー、メインエリアにメールスレッド一覧を配置。フォルダ分類、検索、フィルタリング機能を備えた本格的なメールUI。
主なユースケース: ウェブメールクライアント、サポートチケット管理システム、社内メールアプリケーション

### ChatEmptyStateLayout
説明: チャット画面の空状態（会話未選択時）レイアウト。サイドバーにスレッド一覧、メインエリアに「会話を選択してください」などの案内メッセージやイラストを表示。ユーザーにアクションを促すためのビジュアルと説明文を含む。
主なユースケース: チャットアプリの初期表示画面、会話未選択時のプレースホルダー、新規ユーザーへのガイダンス表示

### ChatActivityLayout
説明: チャットアクティビティ表示レイアウト。Activity/Mentions/Assignedのタブ切り替え、日付範囲指定のカレンダードロップダウンを備える。チームメンバーのアクティビティログ、メンション通知、割り当てられたタスクを一覧表示。
主なユースケース: チームコラボレーションツールのアクティビティフィード、通知センター、タスク割り当て管理画面

### ChatReportsLayout
説明: チャットシステムのレポート・分析画面レイアウト。リフレッシュボタン、最終更新時刻表示、日付範囲選択のカレンダードロップダウンを備えたヘッダー。グラフやチャート、統計データを表示するためのメインコンテンツエリア。
主なユースケース: カスタマーサポートの分析ダッシュボード、チャット統計レポート、パフォーマンス指標表示

### CRMCustomerDetailsLayout
説明: CRMシステムの顧客詳細画面レイアウト。ダークテーマのヘッダーにロゴ、グローバル検索、ユーザーアクションを配置。顧客情報、連絡履歴、取引データなどを詳細表示するためのレイアウト。検索ショートカット（Cmd+/）対応。
主なユースケース: CRMの顧客詳細ページ、営業管理システムの顧客プロファイル、カスタマーサクセス管理画面

### ChatTicketsLayout
説明: サポートチケット管理のレイアウト。左サイドバーにナビゲーション（Inbox、Tickets、Visitors、Activity、Reports）とアイコンメニュー、中央にチケット一覧、右にチケット詳細を配置した3カラムレイアウト。チケットのステータス、優先度、担当者管理機能を備える。
主なユースケース: ヘルプデスクシステム、カスタマーサポートチケット管理、問い合わせ対応ダッシュボード
