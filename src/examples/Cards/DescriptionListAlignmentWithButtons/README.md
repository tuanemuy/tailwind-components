# Description List Alignment With Buttons

Cards with description lists and action buttons for displaying structured information.

## Components

### index

説明: デバイスセッション情報を表示するカードコンポーネント。ヘッダーにはデバイスアイコンとサインアウトボタン、デバイス名と「Current session」バッジ、詳細情報リスト（Location、Device、IP address、Recent activity）、および問い合わせ用ヘルプボタンが含まれる。

主なユースケース: アクティブセッション管理画面、ログイン履歴の詳細表示、セキュリティ設定でのデバイス管理、アカウント保護のためのセッション確認
