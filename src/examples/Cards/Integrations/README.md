# Integrations

Explore Integrations UI blocks showcasing connected apps, sync status, and actions. Ideal for building integration panels, settings, or plugin pages.

## Components

### IntegrationCard

説明: 連携サービスカードを2列グリッドで表示するコンポーネント。各カードにはサービスアイコン（Slack、Notion、GitHub、Gmail、Mailchimp、GitLab等）、サービス名、説明文、インストール状態（Install now/Installed/Coming soon）が含まれる。インストール済みのカードは無効化状態で表示される。

主なユースケース: アプリ連携設定画面、マーケットプレイスのアプリ一覧、サードパーティサービスとの接続管理

### IntegrationGridView

説明: 多数の連携サービスをグリッドレイアウトで一覧表示するコンポーネント。各サービスのアイコン、名称、説明、接続状態を含む。大規模なインテグレーション選択画面に最適化されたレイアウト。

主なユースケース: 大規模なアプリストア・インテグレーションカタログ、プラグイン一覧ページ、ワークフロー自動化ツールの連携選択
