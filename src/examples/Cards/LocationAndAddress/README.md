# Location & Address

Explore Location & Address UI blocks featuring maps, contact details, and address layouts. Ideal for building location, directory, or contact pages.

## Components

### AddressCardsGridView

説明: 住所カードを3列グリッドで表示するコンポーネント。各カードにはラベル名（Breannabury、Portland等）、Defaultバッジ、住所情報（名前、住所、電話番号）、アクションリンク（Edit、Remove、Set as default）を含む。最後に破線ボーダーの「Add address」追加ボタンが配置される。

主なユースケース: EC サイトの配送先住所管理、複数住所の登録・管理画面、ユーザーアカウントの住所帳

### StoreContacts

説明: 店舗連絡先カードを国別にグループ化してグリッド表示するコンポーネント。各カードには店舗外観画像、都市名、住所、営業時間、「Get direction」リンクが含まれる。国旗アイコン付きの国名ヘッダーでセクション分け。

主なユースケース: 実店舗一覧ページ、グローバル企業の拠点紹介、店舗検索結果の表示、問い合わせ先の地域別一覧
