# Project Details

Discover Project Details UI blocks featuring timelines, team info, status updates, and summaries—ideal for dashboards and collaboration pages.

## Components

### AlternativeListingOptionDetails

説明: プロジェクト詳細を横並びで表示するカード。左側にプロジェクト名と会社ロゴ、右側に総合進捗（円形プログレス）、タスク進捗（バーインジケーター）、プロジェクトコスト状況を3列グリッドで表示。コンパクトな横長レイアウト。

主なユースケース: プロジェクト一覧でのインライン詳細表示、ダッシュボードのプロジェクトサマリー、横スクロール可能なプロジェクトリスト

### DefaultOptionProjectDetails

説明: プロジェクト詳細を縦方向のグリッドで表示するカード。ヘッダーにプロジェクト名・会社ロゴ・ドロップダウンメニュー（Edit、Duplicate、Archive、Delete）、ボディに総合進捗、タスク進捗、コスト、経過時間（総時間・日数）を4列グリッドで表示。詳細な情報表示に最適化。

主なユースケース: プロジェクト詳細ページ、ダッシュボードのメインプロジェクトカード、プロジェクト管理画面での詳細表示
