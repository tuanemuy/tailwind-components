# Inline Cards

Browse Inline Cards UI blocks designed for compact content display, featuring quick actions, summaries, and contextual details within layouts.

## Components

### FloatingUserImages

説明: 友人招待を促すカード。中央にタイトル、説明文、「Send an invite」ボタンを配置し、カード周囲に複数のユーザーアバター画像が浮遊するデザイン。コミュニティやチーム感を視覚的に演出する。

主なユースケース: チームメンバー招待機能、リファーラルプログラムのCTA、コラボレーションツールの招待画面、コミュニティ参加促進

### InlineWithImageWithListView

説明: 左側にテキストコンテンツ（タイトル、説明、アクションリンク）、右側にプレビュー画像を持つ横長カードのリスト。2列グリッドレイアウトでレスポンシブ対応。各カードは独立したプロモーションコンテンツとして機能。

主なユースケース: 機能紹介カード、テンプレートやドキュメンテーションへの誘導、製品のセクション別案内、オンボーディングフローのステップ表示
