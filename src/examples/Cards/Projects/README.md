# Projects

Explore Projects UI blocks showcasing overviews, statuses, team members, and timelines—ideal for portfolio, dashboard, and collaboration interfaces.

## Components

### FilesCardGridView

説明: ファイルカードをグリッド形式で表示するコンポーネント。各カードにはファイルプレビュー、ファイル名、メタ情報、ドロップダウンメニューが含まれる。カード形式でのファイル一覧に最適化。

主なユースケース: ギャラリー形式のファイルブラウザ、プロジェクト内のアセット一覧、ドキュメントギャラリー

### FilesCardListView

説明: ファイルカードをカード形式のリストで表示するコンポーネント。各カードにはファイルプレビュー、ファイル名、編集日時、サイズ、ドロップダウンメニューが含まれる。

主なユースケース: ドキュメント一覧、プロジェクトファイルリスト、メディアライブラリ

### FilesGridView

説明: ファイルを4列グリッドで表示するコンポーネント。各アイテムにはサムネイルプレビュー、ファイル名、編集日時・サイズ情報、Moreドロップダウンが含まれる。

主なユースケース: 画像やドキュメントのサムネイル一覧、アセット管理画面、ファイル選択インターフェース

### FilesListView

説明: ファイルをリスト形式で表示するコンポーネント。各アイテムにはサムネイルプレビュー、ファイル名、編集日時・サイズ情報、Moreドロップダウンが含まれる。最初と最後の角丸処理でグループ化された外観。

主なユースケース: ファイルエクスプローラー、最近のファイル一覧、添付ファイルリスト

### FilesMultipleImagesGridView

説明: 複数画像を含むファイルカードをグリッドで表示するコンポーネント。各カードには複数のサムネイル画像、ファイル名、メタ情報が含まれる。

主なユースケース: フォトアルバム一覧、画像コレクション、プロジェクトのビジュアルアセット

### FilesMultipleImagesListView

説明: 複数画像を含むファイルカードをリスト形式で表示するコンポーネント。各アイテムには複数のサムネイル画像、ファイル名、メタ情報が含まれる。

主なユースケース: アルバムリスト表示、画像セットの一覧、マルチメディアファイル管理

### Projects

説明: プロジェクトカードを3列グリッドで表示するコンポーネント。各カードにはサービスロゴ、会社名、プロジェクト名、ステータスバッジ、アバターグループ、進捗情報、アクションドロップダウンが含まれる。

主なユースケース: プロジェクト一覧ダッシュボード、クライアントプロジェクト管理、ワークスペースのプロジェクト概要

### TeamListView

説明: チームカードをリスト形式で表示するコンポーネント。各カードにはチーム名（ハッシュタグ形式）、タグバッジ、説明文、メンバーアバターグループ、アクションボタン（チーム切り替え、参加/離脱）が含まれる。

主なユースケース: チーム一覧ページ、コミュニティグループリスト、部門・チーム管理画面

### UserCardWithHeroBackground

説明: ユーザープロフィールカードをヒーロー背景付きで表示するコンポーネント。上部にカラフルなSVG背景、大きなアバター画像（Proバッジ付き）、名前、職種、フォロワー/フォロー数、ソーシャルリンクアイコン、アクションボタンが含まれる。

主なユースケース: チームメンバー紹介、ユーザープロフィール表示、クリエイター・専門家の紹介カード
