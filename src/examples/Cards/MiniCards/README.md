# Mini Cards

Explore Mini Cards UI blocks for showcasing free trials, plan highlights, and upgrade prompts—ideal for pricing pages and onboarding flows.

## Components

### Credits

説明: クレジット使用量を表示するミニカード。使用量テキスト（65/100 credits used）、サブテキスト、「Upgrade now」リンク、円形プログレスインジケーターを含む。カード全体がリンクとして機能。

主なユースケース: APIクレジット残量表示、サブスクリプション使用量の表示、リソース消費状況の可視化

### HelpAndResources

説明: ヘルプとリソースを提供するミニカード。サポートチームのアバターグループ、「Need help?」タイトル、説明文、「Book a call」と「Resources」アクションボタンを含む。

主なユースケース: サイドバーのサポートセクション、オンボーディング中のヘルプ案内、ダッシュボードのサポートウィジェット

### IntroVideo

説明: イントロダクションビデオを表示するミニカード。タイトル、説明文、サムネイル画像付きビデオプレビュー（中央に再生ボタン）を含む。ダーク/ライトモード両対応のサムネイル。

主なユースケース: 製品チュートリアルの紹介、オンボーディングビデオの埋め込み、機能説明動画へのアクセスポイント

### Trial

説明: トライアル期限を通知するミニカード。残り日数（Trial ends in 7 days）、プログレスバー、「Schedule a call」ボタン、閉じるボタンを含む。削除可能なアラート形式。

主なユースケース: トライアル期限のリマインダー、サブスクリプション更新促進、コンバージョン向上のためのCTA

### UpgradeToProCard

説明: Proプランへのアップグレードを促すミニカード。ギフトアイコン、タイトル（Upgrade to Pro for unlimited access）、説明文、「Upgrade now」ボタンを含む。グレー背景でサイドバーに最適化。

主なユースケース: サイドバーでのアップグレード促進、フリープランユーザーへのプロモーション、機能制限時のアップグレード案内
