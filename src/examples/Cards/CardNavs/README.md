# Card Navs

From segmented to bordered admin navigation tabs put every action at your fingertips.

## Components

### BarOnTopNavs

説明: カード上部にカラーバーインジケーターを持つタブナビゲーション。各タブにはアイコン、ラベル、数値が含まれ、選択時に上部のカラーバーがアクティブになる。ダッシュボードのメトリクス表示に最適化されたレイアウト。

主なユースケース: アナリティクスダッシュボードのKPI切り替え、ユーザー統計の表示切り替え、パフォーマンスメトリクスの分類表示

### BorderedNavs

説明: ボーダー付きのピル型タブナビゲーション。丸みを帯びた外枠の中にタブボタンが配置され、選択時に背景色が変わるセグメントコントロール風のデザイン。

主なユースケース: フィルター切り替えUI、コンテンツカテゴリの選択、ステータス別の表示切り替え

### ButtonBorderNavs

説明: タブボタンの下にアンダーラインインジケーターを持つナビゲーション。ヘッダー右端に設定ボタンを配置できる構成。タブ選択時にアンダーラインがアクティブになる。

主なユースケース: カード内のコンテンツ切り替え、設定アクションを併せ持つタブUI、All/Archivedなどのステータスフィルター

### ButtonNavs

説明: シンプルなボタン形式のタブナビゲーション。選択時に背景色がハイライトされる角丸ボタンが横並びに配置される。

主なユースケース: シンプルなカテゴリ切り替え、データフィルタリング、コンテンツのグループ切り替え

### CardWithIconsNavs

説明: アイコン、カウント数、ラベルを含むカード型ナビゲーション。各カードがタブとして機能し、グリッドレイアウトで配置される。カウントバッジとカラーアイコンで視覚的に情報を伝える。

主なユースケース: タスク管理ダッシュボード（今日・予定・全件の切り替え）、メールボックス型UI、通知やアラートのカテゴリ表示

### CenterAlignedWithCheckboxNavs

説明: 支払い方法選択用のカード型ナビゲーション。ラジオボタン形式で排他的選択が可能。選択時にチェックマークが表示され、サービスロゴ・名称・説明文を含むカードデザイン。

主なユースケース: 支払い方法の選択、配送方法の選択、プランやオプションの単一選択フォーム

### FullWidthBorderNavs

説明: 横幅いっぱいの下線ボーダーを持つタブナビゲーション。選択時にタブ下部にアンダーラインインジケーターが表示される。カード全幅に渡るボーダーラインがデザインの特徴。

主なユースケース: カードコンテンツの切り替え、データのステータス別フィルタリング、シンプルなタブインターフェース

### HoverTypeNavs

説明: ホバー操作でタブ切り替えが発生するナビゲーション。クリック不要でマウスオーバー時にコンテンツが切り替わる。支払い方法選択カード形式でチェックマーク付き。

主なユースケース: クイックプレビュー機能を持つ選択UI、素早い情報閲覧が必要なダッシュボード、オプション比較インターフェース

### MaterialDesignNavs

説明: カード下部に配置されるマテリアルデザインスタイルのボトムナビゲーション。アイコンとラベルが縦に並び、モバイルアプリのような外観を持つ。

主なユースケース: モバイルアプリ風のWebダッシュボード、カードウィジェットのナビゲーション、コンパクトなセクション切り替え

### SegmentNavs

説明: iOSのセグメントコントロールに似たピル型のナビゲーション。完全に丸みを帯びたボタンが横に並び、選択時に背景色が変わる。

主なユースケース: バイナリ切り替え（All/Archived等）、モバイルフレンドリーなフィルター、コンパクトなカテゴリ選択

### VerticalNavs

説明: 左側に縦型ナビゲーション、右側にコンテンツを配置するレイアウト。モバイルではセレクトボックスに変換されるレスポンシブ対応。各タブにはアイコンとラベルが含まれる。

主なユースケース: 設定画面のセクション切り替え、サイドバー型ナビゲーション、多階層のコンテンツ整理
