# Data Snippets

Data Snippets offer concise data points for quick reference and easy analysis, helping users gain insights efficiently.

## Components

### CardGridWithBadge

説明: プロジェクトサマリーカードをグリッド形式で表示するコンポーネント。各カードにはプロジェクト名、合計作業時間、作業日数、ステータスバッジ（Completed/Active/On hold/No status）が含まれる。ステータスに応じてバッジの色が変化。

主なユースケース: プロジェクト進捗ダッシュボード、タイムトラッキング結果の一覧表示、クライアント別作業サマリー

### FollowUsCards

説明: SNSフォローリンクを2列グリッドで表示するコンポーネント。各カードにはSNSサービスのアイコン（丸形背景）、サービス名、説明文が含まれ、カード全体がリンクとして機能する。

主なユースケース: フッターやサイドバーのSNSリンク、コンタクトページのSNS一覧、マーケティングページのフォロー誘導

### UserList

説明: ファイルスニペットを2列グリッドで表示するコンポーネント。各アイテムにはファイルタイプアイコン（Notion、GitHub、Excel、Word等）、ファイル名（切り詰め表示対応）、ファイルサイズが含まれる。コンパクトなファイル情報表示に最適化。

主なユースケース: 添付ファイル一覧の表示、最近使用したファイルのリスト、チャットやメッセージ内のファイル参照表示
