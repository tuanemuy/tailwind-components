# Contacts

The Contacts section features inquiry forms, company details such as addresses, and other essential contact information.

## Components

### AvatarAndGrayBackgroundIconCards

説明: アバター画像またはアイコンを含むミニカードを3列グリッドで表示するコンポーネント。グレー背景の丸形アイコンとラベルで構成され、送金先やアクション対象の選択に使用される。

主なユースケース: 送金先の種類選択（自分・他者・会社）、アクション対象のカテゴリ選択、ペイメントアプリでの受取人タイプ選択

### AvatarWithCheckboxButtons

説明: ユーザーリストにConnect/Unconnectトグルボタンを配置したコンポーネント。各アイテムにはアバター（画像またはイニシャル）、名前、ユーザー名が表示され、チェックボックスで接続状態を切り替え可能。

主なユースケース: フォロー/フォロー解除リスト、ユーザー接続管理、招待するユーザーの選択、チーム編成時のメンバー選択

### CenterAlignedMiniCardsWithAvatars

説明: アバターとサービスアイコンバッジを含むミニカードをグリッドで表示するコンポーネント。各カードは中央揃えで、アバター下部にサービスロゴ（Prelineや国旗アイコン等）が配置される。

主なユースケース: 送金先アカウントの選択、連携サービス別のユーザー一覧、マルチアカウント対応の受取人選択

### UserListCardListView

説明: ユーザーカードを2列グリッドで表示するコンポーネント。各カードにはアバター（画像またはイニシャルバッジ）、Proステータスバッジ、名前、職種、電話番号、メールアドレスが含まれる。カード全体がリンクとして機能。

主なユースケース: チームメンバー一覧、人材ディレクトリ、顧客・クライアントリスト、連絡先検索結果の表示

### UserProfileConnectionsGridView

説明: ユーザープロフィールカードをグリッド形式で表示するコンポーネント。アバター、オンライン状態インジケーター、名前、所属情報、接続数、Connect/Connectedトグル、その他アクション用のドロップダウンメニューを含む。

主なユースケース: ソーシャルネットワークの接続候補表示、コミュニティメンバー一覧、プロフェッショナルネットワークのプロフィールカード

### UserProfileConnectionsListView

説明: ユーザープロフィールカードをリスト形式で表示するコンポーネント。各カードにはアバター、オンライン状態、名前、所属会社、接続数が表示され、Connect/Connectedボタンで接続状態を管理できる。

主なユースケース: 接続リクエスト管理、フォロワー/フォロー中リスト、ネットワーク上の繋がり一覧、チームメンバーの詳細表示
