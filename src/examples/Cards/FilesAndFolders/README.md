# Files & Folders

Neatly designed files & folders UI built with Tailwind CSS for efficient organization and display.

## Components

### FilesListView

説明: ファイル一覧をリスト形式で表示するコンポーネント。各ファイルにはアイコン、ファイル名、詳細情報、コンテキストメニュー（右クリック操作）が含まれる。ファイル管理ダッシュボード向けの詳細表示。

主なユースケース: クラウドストレージのファイル一覧、ドキュメント管理システム、ファイルエクスプローラー風インターフェース

### Folders

説明: フォルダカードを横スクロール可能なリストで表示するコンポーネント。各カードにはフォルダアイコンとフォルダ名が含まれ、右クリックでコンテキストメニュー（Share、Move to、Starred、Rename、Download、Report、Delete）が表示される。

主なユースケース: クイックアクセス用フォルダ一覧、最近使用したフォルダの表示、フォルダナビゲーション

### PreviewFilesCard

説明: ファイルプレビューカードを横スクロール可能な形式で表示するコンポーネント。各カードにはサムネイル画像（画像ファイルの場合）またはファイルタイプアイコン（PDF、フォルダ等）、ファイル名、所属フォルダ/カテゴリが含まれる。右クリックでコンテキストメニューが表示される。

主なユースケース: スター付きファイルの表示、最近のファイルプレビュー、ギャラリー形式のファイルブラウザ、メディアファイルの一覧表示
