# Masonry Layout Cards

Transform data into captivating visual insights with admin dashboard responsive masonry layout cards.

## Components

### GridView

説明: チームやプロジェクトのカードをメイソンリー（石積み）レイアウトで表示するコンポーネント。各カードには画像、ハッシュタグリンク、ドロップダウンメニュー（Edit team、Add to favorites、Archive team、Delete）が含まれる。異なる高さのカードが自然に配置されるPinterest風レイアウト。

主なユースケース: チームコレクションの表示、プロジェクトギャラリー、クリエイティブポートフォリオ、ソーシャルメディア風フィード
