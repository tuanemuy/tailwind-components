# Branding Featured Cards

Stylish and responsive branding featured cards built with Tailwind CSS. Perfect for highlighting key features and services.

## Components

### AppsGridView

説明: アプリダウンロードカードをグリッド形式で表示するコンポーネント。各カードにはプラットフォームアイコン、タイトル、説明文、過去バージョンへのリンク、ダウンロードボタンが含まれる。iOS、Android、MacOS向けのアプリ配布ページに適したデザイン。

主なユースケース: モバイルアプリやデスクトップアプリのダウンロードページ、プラットフォーム別アプリ紹介セクション、ソフトウェア製品のダウンロード案内

### StackedBrandsListCardListView

説明: 外部サービスやブランドのカードを2カラムのリスト形式で表示するコンポーネント。各カードにはサービスのアイコンまたはイニシャルバッジ、サービス名、カテゴリータグが含まれる。クリック可能なリンクカードとして機能し、ホバー時にハイライトされる。

主なユースケース: 連携サービス・インテグレーション一覧、パートナー企業やツールの紹介、アプリマーケットプレイスでのサービス一覧表示
