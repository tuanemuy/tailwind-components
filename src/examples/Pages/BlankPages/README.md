# Blank Pages

ダッシュボードアプリケーション用の特殊ページ集。機能アップグレード促進ページやエラーページなど、コンテンツが限定された状態で表示するページテンプレート。サイドバー付きレイアウトで、メインコンテンツエリアにセンター配置のメッセージを表示。

## UpgradeBlankPage
説明: 機能アップグレード促進ページ。分析チャートのモックアップ画像、「Available with growth」バッジ、機能説明テキスト、アップグレードボタンと詳細リンクを配置。プレミアム機能へのアクセスを促すデザイン。
主なユースケース: フリーミアムモデルでの有料プラン誘導、機能制限時のアップグレード案内、プレミアム機能のプレビュー表示

## ServerErrorPage
説明: 500サーバーエラーページ。大きな「500」エラーコード、「Page not found」見出し、サーバー内部エラーの説明メッセージ、「Reload page」ボタンを配置。
主なユースケース: サーバー内部エラー発生時の表示、システム障害時のユーザー案内、API通信エラー時のフォールバック

## NotfoundErrorPage
説明: 404ページ未検出エラーページ。大きな「404」エラーコード、「Page not found」見出し、ページが見つからない旨のメッセージ、「Back to home」ボタンを配置。
主なユースケース: 存在しないURLへのアクセス時の表示、削除されたコンテンツへのアクセス時、無効なリンクのフォールバック
