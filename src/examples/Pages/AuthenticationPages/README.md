# Authentication Pages

認証関連のフルページレイアウト集。ログイン、サインアップ、パスワードリセット、二段階認証、オンボーディングなど、ユーザー認証フローに必要なページテンプレート。サイドバー付きレイアウトとセンター配置レイアウトの2種類のスタイルを提供。

## サイドバーレイアウト

左側にマーケティングコンテンツ、言語セレクター、パートナーロゴを配置したレイアウト。SaaSやダッシュボードアプリケーション向け。

### LoginPage
説明: サイドバー付きログインページ。メール・パスワード入力、Google/Apple/Facebookなどのソーシャルログインボタン、「ログイン状態を保持」チェックボックス、パスワード忘れリンクを配置。パスワード表示/非表示トグル付き。
主なユースケース: SaaSアプリケーションのログインページ、管理画面へのログイン、ダッシュボードアクセス

### SignupPage
説明: サイドバー付きサインアップページ。メール・パスワード入力フォーム、ソーシャルアカウントでの登録オプション、利用規約への同意チェック、既存アカウントへのログインリンクを配置。
主なユースケース: 新規ユーザー登録ページ、SaaSアプリのアカウント作成、サービス新規申込

### ResetPasswordPage
説明: サイドバー付きパスワードリセットページ。メールアドレス入力フォームとリセットリンク送信ボタン、ログインページへの戻りリンクを配置。
主なユースケース: パスワードを忘れたユーザーへのリセットメール送信ページ、セキュリティ上のパスワード再設定

### TwoStepVerificationPage
説明: サイドバー付き二段階認証ページ。6桁のPINコード入力欄（個別入力フィールド）、再送信リンク、確認ボタンを配置。入力フィールド間の自動フォーカス移動対応。
主なユースケース: SMS/メール認証コード入力、二要素認証の確認画面、セキュリティコード入力

### EmailVerificationPage
説明: サイドバー付きメール認証確認ページ。認証メール送信完了メッセージ、送信先メールアドレス表示、再送信リンク、確認ボタンを配置。
主なユースケース: メールアドレス確認の待機画面、認証メール送信後の案内ページ、アカウント有効化の確認

### LockScreenPage
説明: グラデーション背景のセンター配置ロック画面。ユーザーアバターと名前表示、パスワード入力フォーム、ログアウトリンクを配置。一時的に離席したユーザーの再認証用。
主なユースケース: セッションタイムアウト後の再認証、離席後のロック解除、既存ログインユーザーの本人確認

### OnboardingPlansPage
説明: サイドバー付き料金プラン選択ページ。Free/Startup/Enterpriseなど複数プランをカード形式で表示。各プランの特徴、価格、機能比較を含む。スキップボタン付き。
主なユースケース: サービス登録時のプラン選択、フリーミアムモデルのアップグレード誘導、サブスクリプション選択

### OnboardingWhatIsYourNamePage
説明: サイドバー付き職種選択ページ。Design/Product Management/Sales/User Researchなど複数の役割をチップ形式で選択。パーソナライズ体験の設定用。スキップボタン付き。
主なユースケース: オンボーディング時のユーザー属性収集、カスタマイズ体験の初期設定、ユーザーセグメント分類

### OnboardingSyncYourContactsPage
説明: サイドバー付き連絡先同期ページ。Google/Outlook/iCloud/CSVなど複数の連絡先ソースからのインポートオプションを表示。
主なユースケース: 連絡先データのインポート設定、外部サービス連携の初期設定、アドレス帳同期

### OnboardingCreateYourProjectPage
説明: サイドバー付きプロジェクト作成ページ。プロジェクト名入力、チームメンバー招待、ファイルアップロードなどの初期設定フォームを配置。
主なユースケース: 初回ログイン時のワークスペース作成、チーム/プロジェクトの初期セットアップ

## センターレイアウト（ECサイト向け）

ヘッダーにロゴ、フッターに利用規約リンクを配置したシンプルなセンター配置レイアウト。ECサイトやショップアプリケーション向け。

### LoginCenteredPage
説明: センター配置のログインページ（ECサイト向け）。ゲストチェックアウトオプション、メール・パスワードログインフォーム、アカウント作成リンクを配置。シンプルなヘッダー・フッター付き。
主なユースケース: ECサイトのログインページ、ゲスト購入とログイン購入の選択画面

### CheckboxNotLoggedInCenteredPage
説明: センター配置のゲストチェックアウト/ログイン選択ページ。ゲストとして続行ボタン、ログインフォーム、アカウント作成リンクを配置。未ログインユーザー向け。
主なユースケース: ECサイトのチェックアウト前認証、会員/非会員の購入フロー分岐

### CreateAccountCenteredPage
説明: センター配置のアカウント作成ページ（ECサイト向け）。メール・パスワード入力（強度インジケーター付き）、生年月日、性別選択、メールマガジン購読、利用規約同意チェックを配置。
主なユースケース: ECサイトの会員登録ページ、顧客情報収集を含む詳細なサインアップ

### ForgotPasswordCenteredPage
説明: センター配置のパスワードリセットページ（ECサイト向け）。メールアドレス入力フォーム、送信ボタン、ログインページへの戻りリンクを配置。
主なユースケース: ECサイトのパスワード再設定、顧客向けパスワードリセット

### OrderCheckupCenteredPage
説明: センター配置の注文確認ページ。タブ切り替えで「既存顧客ログイン」と「注文追跡」を表示。注文追跡は電話番号/注文番号での検索、メール、郵便番号入力に対応。
主なユースケース: ECサイトの注文履歴・追跡ページ、会員/非会員向け注文状況確認

### CheckOrderPage
説明: センター配置の注文追跡フォーム。電話番号または注文番号での検索切り替え、メール、郵便番号入力。ツールチップで注文番号の確認方法を案内。
主なユースケース: 非会員向け注文追跡、配送状況確認フォーム
