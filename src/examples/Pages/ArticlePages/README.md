# Article Pages

記事形式のコンテンツを表示するフルページレイアウト集。求人詳細、事例紹介、顧客ストーリーなど、長文コンテンツを読みやすく構成したページテンプレート。

## AgencyCareersDetails

説明: ダークテーマの求人詳細ページ。職種タイトル、業務内容、応募要件、福利厚生などを詳細に記載し、右サイドバーに応募情報（締め切り、部署、雇用形態、勤務地、報酬）と応募ボタンを配置。バックリンク付き。

主なユースケース: 採用サイトの求人詳細ページ、キャリアページの個別職種紹介、求人募集の詳細情報表示

## AgencyCaseStudyDetails

説明: ダークテーマのケーススタディ（事例紹介）ページ。ヒーロー画像を上部に配置し、課題・解決策・効果のセクション構成でプロジェクト成果を紹介。右サイドバーに企業情報（業界、設立年、資金調達、CEO）を表示。

主なユースケース: クライアント事例紹介、プロジェクト成功事例の詳細ページ、導入事例・ケーススタディの掲載

## StartupCustomersArticle

説明: ライト/ダークモード対応の顧客事例記事ページ。左サイドバーに戻るリンク、中央に主要成果指標のハイライト表示と対談者アバター、課題・解決策・効果のセクション構成。下部に「More stories」として関連事例カードを配置。右サイドバーに企業情報（会社名、業界、設立年、資金調達、CEO）をスティッキー表示。

主なユースケース: 顧客成功事例の詳細記事、インタビュー形式のケーススタディ、導入企業の声・効果を紹介するページ
