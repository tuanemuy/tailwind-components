# Sidebar Filters

サイドバーに配置するフィルターコンポーネント集です。価格範囲、チェックボックス、スイッチ、バッジなど、様々なフィルタリングオプションを組み合わせて使用します。

---

## コンポーネント一覧

### ShopMarketplaceSidebarFilter
**マーケットプレイスサイドバーフィルター**

説明: マーケットプレイス向けの包括的なサイドバーフィルター。カテゴリー選択、価格範囲スライダー、ブランドチェックボックス、カラー選択、レーティングフィルターなど、ECサイトに必要なフィルタリング機能を網羅しています。

主なユースケース: ECサイトの商品絞り込み、マーケットプレイスのフィルタリング

---

### ShopSidebarFilter
**ショップサイドバーフィルター**

説明: モバイル対応のオーバーレイ式サイドバーフィルター。ボタンクリックで開閉し、各種フィルターオプション（カテゴリー、価格、サイズ、色など）を表示します。レスポンシブ対応で、デスクトップでは常時表示、モバイルではオーバーレイ表示になります。

主なユースケース: ECサイトの商品フィルター、検索結果の絞り込み
