# Range Filter with Charts

レンジスライダーとチャートを組み合わせたフィルタリングコンポーネント集です。データ範囲の調整と視覚化を動的に行えます。

---

## コンポーネント一覧

### CenterAlignedRangeSliderWithIconInSlider
**カルーセル式照明コントロールスライダー**

説明: カルーセル内に配置された照明調整用のレンジスライダー。中央にアイコンと現在値（パーセンテージ）が大きく表示され、下部のスライダーで値を調整します。スマートホームアプリのような洗練されたUIです。

主なユースケース: スマートホーム照明制御、IoTデバイス設定、明るさ調整

---

### PaymentTransactionsPriceRange
**価格範囲フィルタ（チャート連動）**

説明: 価格範囲を設定するデュアルハンドルのレンジスライダー。背景にデータ分布を示すチャートが表示され、Min/Maxの数値入力欄も備えています。フィルタリング対象のデータ量を視覚的に確認できます。

主なユースケース: ECサイトの価格フィルター、取引金額の絞り込み、データ分析ダッシュボード

---

### TransparentRangeWithIcons
**透明背景アイコン付きレンジスライダー**

説明: 透明な背景に配置されたアイコン付きレンジスライダー。左側にアイコンとラベル、右側にスライダーが配置され、現在値がパーセンテージで表示されます。ダーク系UIに適したデザインです。

主なユースケース: 照明・音量調整、設定パネル、コントロールダッシュボード
