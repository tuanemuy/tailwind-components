# Add/Create Forms

新規作成・追加用フォームのコンポーネント集です。管理画面やダッシュボードで使用する、データ入力に特化したフォームUIを提供します。

---

## コンポーネント一覧

### AddDiscount
**ECサイト向け割引コード作成フォーム**

クーポンやプロモーションコードを作成するためのフォームです。

- 割引名の入力（請求書に表示される名前）
- 割引コードの入力と自動生成ボタン
- 割引額の設定（パーセント `%` または固定金額 `$` を選択可能）
- 開始日・終了日の設定（カレンダーピッカー付きドロップダウン）
- 利用回数制限の設定
- 「下書き保存」と「公開」の2つのアクションボタン

**適したユースケース:** ECサイト、SaaSの料金管理、マーケティングキャンペーン管理

---

### AddProject
**シンプルなプロジェクト作成フォーム**

新規プロジェクトを素早く作成するためのミニマルなフォームです。

- プロジェクトロゴのアップロード（円形プレビュー）
- プロジェクト名の入力
- サブドメインの設定（`.preline.co` などの固定サフィックス付き）
- キャンセル・保存ボタン

**適したユースケース:** プロジェクト管理ツール、ワークスペース作成、SaaSのテナント作成

---

### AvatarUploadWithSwitchesAndInlineInputs
**ストア/ショップ設定フォーム**

ECストアやショップの基本情報を設定する包括的なフォームです。

- アバター/ロゴのアップロード（削除ボタン付き）
- ストア名とストアURL（2カラムレイアウト）
- 連絡先メールアドレスと電話番号
- 通貨の選択（多数の通貨コードから選択可能）
- その他のスイッチやインライン入力フィールド

**適したユースケース:** ECプラットフォーム、マルチテナントSaaS、店舗管理システム

---

### CalendarCreateEvent
**カレンダーイベント新規作成フォーム（コンパクト版）**

カレンダーアプリでイベントを素早く作成するためのインライン形式のフォームです。

- タイトル入力（自動高さ調整テキストエリア）
- アイコン付きラベル（左側にアイコン、右側に入力フィールド）
- 開始時刻・終了時刻の選択（15分刻み、AM/PM表示）
- 省スペースでコンパクトなデザイン

**適したユースケース:** カレンダーアプリ、スケジュール管理、会議予約システム

---

### CalendarEditEvent
**カレンダーイベント編集フォーム**

既存のカレンダーイベントを編集するためのフォームです。CalendarCreateEventと同様の構成ですが、既存データが入力済みの状態で表示されます。

- 既存のイベントタイトルが入力済み
- 開始時刻・終了時刻が設定済み
- 変更を保存するためのUIフロー

**適したユースケース:** カレンダーイベントの詳細編集、予定の変更

---

### CreateNewUser
**ユーザー作成フォーム（詳細版）**

システムに新しいユーザーを追加するための包括的なフォームです。

- アバター画像のアップロード
- 名前とメールアドレス
- 電話番号（国コード + 番号、複数追加可能）
- 住所（Street、City、Zip/Postal code）
- 国の選択（検索機能付きセレクトボックス）
- ラベルが左側、入力フィールドが右側の3:9レイアウト

**適したユースケース:** 管理者向けユーザー管理、CRM、従業員登録システム

---

### CreateTeam
**チーム作成フォーム**

コラボレーション機能を持つアプリでチームを作成するためのフォームです。

- チームタイトルの入力
- 業種（Industry）の入力
- 説明文（自動高さ調整テキストエリア）
- メンバー追加（入力フィールド + 追加ボタン）
- 推奨メンバーの選択（チェックボックス付きアバターリスト、ツールチップで名前表示）
- カバー画像のドラッグ&ドロップアップロード

**適したユースケース:** チームコラボレーションツール、プロジェクト管理、組織管理

---

### CustomRecurrenceEvent
**繰り返しイベント設定フォーム**

定期的なイベントの繰り返しルールを設定するためのフォームです。

- 繰り返し間隔の設定（数値 + 単位：Daily/Weekly/Monthly/Yearly）
- 繰り返す曜日の選択（月〜日のボタン型チェックボックス）
- 終了条件の設定
  - 終了しない（Never）
  - 特定の日付で終了（On + 日付入力）
  - 指定回数で終了（After + 回数入力）

**適したユースケース:** カレンダーの繰り返しイベント、定期タスク、サブスクリプション設定

---

### ProductInfoUpload
**商品情報入力フォーム**

ECサイトの商品を登録・編集するためのフォームです。

- サムネイル画像のアップロード（サイズ・形式の説明付き）
- 商品名（ツールチップでヒント表示）
- SKUと重量（2カラムレイアウト、重量は単位選択付き：lb/oz/kg/g）
- 商品説明（リッチテキストエディター）
  - 太字、斜体、下線、取り消し線などの装飾
  - 箇条書き、番号付きリスト
  - 引用、コードブロック

**適したユースケース:** ECサイトの商品管理、在庫管理システム、カタログ作成

---

### UserProfileCreateEvent
**ユーザープロフィールからのイベント作成フォーム**

ユーザーが自分のカレンダーにイベントを追加するためのフォームです。

- イベントタイトルの入力
- 日時の選択（カレンダードロップダウン + 時刻入力）
- 月/年の選択機能付きカレンダー
- シンプルで使いやすいデザイン

**適したユースケース:** 個人用カレンダー、予定追加、リマインダー設定

---

## 共通の特徴

- **ダークモード対応:** すべてのコンポーネントが `dark:` プレフィックスでダークモードに対応
- **レスポンシブデザイン:** モバイル・デスクトップ両対応のレイアウト
- **アクセシビリティ:** 適切なラベル、フォーカス状態、`aria-*` 属性を使用
- **Preline UI統合:** ドロップダウン、セレクト、ツールチップなどのPreline UIコンポーネントを活用

## 選び方ガイド

| やりたいこと | 推奨コンポーネント |
|------------|------------------|
| 割引・クーポンを作成したい | AddDiscount |
| プロジェクト/ワークスペースを作成したい | AddProject |
| ストア・ショップ情報を設定したい | AvatarUploadWithSwitchesAndInlineInputs |
| カレンダーにイベントを追加したい | CalendarCreateEvent, UserProfileCreateEvent |
| 既存のイベントを編集したい | CalendarEditEvent |
| 繰り返しスケジュールを設定したい | CustomRecurrenceEvent |
| ユーザーを登録したい | CreateNewUser |
| チームを作成したい | CreateTeam |
| 商品を登録したい | ProductInfoUpload |
