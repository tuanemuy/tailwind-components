# Data Cards

A sleek dashboard with various data cards featuring progress bars, charts, and timesheet summaries.

## BlockStatsCard

説明
メール送信・開封などの2つの指標を比較表示するカード。ブロック形式のプログレスバーで視覚的に比率を表現し、凡例インジケーターと変化率も表示する。

主なユースケース
メールマーケティングの送信・開封率の比較表示。キャンペーン成果の可視化。2つの関連指標の対比が必要な統計表示。

## Browsers

説明
ブラウザ別の利用統計をアイコン付きグリッドで表示するカード。メインの統計値をグラデーションで強調し、Chrome/Firefox/Safariなどの主要ブラウザをロゴと利用率で表示する。プッシュ通知設定のトグルスイッチを含む。

主なユースケース
Webアプリケーションのブラウザ別アクセス統計。ブラウザ互換性の分析ダッシュボード。通知機能の設定カード。

## ListBar

説明
コンバージョン目標の達成状況をリスト形式で表示するカード。各項目に背景バーを配置し、Unique/Total/コンバージョン率の3列で数値を表示する。Excel/Word形式でのダウンロード機能付き。

主なユースケース
マーケティングファネルのコンバージョン分析。ゴール別の達成率比較。アナリティクスダッシュボードのレポートカード。

## Progressbars

説明
国別アクセス統計など、複数項目をシンプルな細いプログレスバー付きリストで表示するカード。項目名とパーセンテージを並べて表示し、ダウンロード機能とその他メニューを含む。

主なユースケース
地域別アクセス解析。カテゴリー別の割合表示。シンプルなランキング形式の統計表示。

## ProjectCosts

説明
プロジェクトの総コストと目標達成度を表示するカード。ハンドル（つまみ）付きプログレスバーで進捗を示し、下部に人材・チーム別のコスト内訳をイニシャルアバター付きリストで表示する。

主なユースケース
プロジェクト予算管理。チームメンバー別のコスト配分表示。予算消化状況の可視化。

## SegmentedProgressBarsWithLegendIndicators

説明
インシデントなどの重要度別統計をセグメント化されたプログレスバーで表示するカード。赤/オレンジ/黄/灰色の4色でHigh/Medium/Low/Informationalを視覚的に区別し、縦棒3本のレベルインジケーターで凡例を表現する。

主なユースケース
セキュリティインシデントの重要度分布。タスクの優先度別統計。アラートレベルの可視化。SOC効率分析。

## StackedMultiprogressbar

説明
リードファネルなど複数段階の状態を表示するカード。各ステージに凡例とスタック型プログレスバーを配置し、斜線パターンで区別する。フッターにインサイトメッセージを表示。

主なユースケース
営業パイプラインのステージ管理。リードの有効/ブロック/適格/コールド状態の可視化。ファネル分析ダッシュボード。

## StasCardsWithProgress

説明
売上統計を総合的に表示するカード。ヘッダーにカレンダー日付選択、3つのミニカードで店舗売上/Web売上/割引を表示し、グラデーションプログレスバーで月間目標達成度をBad/Average/Good/Excellentの凡例付きで表示する。

主なユースケース
Eコマースの売上ダッシュボード。店舗とオンラインの売上比較。月間売上目標の進捗管理。

## SurveyDataWithProgress

説明
アンケートの回答結果を水平プログレスバーで表示するカード。ヘッダーに質問内容・回答形式・総回答数を表示し、各選択肢のパーセンテージと回答数を可視化する。

主なユースケース
アンケート結果の可視化。選択式質問の回答分布表示。フィードバック集計結果の表示。

## SurveyDataWithVerticalProgressBar

説明
意見スケール（1-5段階評価など）のアンケート結果を垂直バーチャートで表示するカード。平均スコアをバッジで表示し、各スケールの回答数とパーセンテージを縦バーで可視化する。両端にラベル（例: Difficult / Easy）を配置。

主なユースケース
NPS・満足度調査の結果表示。5段階評価の分布可視化。意見スケールのアンケート結果表示。

## TieredStatsCard

説明
メンバーシップステータスなど、階層型の統計を表示する2段構成のカード。上段でFree/Paidの比率を2色のセグメントプログレスバーで表示し、下段で各ティア（Free/Gold/Silver/Bronze）の詳細を背景バー付きリストで表示する。

主なユースケース
サブスクリプションプランの分布表示。会員ランク別の統計。フリーミアムモデルの有料転換率分析。

## TimeSheet

説明
週間タイムシートを日別のドットインジケーターで表示するカード。各日の勤務時間と目標との差異（over/under）をツールチップで表示し、18個のドットで30分単位の勤務状況を可視化する。目標達成は緑、未達成は灰色で表示。

主なユースケース
従業員の勤怠管理。週間労働時間の可視化。勤務時間の過不足確認。

## TopCardWithProgressAndLinks

説明
タブ切り替え式で注文数/売上を表示するカード。ハンドル付きプログレスバーで目標達成度を示し、下部に「ハイライト表示」「売上データ表示」などのアクションリンクを配置する。

主なユースケース
Eコマースダッシュボードのサマリー表示。注文数と売上金額の切り替え表示。目標達成度の可視化と詳細へのナビゲーション。

## TopCountries

説明
上位3か国などのランキングを国旗と高さの異なるバーで表示するカード。1位/2位/3位を視覚的な高さで区別し、各国の数値をバッジで表示する。フッターにパフォーマンス評価（Good/Badなど）を配置。

主なユースケース
地域別売上ランキング。国別トラフィックの上位表示。市場別パフォーマンス比較。

## Traffic

説明
リファラルトラフィックを色分けされたセグメントプログレスバーと凡例リストで表示するカード。各参照元（github.com、Google等）を色付きインジケーターで区別し、訪問者数を表示する。下部にアバターグループで主要ユーザーを表示。

主なユースケース
Webサイトのトラフィック分析。リファラル元の比較。アクセス解析ダッシュボード。
