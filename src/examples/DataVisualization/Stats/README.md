# Stats

Stats cards with avatars and badges that provide a clear overview of key metrics.

## ChatReports

説明
チャットサポートの統計を3列のグリッドで表示するコンポーネント。サポートリクエスト数、解決済みリクエスト数、AI解決数を先週比の増減率と共に表示する。区切り線で各カードを視覚的に分離。

主なユースケース
カスタマーサポートダッシュボード。チャットボットの効果測定。サポートチームのパフォーマンス概要表示。

## InboxContacts

説明
連絡先管理の統計を4カードのグリッドで表示するコンポーネント。総連絡先数（アバターグループ付き）、新規連絡先、アクティブ連絡先、エンゲージメント率を表示し、各カードに詳細内訳（Synced/Manual、チャット経由/インポートなど）を含む。一部カードはクリック可能。

主なユースケース
CRMダッシュボード。メールマーケティングの連絡先管理。顧客エンゲージメント分析。

## InboxReports

説明
受信トレイの統計を4カードのグリッドで表示するコンポーネント。総メール数（Gmail/Mail等のサービスロゴ付き）、自分への割り当て数、トップソース、平均アクション時間を表示。各カードにReplied/Tagged/Resolved等の詳細内訳とアイコンを含む。

主なユースケース
メール受信箱の統計ダッシュボード。チーム共有メールボックスの管理。メール対応時間の分析。

## PrimaryColorStatsWithBrandsAndBadges

説明
プライマリカラーで統一された売上統計をブランドロゴ付きで表示するコンポーネント。4つのブランド（例: Framer、Figma等）の売上金額を変化率バッジと共に表示。ボーダーで区切られたグリッドレイアウト。

主なユースケース
マーケットプレイスの売上ダッシュボード。プラットフォーム別の収益表示。パートナー/チャネル別の売上比較。

## ProjectReports

説明
プロジェクト管理の統計を4カードで表示するコンポーネント。総時間（請求可能/非請求可能）、残予算（パーセンテージ付き）、内部コスト（時間/経費）、未請求額を表示。最後のカードには請求書作成リンクを含む。

主なユースケース
プロジェクト予算管理ダッシュボード。時間追跡システムのサマリー表示。請求管理画面。

## StatsAnimatedOnHover

説明
ホバー時に浮き上がりアニメーションを持つ統計カード。週間総時間、日平均時間、超過時間、不足時間を表示し、各カードに「View reports」リンクとアイコンを配置。インタラクティブなUIを提供。

主なユースケース
タイムトラッキングダッシュボード。勤怠管理システムの概要表示。プロジェクト時間管理のサマリー。

## StatsWithAvatarGroup

説明
アバターグループ付きで権限別のメンバー統計を表示するコンポーネント。管理者数、メンバー数（アバターグループ付き）、制限付きアクセスメンバー数、保留中の招待数を表示。情報アイコンにツールチップで詳細説明を表示。

主なユースケース
チーム管理ダッシュボード。組織のメンバーシップ概要。権限レベル別のユーザー統計。

## StatsWithIcons

説明
カテゴリー別の売上統計をアイコン付きで表示するコンポーネント。店舗売上、Web売上、割引、アフィリエイト売上を注文数と前期比の変化率（上昇は緑、下降は赤）と共に表示。各カードにカテゴリーを示すアイコンを配置。

主なユースケース
Eコマースの販売チャネル別ダッシュボード。売上分析レポート。複数販売経路の比較表示。
