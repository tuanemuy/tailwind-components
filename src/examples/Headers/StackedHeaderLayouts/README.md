# Stacked Header Layouts

Discover a wide range of stacked admin dashboard header options for landing layouts.

## CenteredNavLinksStackedHeaderLayout

説明
固定位置のスタック型ヘッダー。ロゴ（レスポンシブ対応）、中央配置のナビゲーションリンク、通知・アカウントドロップダウン、コラップスメニュー（モバイル用）を備える3カラムグリッドレイアウト。

主なユースケース
サイドバーを使わないシンプルな管理画面やダッシュボード。ナビゲーションを中央に目立たせたいランディングページ風のアプリケーションに適している。

## NavLinksStackedHeaderLayout

説明
固定位置のスタック型ヘッダー。アイコンロゴ、ナビゲーションリンク、検索ボタン、通知ドロップダウン、アカウントドロップダウン、コラップスメニュー（モバイル用）を備える。

主なユースケース
コンパクトなロゴと複数のナビゲーションリンクを配置したいダッシュボード。ページ間の移動が多いアプリケーションに適している。

## OverlayStackedHeaderLayout

説明
固定位置のスタック型ヘッダー。ロゴ（レスポンシブ対応）、検索入力フィールド（オーバーレイ型）、通知ドロップダウン、アカウントドロップダウンを備える3カラムグリッドレイアウト。

主なユースケース
検索機能を重視するスタック型レイアウトのダッシュボード。コマンドパレット的な検索UIを組み込みたい場合に適している。

## ProjectStackedHeaderLayout

説明
固定位置のスタック型ヘッダー。アイコンロゴ、プロジェクト切り替えドロップダウン（ドラッグハンドル付きリスト）、通知ドロップダウン、アカウントドロップダウンを備える3カラムグリッドレイアウト。

主なユースケース
複数プロジェクトを切り替えながら作業するアプリケーション。サイドバーを使わずにプロジェクト切り替え機能を提供したい場合に適している。

## ClassicStackedHeaderLayout

説明
固定位置のスタック型ヘッダー。ロゴ（レスポンシブ対応）、検索入力フィールド、通知ドロップダウン、アカウントドロップダウンを備える標準的な3カラムグリッドレイアウト。

主なユースケース
汎用的なスタック型ダッシュボードのベースレイアウト。サイドバーなしで基本機能（検索、通知、アカウント）を提供したい場合に適している。

## InvertedStackedHeaderLayout

説明
固定位置のスタック型ヘッダー（ダーク背景: blue-950）。白色ロゴ、ナビゲーションリンク、検索ボタン、通知ドロップダウン、アカウントドロップダウンを備える反転カラースキーム。

主なユースケース
ダークテーマやブランドカラーを強調したいダッシュボード。コントラストの高いヘッダーデザインで視認性を高めたい場合に適している。

## LanguageStackedHeaderLayout

説明
固定位置のスタック型ヘッダー。ロゴ（レスポンシブ対応）、検索入力フィールド、通知ドロップダウン、言語選択ドロップダウン、アカウントドロップダウンを備える3カラムグリッドレイアウト。

主なユースケース
多言語対応が必要なスタック型ダッシュボード。サイドバーなしで言語切り替え機能を提供したい場合に適している。

## DoubleLineStackedHeaderLayout

説明
2行構成のスタック型ヘッダー。上部にダーク背景（blue-950）のプライマリナビゲーション（ロゴ、検索、通知、アカウント）、下部にセカンダリナビゲーションを配置。

主なユースケース
情報量の多いダッシュボードやポータルサイト。プライマリ機能とセカンダリ機能を視覚的に分離したい場合に適している。
