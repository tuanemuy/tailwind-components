# Dashboard Header Layouts

Infinite admin dashboard header options from nav links to overlay contents and language dropdowns.

## ProjectHeaderLayout

説明
サイドバーを考慮した左オフセット配置の固定ヘッダー。プロジェクト切り替えドロップダウン（ドラッグハンドル付きリスト、複数プロジェクト対応）、3カラムグリッドレイアウトを備える。

主なユースケース
複数のプロジェクトやワークスペースを切り替えながら作業するダッシュボード。Notionのようなプロジェクト管理ツールで、現在のプロジェクトを表示・切り替えしたい場合に適している。

## OverlayHeaderLayout

説明
サイドバーを考慮した左オフセット配置の固定ヘッダー。サイドバートグルボタン、検索入力フィールド（キーボードショートカット付き）、通知ドロップダウン、アカウントドロップダウンを備える3カラムグリッドレイアウト。

主なユースケース
検索機能を重視する管理画面やダッシュボード。コマンドパレット的な検索UIを組み込みたい場合に適している。

## LanguageHeaderLayout

説明
サイドバーを考慮した左オフセット配置の固定ヘッダー。検索入力フィールド、通知ドロップダウン、言語選択ドロップダウン（English等の言語切り替え）、アカウントドロップダウンを備える。

主なユースケース
多言語対応が必要なダッシュボードや管理画面。ユーザーがUIの言語を動的に切り替えられる機能を提供したい場合に適している。

## LogoHeaderLayout

説明
サイドバーを考慮した左オフセット配置の固定ヘッダー。ロゴ（デスクトップでは完全版、モバイルではアイコンのみ）、サイドバートグルボタン、通知・アカウントドロップダウンを備える。

主なユースケース
ブランドロゴを常時表示したいダッシュボード。サイドバーにロゴがない、またはヘッダーにもブランディングを表示したい場合に適している。

## ClassicDashboardHeaderLayout

説明
サイドバーを考慮した左オフセット配置の固定ヘッダー。検索入力フィールド（キーボードショートカット付き）、通知ドロップダウン（タブ付き）、アカウントドロップダウンを備える標準的なダッシュボードヘッダー。

主なユースケース
一般的な管理画面やSaaSダッシュボードのベースレイアウト。検索、通知、アカウント管理といった基本機能を網羅したい場合に適している。

## SimpleHeaderLayoutWithButton

説明
サイドバーを考慮した左オフセット配置の固定ヘッダー。サイドバートグルボタン、テンプレートドロップダウン（通知バッジ付き）、通知ドロップダウン、アカウントドロップダウンを備えるシンプルなレイアウト。

主なユースケース
テンプレートやプリセットの選択機能を提供するダッシュボード。新規作成時のテンプレート選択など、クイックアクションを提供したい場合に適している。

## NavLinksHeaderLayout

説明
サイドバーを考慮した左オフセット配置の固定ヘッダー。サイドバートグルボタン、コラップスメニュー（モバイル用）、通知ドロップダウン、アカウントドロップダウン、ナビゲーションリンクを備える。

主なユースケース
ヘッダー内にセカンダリナビゲーションリンクを配置したいダッシュボード。サイドバーとは別にヘッダーでもページ遷移できるようにしたい場合に適している。

## GraySearchNavbarWithAccountDropdownHeaderLayout

説明
サイドバーを考慮した左オフセット配置のヘッダー（非固定、relative配置）。グレー背景の検索入力フィールド（ピンク系のフォーカスカラー）、通知ドロップダウン、アカウントドロップダウンを備える。

主なユースケース
固定ヘッダーではなくコンテンツと一体化したレイアウトを求めるダッシュボード。検索バーを目立たせつつ、スクロール時にはヘッダーが隠れるデザインにしたい場合に適している。
