# Application Navbars

Tailwind CSS application navbars with menus, icons, and responsive design for seamless navigation.

## ProjectApplicationNavbar

説明
インディゴ色の背景を持つプロジェクト管理向けナビゲーションバー。ロゴ、プロジェクト切り替えドロップダウン、ナビゲーションリンク（Home、Details、Setup、Empty Contents）、アカウントドロップダウン（Billing、Settings、My account、ダークモード切り替え、チーム管理、サインアウト）を含む。レスポンシブ対応でモバイル用のコラップスメニューあり。

主なユースケース
プロジェクト管理ツールやタスク管理アプリケーションのメインナビゲーション。複数プロジェクトを切り替えながら作業するダッシュボードに適している。

## CRMApplicationNavbar

説明
グレー/ダーク背景を持つCRM向けの多機能ナビゲーションバー。ロゴ、中央配置の検索入力フィールド（キーボードショートカット表示付き）、ヘルプドロップダウン（Help Centre、Community、What's New、Documentation、Contact Us等）、通知ドロップダウン（バッジ付き、All/Archivedタブ）、アカウント機能を含む。

主なユースケース
CRM（顧客管理）システムや営業支援ツールのナビゲーション。グローバル検索機能を中心に据えたいデータ量の多いアプリケーション、ヘルプ・サポート機能を充実させたい業務システムに適している。

## WorkspaceApplicationNavbar

説明
グレー背景を持つワークスペース/チームコラボレーション向けナビゲーションバー。ロゴ、通知ドロップダウン（バッジ付き、All/Archivedタブ、アバター・時刻・ユーザー名・アクション説明を含む通知リスト）、各通知のクイックアクションボタン（チェック、アーカイブ等）を備える。

主なユースケース
チームコラボレーションツールやプロジェクト管理ダッシュボードのナビゲーション。チームメンバーの活動やタスクの状態変更をリアルタイムで通知したいアプリケーションに適している。
