# Application Toolbars

Application Toolbars demo showcases a collection of modern, responsive navigation components for web applications, featuring chat activity panels, reports interfaces, and customizable utility bars.

## ChatReportsApplicationToolbar

説明
チャット/レポート分析ツール向けの固定位置ツールバー。サイドバーを考慮した左オフセット配置で、サイドバートグルボタン（モバイル用）、リフレッシュボタン、最終更新時刻・タイムゾーン表示、カレンダードロップダウン（日付範囲選択、月/年のセレクトボックス付きカレンダーUI）を備える。

主なユースケース
チャットサポートシステムのレポート画面、顧客対応の分析ダッシュボード。サイドバーと組み合わせて使うレポート閲覧画面で、期間を指定してデータを絞り込みたい場合に適している。

## ChatActivityApplicationToolbar

説明
チャットアクティビティ監視向けの固定位置ツールバー。中央配置・最大幅制限付きレイアウトで、サイドバートグルボタン（モバイル用）、ピル型タブナビゲーション（Activity、Mentions、Assigned）、カレンダードロップダウン（日付範囲選択）を備える。モバイルではアイコンのみ表示に切り替わるレスポンシブ対応。

主なユースケース
チャット・メッセージングアプリのアクティビティ一覧画面、チームコミュニケーションツールの通知センター。複数のビュー（全アクティビティ、メンション、割り当て済み）を切り替えながら確認したい場合に適している。
