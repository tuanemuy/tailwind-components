# Navbars

Navbars demo showcases a collection of modern, responsive navigation components for web applications, featuring e-commerce layouts, smart home interfaces, and customizable navigation patterns.

## StartupNavbar

説明
スタートアップ/SaaS向けのstickyナビゲーションバー。ロゴ、Get a demo・Log in・Try it freeボタン、コラップスメニュー（モバイル用）を備える。青色のCTAボタンとシンプルなレイアウト。

主なユースケース
SaaSプロダクトやスタートアップのランディングページ、製品紹介サイト。無料トライアルやデモ申し込みへの導線を重視したい場合に適している。

## PaymentWithStepsNavbar

説明
支払い/チェックアウトプロセス向けのナビゲーションバー。ロゴ、アバター付きアカウントドロップダウン、4ステップのプログレスインジケーター（グラデーション表示）を備える。パープル系カラーテーマ。

主なユースケース
ECサイトのチェックアウトフロー、サブスクリプション申し込みプロセス。複数ステップの決済フローで進捗状況を視覚的に表示したい場合に適している。

## ShopNotLoggedInNavbar

説明
ショップ/EC向けのstickyナビゲーションバー（未ログイン状態用）。ロゴ、検索ボタン、ログインリンク、カートアイコン、ナビゲーションリンクを備える。インディゴ系カラーテーマ。

主なユースケース
ECサイトのゲストユーザー向けヘッダー。ログインを促しつつ、商品検索やカート確認ができるようにしたい場合に適している。

## ShopLoggedInNavbar

説明
ショップ/EC向けのstickyナビゲーションバー（ログイン状態用）。ロゴ、検索ボタン、お気に入りボタン、カートアイコン、アカウントドロップダウン、ナビゲーションリンクを備える。インディゴ系カラーテーマ。

主なユースケース
ECサイトのログインユーザー向けヘッダー。お気に入り商品やアカウント管理機能へのアクセスを提供したい場合に適している。

## VideoCallNavbar

説明
ビデオ通話アプリ向けの固定ナビゲーションバー。ロゴアイコン、通知ボタン、AIボタン、参加者管理、設定ボタンを備える12カラムグリッドレイアウト。コンパクトなアイコン中心のデザイン。

主なユースケース
ビデオ会議アプリやオンラインミーティングツールのヘッダー。画面スペースを最小限に抑えつつ、必要な操作ボタンを配置したい場合に適している。

## E-CommerceNavbar

説明
ECサイト向けのナビゲーションバー。ロゴ（レスポンシブ対応）、中央に検索入力フィールド、カートアイコン、メニューリンクを備える。グリーン系カラーテーマ。

主なユースケース
汎用的なECサイトのヘッダー。商品検索を中央に配置し、検索性を重視したオンラインストアに適している。

## SmartHomeNavbar

説明
スマートホーム/IoT向けのナビゲーションバー。白色ロゴ（ダーク背景用）、シンプルなレイアウト、レスポンシブ対応を備える。ダークテーマに最適化されたデザイン。

主なユースケース
スマートホーム管理アプリやIoTダッシュボードのヘッダー。ダークテーマを基本としたインターフェースに適している。

## MarketplaceNavbar

説明
マーケットプレイス向けの2段構成ナビゲーションバー。トップバー（配送先住所指定、言語/通貨選択、ヘルプリンク、モバイルアプリリンク、ダークモードトグル）とメインナビゲーションを持つ。

主なユースケース
グローバル展開するマーケットプレイス、多言語・多通貨対応が必要なECサイト。配送先や言語設定など、購入前に確認すべき情報を常時表示したい場合に適している。
