# Navs

Experience the Navs page with user-centric designs that facilitate easy transitions between profiles, settings, and core areas.

## コンポーネント一覧

### BottomBorderedNav

説明: 下部にボーダーラインを持つシンプルなタブナビゲーション。アクティブなタブには下線インジケーターが表示され、ホバー時に背景色が変化。

主なユースケース: ページ内セクション切り替え、ECサイトのストア概要・支払いタブ、シンプルなサブナビゲーション

### HorizontallyScrollableNavGroup

説明: 横スクロール可能なPillボタン（丸角）形式のナビゲーション。アイコン付きカテゴリボタンが横に並び、右端にフェードグラデーションでスクロールを示唆。

主なユースケース: 製品カテゴリフィルター、ECサイトのカテゴリ選択、タグベースの絞り込みUI

### UserProfileNav

説明: アイコン付きの横スクロール可能なナビゲーション。各タブにSVGアイコンとラベルを配置し、アクティブ状態で下部に短いインジケーターバーを表示。自動センタリング機能付き。

主なユースケース: ユーザープロフィールページのタブ（プロフィール・チーム・ファイル・接続）、アカウント管理のサブナビ

### WhitePilledNav

説明: 白背景のPill形式タブナビゲーション。アクティブなタブはカード風の白背景・ボーダー・影を持ち、非アクティブタブは透明。自動センタリングとスナップスクロール対応。

主なユースケース: アカウント設定ページのナビゲーション、ダッシュボードのセクション切り替え、管理画面のタブメニュー
