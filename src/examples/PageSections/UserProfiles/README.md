# User Profiles

Admin user profile navigation, powered by informative modals, avatars come alive with active status indicators.

## コンポーネント一覧

### DescriptionWithAvatarInDarkBackground

説明: ダーク背景のユーザープロフィールヘッダー。大きなアバター画像にProバッジ、ユーザー名、役職・電話・メールの詳細リスト、Contactボタンを配置したリッチなプロフィール表示。

主なユースケース: 管理画面のユーザー詳細ページ、チームメンバーのプロフィール画面、クリエイターやエキスパートの紹介ページ

### BrandingLogoWithDatepickerAndClipboard

説明: ブランディングストア用のプロフィールカード。サービスロゴ（GitLab等）、ストア名、クリップボードコピー可能なリファラルURL、日付ピッカー付きコミッション概要、統計指標（紹介数/完了数/売上）、Make payoutsボタンを配置。

主なユースケース: アフィリエイトダッシュボード、リファラルプログラム管理画面、コミッション収益レポートページ

### UserProfileCardWithSetStatusModal

説明: SVGバナー背景付きのユーザープロフィールカード。アバター画像、Set Statusボタン（絵文字ステータス選択モーダルを開く）、名前とユーザー名、タブナビゲーション（My Profile/Teams/Files/Connections）、Editボタンを配置。モーダルにはオフライン/おやすみモード/スケジュール設定機能あり。

主なユースケース: チームコラボレーションツールのプロフィール、チャットアプリのユーザー設定、社内SNSのプロフィールページ

### CustomerDetails

説明: プロジェクト詳細用のコンパクトなヘッダー。ブランドアイコン（Dropbox等）、プロジェクト名、開始日/締切日、Moreドロップダウン（Edit/Archive/Delete）を白背景で配置。

主なユースケース: プロジェクト詳細ページのヘッダー、タスク管理の詳細表示、クライアント案件の概要表示

### SimpleUserProfile

説明: シンプルなミニプロフィールコンポーネント。丸型アバター、ユーザー名、メールアドレス、Logoutリンクを横並びで配置したコンパクトなデザイン。

主なユースケース: サイドバーのユーザー情報表示、ヘッダーのログインユーザー表示、設定画面のアカウント概要

### PaymentAvatarDetailsWithButtonGroup

説明: 支払いリクエスト用のユーザーヘッダー。イニシャルアバター（受信アイコン付き）、ユーザー名、Request again/Cancel requestボタンを配置。

主なユースケース: 支払いリクエスト詳細ページ、送金依頼の管理画面、請求管理のユーザー表示

### UserProfileCardWithActiveStatusModal

説明: カラフルなSVGバナー背景付きのユーザープロフィールカード。アバター画像、Offline状態インジケーター（赤色）、Follow/Unfollowトグルボタン、タブナビゲーション、ステータス設定モーダルを配置。

主なユースケース: SNS風のユーザープロフィール、コミュニティメンバーの詳細ページ、チームメンバー一覧からの詳細表示

### PaymentTransactionDetails

説明: 支払い取引用のコンパクトなユーザー表示行。イニシャルアバター（送金アイコン付き）、ユーザー名、ステータスバッジ（Succeeded）、金額を横並びで配置。

主なユースケース: 取引履歴のリスト項目、送金完了一覧、支払い明細の相手先表示

### ProjectDetails

説明: プロジェクト詳細用のダーク背景対応ヘッダー。ブランドアイコン（Dropbox等）、クライアント名、プロジェクトタイトル、開始日/締切日、Moreドロップダウンを配置。

主なユースケース: プロジェクト管理画面のヘッダー、案件詳細ページ、タスクボードの上部表示
