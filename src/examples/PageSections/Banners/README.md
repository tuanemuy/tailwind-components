# Banners

Discover customizable banners for alerts, promoting important messages and enhancing customer engagement effectively.

## コンポーネント一覧

### GiftBanner

説明: 絵文字付きのグラデーション背景バナー。プロモーションコードや無料プレゼントキャンペーンの告知に特化したデザインで、閉じるボタン付き。

主なユースケース: キャンペーン告知、クーポンコードの訴求、期間限定オファーの表示、購入特典の案内

### LoginRegisterBanner

説明: グラデーション背景のアラートバナー。送料無料などの特典メッセージとともにログイン・会員登録へのリンクを表示する横長のバナー。

主なユースケース: 未ログインユーザーへの会員登録促進、送料無料などの特典告知、ECサイトのヘッダー下部バナー

### MiniNewsAnnouncementCard

説明: 画像付きの小型お知らせカード。サムネイル画像、ラベル（New等）、タイトルを縦に配置し、ホバー時に矢印アイコンが表示されるインタラクティブなカード。

主なユースケース: 新機能・新サービスの告知、ニュースのハイライト表示、サイドバーやウィジェットエリアでのお知らせ表示

### RatePurchaseInFloatingCard

説明: 画面下部に固定表示されるフローティングカード。購入済み商品をカルーセルで表示し、各商品の評価を促すUI。閉じるボタンと「後で評価する」オプション付き。

主なユースケース: 購入後のレビュー依頼、商品評価の促進、ECサイトでの顧客フィードバック収集

### SimpleSliderInGrayBackgroundBanner

説明: グレー背景の自動再生カルーセルバナー。アイコン付きの短いメッセージを複数スライドで切り替え表示する。

主なユースケース: 商品ページでのトレンド情報表示、閲覧数・販売数の表示、サイズ情報やその他の補足情報の表示
