# Quick Actions

Explore colorful and vibrant quick action card blocks.

## コンポーネント一覧

### ButtonLinks

説明: 画面下部に固定表示されるフローティングアクションバー。選択数バッジ、メール送信・削除などのアクションリンク、閉じるボタンを黒背景の丸角バーに配置。複数アイテム選択時のバルク操作UI。

主なユースケース: テーブルやリストの複数選択時アクション、一括操作のフローティングメニュー、選択モードのコントロールバー

### ButtonLinksInDardBackground

説明: ダーク背景のフローティングアクションバー。Delete（赤）・Cancel・Save changes（緑）のリンクと閉じるボタンを配置。編集モード時の保存・キャンセル操作用。

主なユースケース: フォーム編集時の保存確認バー、変更操作のフローティングコントロール、編集モードのアクションバー

### CardLinkWithIconGridView

説明: 3列グリッドのアカウント設定カード。各カードにアイコン、タイトル、説明を配置し、ホバー時にインディゴ色のボーダーと影が表示。注意が必要な項目には赤いバッジ表示。

主なユースケース: アカウント設定ダッシュボード、ECサイトのマイページメニュー、ユーザー設定のカテゴリ選択

### CardsWithLargeSoftIcons

説明: ソフトカラー背景の大きなアイコンを持つカードグリッド。Blank page、Whiteboard、Diagram、Empathy Mapsなどのテンプレート選択用。「All templates」リンクカードも含む。

主なユースケース: テンプレート選択画面、新規作成ウィザード、ホワイトボードアプリのスタート画面

### ContactsList

説明: アイコン付きの連絡先リスト。Chat、Phone、Email、Feedbackの各項目にアクションリンク（Open chat、電話番号、Send us email等）を配置。

主なユースケース: ヘルプセンターの連絡先一覧、サポートページのコンタクト情報、フッターの連絡先ウィジェット

### DarkCardStyleButton

説明: ダーク背景（黒/白）のコンパクトなアクションボタン。白い丸アイコンとラベルを縦に配置。新規タスク追加などの単一アクション用。

主なユースケース: カンバンボードの新規タスク追加、クイック作成ボタン、コンパクトなCTAボタン

### GridListLinkWithTitleAndGrayBackgroundIcon

説明: グレー背景の丸アイコン、タイトル、右矢印を横並びに配置したグリッドリンク。Show PIN、Card details、Freeze cardなどのカード管理アクション用。

主なユースケース: クレジットカード管理のクイックアクション、設定メニュー、モバイルバンキングアプリの操作メニュー

### GroupHoverActions

説明: 日付でグループ化されたトランザクションリスト。各項目にアバター/ロゴ、名前、送受信アイコン、時刻、金額、残高を表示。受信は緑、送信はデフォルト色で区別。

主なユースケース: 銀行アプリの取引履歴、支払いアプリのトランザクション一覧、決済履歴ページ

### ItemSummaryLinkCard

説明: ボーダー付きカード形式のリンク。左にアイコン、中央にタイトルと説明文、右に矢印を配置。注文追跡などの説明付きアクションに適している。

主なユースケース: FAQのクイックリンク、ヘルプトピックカード、ECサイトの注文状況確認リンク

### ListLinkWithTitleAndAvatar

説明: 国旗画像をアバターに使用したリスト形式のリンク。通貨名と銀行情報（Sort code、IBAN等）を表示。通貨・国選択用のUI。

主なユースケース: 通貨選択メニュー、国際送金の送金先選択、マルチカレンシー口座の選択

### ListLinkWithTitleAndGrayBackgroundIcon

説明: グレー背景の丸アイコン、タイトル、サブテキスト、右矢印を配置したリストリンク。未払い件数などのバッジ表示も可能。支払い管理のナビゲーション用。

主なユースケース: 支払い管理メニュー、スケジュール転送・自動引落の管理、決済設定のナビゲーション

### SimpleBorderedCardWithIcon

説明: グレー背景の丸アイコンとタイトル・サブテキストを横並びに配置したシンプルなボーダーカード。「Open a new balance」などの単一アクション用。

主なユースケース: 新規作成アクション、残高開設ボタン、シンプルなCTAカード

### SimpleCardsWithIcons

説明: 横スクロール可能なカード群。各カードにアイコン、タイトル、説明を縦に配置。Product、Discount、Collection、Get paid、Preline productsなどのアクション選択用。

主なユースケース: ECダッシュボードのクイックアクション、管理画面の機能選択、オンボーディングのオプション選択

### UserDetailsWithMoreButtonsOnHover

説明: ユーザーリストでホバー時にアクションボタン群（表示/非表示、オプション、削除）が表示されるUI。アバター、名前、役職/メールを表示。オーガナイザーには緑のチェックバッジ。

主なユースケース: 会議参加者の管理、チームメンバーリスト、イベント出席者の編集
