# Title Bars

Discover a collection of customizable Title Bars, perfect for enhancing user navigation and interface design.

## コンポーネント一覧

### CalendarHeader

説明: カレンダーアプリ用のタイトルバー。サイドバートグル、表示切替ボタン（日/週/月/年）、日付表示、前後ナビゲーション、Todayボタン、イベント追加ボタンを配置。

主なユースケース: カレンダーアプリのヘッダー、スケジュール管理画面、予定表示ページのナビゲーション

### SubtitleAndNavigationButtons

説明: 商品詳細ページ用のタイトルバー。ブランドリンク（サブタイトル）、商品名、Prev/Nextナビゲーションボタンを横並びで配置。

主なユースケース: ECサイトの商品詳細ページ、記事一覧のナビゲーション、コンテンツ間の移動UI

### ProfileActionButton

説明: プロフィールページ用のタイトルバー。サービスロゴ（GitLab等）、店舗名、クリップボードコピー可能なリファラルURL、アクションボタンを配置。

主なユースケース: リファラルプログラムのダッシュボード、ストア管理画面、アフィリエイトリンク共有ページ

### WorkspaceKanbanGroupHeading

説明: ワークスペース用カンバンボードのカラムヘッダー。ステータスバッジ（カラーインジケーター付き）、アイテム数、ドロップダウンメニュー（カラーピッカー、非表示、削除）、タスク追加ボタンを配置。

主なユースケース: プロジェクト管理のカンバンボード、タスク管理システム、ワークフロー管理画面

### SelectAndShowingAmountTitleBar

説明: レビューセクション用のタイトルバー。セクションタイトル、表示件数（例：4 of 12）、ソート選択ドロップダウン（Relevant/Helpful/Newest等）を配置。

主なユースケース: レビュー一覧ページ、コメントセクション、フィードバック一覧の並べ替え

### DatepickerAndActionButtonTitleBar

説明: 概要セクション用のタイトルバー。セクションタイトル、カレンダードロップダウン（月/年選択付き）、アクションボタン（Add project）を配置。

主なユースケース: ダッシュボードの概要セクション、プロジェクト一覧ページ、日付フィルター付きレポート

### AnalyticsSurveyPageTitleBar

説明: アンケート詳細ページ用のシンプルなタイトルバー。戻るリンクとページタイトルを横並びで配置。

主なユースケース: アンケート結果ページ、詳細画面への遷移後のナビゲーション、サブページのヘッダー

### CRMKanbanGroupHeading

説明: CRM用カンバンボードのカラムヘッダー。ステータスバッジ（カラーインジケーター付き）、アイテム数、ドロップダウンメニュー（カラーピッカー、非表示、削除）、リスト追加ボタンを配置。

主なユースケース: CRMのパイプライン管理、営業案件管理ボード、顧客ステータス管理

### ButtonAndDropdownTitleBar

説明: 注文詳細ページ用のタイトルバー。注文番号タイトル、エクスポートボタン、Moreドロップダウン（Duplicate/Cancel/Archive/Edit/Print/Delete）を配置。

主なユースケース: 注文詳細ページ、取引履歴の詳細画面、レコード操作UI

### DraftDetailsNavbar

説明: 下書き詳細用のメタデータバー。作成日時と更新日時を横並びで表示するシンプルなナビゲーションバー。

主なユースケース: ドキュメント編集画面のメタ情報、下書き管理ページ、コンテンツの更新履歴表示

### TitleDropdownWithActionButtons

説明: プロジェクトヘッダー用のタイトルバー。プロジェクト名ドロップダウン（設定/管理画面/リンクコピー/退出）、アクションボタン群（Project/Invite/Members/お気に入り）を配置。

主なユースケース: チームプロジェクトのダッシュボード、コラボレーションツールのヘッダー、ワークスペース管理画面

### E-CommerceKanbanGroupHeading

説明: EC用カンバンボードのシンプルなカラムヘッダー。ステータス名、アイテム数、タスク追加ボタンのみを配置したミニマルデザイン。

主なユースケース: EC注文管理のカンバンボード、配送ステータス管理、簡易タスク管理

### DashboardKanbanGroupHeading

説明: ダッシュボード用カンバンボードのカラムヘッダー。ステータスドロップダウン（カラーピッカー付き）、アイテム数、タスク追加ボタンを配置。

主なユースケース: 汎用ダッシュボードのカンバンビュー、タスク管理システム、ステータス管理ボード

### DropdownOptionsTitleBar

説明: チャート/レポート用のタイトルバー。セクションタイトル、ダウンロードドロップダウン（Excel/Word切替、オプションチェックボックス）、その他オプションドロップダウン（共有/全画面/連携）を配置。

主なユースケース: 分析ダッシュボードのチャートヘッダー、レポート出力機能、データ可視化コンポーネント

### TitleAndSelect

説明: 注文履歴用のタイトルバー。セクションタイトル、注文件数、年フィルター選択ドロップダウンを配置。

主なユースケース: 注文履歴ページ、年別フィルター付きリスト、履歴データの絞り込み

### AnalyticsVisitorsPageTitleBar

説明: 訪問者分析用の高機能タイトルバー。タイトル、現在の訪問者数（ライブインジケーター）、チャート表示トグルスイッチ、日付範囲ピッカー（2つのカレンダー並列表示）を配置。

主なユースケース: アクセス解析ダッシュボード、リアルタイム訪問者モニター、期間指定分析ページ

### TitleWithTagsAndDropdowns

説明: チャット/会話用のタイトルバー。会話タイトル、タグドロップダウン群（各タグに編集/削除/全会話表示オプション）を配置。

主なユースケース: チャットスレッドのヘッダー、チケット管理画面、タグ付きコンテンツの管理

### LargeSubtitleAndActionButtons

説明: 残高表示用の大型タイトルバー。通貨フラグアイコン、金額表示、サブタイトル、アクションボタン群（Add/Send/Request）、Moreドロップダウン（明細/引落/自動変換等）、カード情報ボタンを配置。

主なユースケース: 決済アプリの残高ページ、ウォレット管理画面、送金サービスのダッシュボード

### TitleWithStorageProgressInfo

説明: ストレージ情報付きのタイトルバー。セクションタイトル、使用量表示テキスト、プログレスバー、ストレージ追加リンクを配置。

主なユースケース: クラウドストレージの管理画面、ファイル管理ページ、容量モニタリングUI

### ClipboardTitleWithButtonGroup

説明: トランザクション詳細用のタイトルバー。取引ID（クリップボードコピー機能付き）、アクションボタン群（Repeat transfer/Cancel transfer）を配置。

主なユースケース: 送金詳細ページ、取引履歴の詳細画面、決済キャンセル/再実行UI

### InputAndButtonTitleBar

説明: プロジェクト一覧用のタイトルバー。セクションタイトル、検索入力欄、追加ボタンを配置。

主なユースケース: プロジェクト一覧ページ、検索機能付きリストヘッダー、アイテム追加機能付きセクション
