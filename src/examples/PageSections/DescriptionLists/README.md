# Description Lists

Explore detailed and clear admin user description lists with vibrant icons, avatars, badges and links.

## コンポーネント一覧

### DescriptionListWithPointerIcon

説明: アイコン付きラベルと矢印、値を横に並べたコンパクトなリスト形式。プロジェクトや作業の統計情報を簡潔に表示する。

主なユースケース: プロジェクト統計の表示、作業時間・日数・コストのサマリー、ダッシュボードのメトリクス表示

### ListGroupWithGroupLinks

説明: カード形式の複合リストグループ。顧客情報、注文数、連絡先、配送先・請求先住所、決済情報などを複数セクションに分けて表示。各セクションに編集リンク付き。

主なユースケース: ECサイトの顧客詳細パネル、注文詳細のサイドバー、顧客管理画面の情報表示

### ListWithDashedDividerAndClipboard

説明: 破線区切りの2カラムリスト。各項目の値にクリップボードコピーボタンが付属し、ワンクリックでテキストをコピー可能。

主なユースケース: ユーザー詳細情報の表示、連絡先情報の参照、コピー操作が頻繁に必要な管理画面

### ListWithDottedDividerAndClipboardAndCollapse

説明: 点線区切りの2カラムリストで、クリップボードコピー機能と折りたたみ（Show more/Show less）機能を併せ持つ。長い仕様リストを省スペースで表示可能。

主なユースケース: 商品仕様の表示、製品詳細ページ、技術仕様やスペック一覧

### SimpleWithIcons

説明: アイコン付きの縦並びリスト。組織名、所在地、タイムゾーン、メール、外部リンクなどを各行にアイコンとともに表示。リンク項目はホバー時にアンダーライン表示。

主なユースケース: ユーザープロフィールのサイドバー、会社情報の表示、連絡先・SNSリンクの一覧

### UserInfoDescriptionList

説明: 3カラムグリッドの区切り線付きリスト。ユーザーID（リンク）、電話番号（国旗付き）、住所、ステータスバッジなど多様な形式の情報を表示可能。

主なユースケース: 管理画面のユーザー詳細、アカウント情報ページ、顧客データの閲覧画面

### UserInfoDescriptionStretchedBetweenList

説明: シンプルな2カラムの定義リスト（dl/dt/dd）。ラベルを左、値を右寄せで配置。区切り線なしのミニマルなデザイン。

主なユースケース: コンパクトなプロフィール情報表示、カード内の詳細情報、ポップオーバーやツールチップ内の情報

### UserProfileCardWithDescriptionList

説明: カラフルなSVGヘッダー背景とアバター画像を持つプロフィールカード。2カラムグリッドで詳細情報を表示し、フッターにアクションボタン（削除・キャンセル・追加）を配置。

主なユースケース: ユーザー追加・編集モーダル、会員情報の確認画面、プロフィール詳細ページ
