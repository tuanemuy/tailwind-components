# Set Up Flows

Tailwind CSS setup flow sections with steps, progress indicators, and forms for guided user onboarding.

## コンポーネント一覧

### SetUpFlowListGroup

説明: プログレスバー付きのリスト形式セットアップカード。4分割のプログレスバーで完了度を可視化し、各タスクをグレー背景のリストアイテムとして表示。完了済みはチェックマーク、未完了はアイコンを表示。

主なユースケース: ダッシュボードのプロフィール完成度ウィジェット、オンボーディング進捗カード、アカウント設定の完了チェック

### SetUpFlowTimeline

説明: タイムライン形式の進捗インジケーター付きセットアップカード。縦のラインで接続されたステップを表示し、完了済みはティールのチェックマーク、未完了はアイコンを表示。各ステップにアクションボタン付き。

主なユースケース: プロフィールセットアップウィザード、新規ユーザーのオンボーディング、マイルストーン形式のタスク管理

### SetUpWithButtons

説明: 番号付きのコンパクトな縦リスト。完了済みはチェックマークと取り消し線、未完了は番号（破線円）で表示。各ステップに小型のアクションボタン（Download、Enable、Connect等）。

主なユースケース: ウィジェット内のクイックセットアップ、サイドバーのタスクリスト、コンパクトなオンボーディングチェックリスト

### SidebarAndAccordionSetUp

説明: 2カラムレイアウトの複合セットアップUI。左サイドバーにタイムライン形式の進捗インジケーター、右にアコーディオン形式の詳細セットアップステップ。アプリ連携（Slack、Notion、Gmail等）、プロジェクト追跡、チーム招待などの複数セクションを展開可能。

主なユースケース: 初回セットアップウィザード、SaaS製品のオンボーディングページ、ワークスペース設定のガイド付きフロー
