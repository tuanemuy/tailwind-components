# Notification Systems

Table-organized admin notification system.

## コンポーネント一覧

### TableFormatListOfCheckboxes

説明: 通知設定のマトリックス形式UI。Email・Desktop・Mobileの3つの通知チャンネルを列に、各通知項目を行に配置したグリッドレイアウト。セクションヘッダー（General、API、Tasks）で項目を分類し、各セクションに「Toggle all」リンクを配置。レスポンシブ対応でモバイルでは2列表示。

主なユースケース: アカウント設定の通知管理、アプリの通知設定画面、チームワークスペースの通知プリファレンス
