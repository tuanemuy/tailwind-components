2020

Initial concept for Preline begins, inspired by the need for a streamlined UI component library built on Tailwind CSS.

2021

Q1: Development of the first version of Preline, featuring a small set of core components designed for responsive web design.

Q3: Launch of Preline as an open-source project, introducing over 500 components and templates, along with detailed documentation.

2024

Q2: Expansion of the library to include 750+ components, integration with frameworks like React and Vue, and release of a Figma design system for seamless workflows.

Q4: Preline establishes a strong developer community, introduces custom web design services, and continues to enhance its offerings with new features and updates.
