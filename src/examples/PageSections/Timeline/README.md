# Timeline (Feeds)

Navigate your admin timeline with ease, and collapsible content revealing past entries for a clean view.

## コンポーネント一覧

### OrderStatus

説明: 縦型ステップタイムラインで注文の進捗状況を表示。各ステップにチェックマークアイコン、ステータス名、日時を配置し、完了済みステップは緑色、進行中は点滅アニメーション、未完了はグレーで視覚的に区別。

主なユースケース: 注文追跡ページ、配送状況確認画面、ECサイトのマイページ注文詳細

### Activity

説明: タスクやファイルの操作履歴を表示するアクティビティフィード。各アクティビティにアイコン、操作内容、タイムスタンプを縦に配置し、コネクターラインで接続。

主なユースケース: プロジェクト管理のアクティビティログ、ファイル操作履歴、タスク変更履歴

### PaymentTransactionsUpdates

説明: 支払い処理の進行状況を表示するステップタイムライン。各ステップにチェックマークアイコン、処理名、日時を配置し、完了済みは紫色のチェック、未完了はグレーの円で表示。

主なユースケース: 決済処理状況の確認画面、振込進捗ページ、支払いトラッキング

### CommentWithImages

説明: チャット形式のコメントタイムラインで画像添付機能付き。各コメントにユーザーアバター、名前、投稿日時、コメント本文、添付画像を表示し、ホバー時に絵文字リアクションと返信ボタンを表示。

主なユースケース: チームコラボレーションのコメント機能、タスクへのコメントスレッド、ファイルレビューのフィードバック

### CollapsableShowMoreButton

説明: 折りたたみ可能なアクティビティフィード。各アクティビティにアバター、ユーザー名、アクション内容、ファイルプレビュー（サムネイル・ファイル名・サイズ）を表示し、「Show more」ボタンで過去のアクティビティを展開表示。

主なユースケース: ダッシュボードのアクティビティ概要、通知一覧の省スペース表示、長いフィードの折りたたみ表示

### HistoryTimelineWithImages

説明: 会社や組織の歴史を年単位で表示するタイムライン。左側に年ラベル、右側に画像と説明文を2カラムレイアウトで配置し、重要なマイルストーンを視覚的に表現。

主なユースケース: 会社沿革ページ、組織の歴史紹介、プロジェクトの経緯説明

### ActivityTimeline

説明: プロジェクトのアクティビティを日付別にグループ化して表示するタイムライン。各エントリにブランドロゴ/アバター、アクション説明、タイムスタンプを配置し、日付ヘッダーで区切りを表示。

主なユースケース: プロジェクト活動ログ、インテグレーション履歴、外部サービス連携の操作履歴

### ReceiptTimeline

説明: 支払い明細をタイムライン形式で表示するレシートコンポーネント。各項目に金額と説明を配置し、手数料、税金、合計金額を明確に区分して表示。

主なユースケース: 支払い明細ページ、手数料内訳の表示、取引レシートの詳細画面

### ChatActivity

説明: サポートチケットのアクティビティをチャット形式で表示するタイムライン。各エントリにアバター、ユーザー名、アクション内容、タイムスタンプを配置し、Telegram等の外部サービス連携も表示可能。

主なユースケース: カスタマーサポートのチケット履歴、ヘルプデスクの対応ログ、問い合わせ対応の経緯表示

### HistoryTimeline

説明: ダークテーマのカルーセル形式で会社のマイルストーンを表示するタイムライン。スライダーで年を切り替え、各年の画像と説明を大きく表示。ナビゲーションボタンと年ラベルで操作。

主なユースケース: コーポレートサイトの沿革セクション、ブランドストーリーページ、会社紹介のインタラクティブ年表
