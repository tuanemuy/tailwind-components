# Invoice

Discover Invoice UI blocks designed for billing, payment summaries, and downloadable receipts—perfect for SaaS, services, and e-commerce platforms..

## コンポーネント一覧

### ListingInvoice

説明: 請求書リスト項目のカード形式コンポーネント。顧客名、請求書番号、日付、金額のサマリーを表示し、折りたたみで詳細（請求先・送信元住所・銀行情報）を展開可能。ステータスバッジ（Overdue等）、再送・キャンセルボタン、ダウンロード・アーカイブ・削除のアクションを備える。

主なユースケース: 請求書管理画面のリスト表示、未払い請求書の一覧、SaaS管理ダッシュボードの請求履歴
