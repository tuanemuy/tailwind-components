# List Group

Discover admin list group options with user-friendly tabs, colorful icons, and friendly avatars.

## コンポーネント一覧

### AttachedFilesAndAvatarGroup

説明: カード形式の複合リスト。予定項目（時間・タイトル・カラーバー）、添付ファイルアイコン（Excel、Notion等）、参加者アバターグループを表示。会議やイベントの詳細表示に適している。

主なユースケース: カレンダーイベントの詳細表示、会議情報カード、タスク詳細のサイドパネル

### BrandingLogosAndIcons

説明: ブランドロゴ付きプロジェクトリスト。各項目にサービスロゴ、タイトル、更新日時、コピー・アーカイブボタングループを配置。ツールチップ付き。

主なユースケース: 連携サービスのプロジェクト一覧、外部ツール統合の管理画面、最近のファイル一覧

### BrandingLogosWithButtons

説明: ブランドロゴ（Gmail、Notion、Word等）と説明文、Connectボタンを横並びに配置したリスト。外部サービス連携のUI。

主なユースケース: サードパーティ連携設定、インポートオプションの選択、アカウント連携画面

### CardListWithBrandingLogosAndButtons

説明: カード形式のリストで、ブランドロゴ、組織名とコラボレーション情報、Leaveボタンを配置。組織やプロジェクトへの参加状況を表示。

主なユースケース: 参加中の組織一覧、コラボレーションの管理、チーム脱退の確認画面

### ColorfulIconsInGroupHover

説明: カラフルなアイコン（青・紫・シアン・藍）付きのリンクリスト。各項目にラベルと矢印アイコンを配置し、ホバー時に背景色が変化。

主なユースケース: ヘルプメニュー、リソースリンク一覧、サポートオプションの表示

### DownloadAppsListGroup

説明: アプリダウンロードリスト。各プラットフォーム（MacOS、App Store、Google Play）のアイコン、名前、評価（星）、レビュー数を表示。

主なユースケース: アプリダウンロードページ、プラットフォーム選択画面、アプリ紹介セクション

### IconAndButtonListWithDashedDivider

説明: 破線区切りのタスクリスト。アイコン、タスク名、Done/Startボタンを配置。完了済みタスクはボタンが無効化される。

主なユースケース: オンボーディングチェックリスト、セットアップウィザード、タスク進捗の表示

### ListGroupWithLegendIndicators

説明: 曜日別リストで、各項目に会話数とレベルインジケーター（Low/Medium/High）をドット表示で可視化。色分けで優先度や負荷を表現。

主なユースケース: 週間アクティビティレポート、サポート負荷の可視化、スケジュールの混雑度表示

### PendingTeamInvitationsList

説明: チーム招待リスト。アバター、名前、メールアドレス、ステータス（Accepted/Requested）をアイコンとともに表示。

主なユースケース: 招待管理画面、チームメンバーの承認状況、ペンディング招待の一覧

### TransparentBackgroundWithIconAndTextsInListView

説明: ガラスモーフィズムスタイル（半透明・ぼかし）の暗い背景カードに、アイコン付きデバイスリストを表示。各項目にデバイス名、モデル名、消費電力（kWh）を配置。スクロール可能。

主なユースケース: スマートホームダッシュボード、エネルギー消費モニター、IoTデバイス管理画面
