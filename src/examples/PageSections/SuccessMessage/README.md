# Success Message

Display clear and concise success messages to inform users of completed actions or achieved milestones.

## コンポーネント一覧

### PaymentShareLink

説明: 支払いリクエスト送信完了後の共有リンクUI。紫色のチェックマークアイコン、タイトル、説明文を中央配置し、共有URL入力欄（Openリンク付き）、コピー・新規リクエストボタン、Doneボタンを縦に配置。

主なユースケース: 支払いリクエスト完了画面、共有リンク生成後の確認画面、決済リンクの共有ページ

### PaymentSuccessMessage

説明: シンプルな送金完了の成功メッセージ。紫色のチェックマークアイコン、見出し、説明文を中央配置し、「転送を管理」と「追加送金」の2つのアクションボタンを横並びで配置。

主なユースケース: 送金完了確認画面、支払い処理成功ページ、入金スケジュール完了通知
