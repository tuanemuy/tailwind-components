# Galleries

Explore Galleries UI blocks featuring image grids, lightboxes, and media sliders—ideal for portfolios, product showcases, and visual storytelling.

## コンポーネント一覧

### SmartHomeGallery

説明: 無限ループのカルーセル形式ギャラリー。ガラスモーフィズム（半透明・ぼかし）スタイルのカードに画像を表示し、各スライドに「Live」バッジとデバイス名ラベルを配置。左右のナビゲーションボタンで操作可能。

主なユースケース: スマートホームアプリの監視カメラ映像表示、IoTデバイスのライブフィード一覧、セキュリティカメラのダッシュボード
