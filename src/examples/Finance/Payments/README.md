# Payments

Experience detailed payment lists, grouped by cards, empowered by modals, and controlled by intuitive button groups.

## ManageCardsWithSavedPaymentOptions

説明: ヘッダー付きのカード管理リストコンポーネント。登録済み決済カード（Visa、Mastercardなど）を一覧表示し、カードブランドロゴ、マスキングされたカード番号、カード種類（Debit）、有効期限を表示する。デフォルトカードの設定・変更ボタン、編集ボタンを備え、有効期限切れカードも赤色テキストで識別可能。

主なユースケース: アカウント設定画面での決済カード管理、複数カードの一元管理UI、デフォルト決済手段の変更、カード情報の編集・更新

## PaymentCards

説明: 通貨残高を表示するコンパクトなカードコンポーネント。国旗アイコン（USD、GBPなど）と残高金額、通貨コードを表示する。関連付けられたカード番号の末尾4桁も表示可能。クリックで残高詳細ページへ遷移できるリンクカード形式。

主なユースケース: マルチカレンシーウォレットでの残高一覧表示、外貨口座の残高確認、決済アプリのホーム画面での残高サマリー

## PaymentDetailsCard

説明: 選択された決済方法を表示するコンパクトなボタン型カード。カードブランドロゴ、カード種類、マスキングされたカード番号、有効期限を横並びで表示し、「Change」リンクで決済方法変更モーダルを開く。

主なユースケース: チェックアウト画面での決済方法確認・変更、注文確認画面での支払い方法表示、定期購読の支払い方法表示

## PaymentListGroupWithDescriptionList

説明: 決済プロバイダーとの連携オプションを表示するリストコンポーネント。PayPal、Stripeなどのサービスロゴと、各サービスの特徴（処理時間、手数料情報など）を箇条書きで表示。各サービスに「Connect」ボタンを配置。

主なユースケース: 出金方法の選択画面、決済プロバイダー連携設定、売上受け取り方法の設定、外部サービス連携オプションの提示

## SavedPaymentOptions

説明: 保存された決済オプションのシンプルなリストコンポーネント。ヘッダーなしのリスト形式で、カードブランドロゴ、カード番号、種類、有効期限、デフォルト設定ボタン、編集ボタンを表示する。

主なユースケース: 決済方法選択画面でのカードリスト、シンプルな決済カード一覧表示、設定画面内の決済方法セクション

## SavedPaymentOptionsWithModalsInCard

説明: 決済方法管理の完全なUIセット。タイトル・説明文付きのカード内に保存済み決済カードをリスト表示し、編集・追加用のモーダルダイアログも統合されている。カードの追加、編集、デフォルト設定の変更がすべてこのコンポーネント内で完結する。

主なユースケース: 設定ページ内の決済方法管理セクション、独立した決済方法管理ページ、カード追加・編集機能が必要なシナリオ
