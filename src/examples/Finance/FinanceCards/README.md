# Finance Cards

Finance Cards present key financial details, metrics, or summaries in a clean and visually accessible format.

## PaymentGradientCards

説明: クレジットカードやデビットカードを視覚的に再現したカードコンポーネント。ICチップ、NFC/タップ対応アイコン、マスキングされたカード番号、通貨、有効期限、カードブランドロゴ（Mastercard）を表示する。ダーク背景とライト背景の2種類のカラーバリエーションがあり、有効期限切れの状態も赤色テキストで表現できる。グリッドレイアウトで複数カードを並べて表示可能。

主なユースケース: ウォレットアプリやダッシュボードでの登録済み決済カード一覧表示、カード選択UI、決済手段管理画面、カード詳細ページへのナビゲーション、有効期限切れカードの視覚的警告表示
