# Kanban Boards Cards

Explore various card UI options, offering flexible and customizable designs for visually managing tasks and projects.

## CRMKanbanBoardsCard

説明: CRM（顧客関係管理）向けのカンバンカード。ヘッダーに会社ロゴとタイトルを表示し、本体に合計時間・日数・プロジェクトコストのサマリー情報と担当者アバターを表示する。ホバー時の視覚フィードバックとモーダル連携に対応。

主なユースケース: 営業パイプライン管理、顧客案件のステータス管理、クライアントごとのプロジェクト進捗追跡、商談管理ボード

## BadgesAndProgressbarKanbanBoardsCard

説明: イシュー・タスク管理向けのカンバンカード。タイトル、複数担当者のアバター、優先度バッジ（High等）、タイプバッジ（Support, Improvement等）、コメント数・添付ファイル数のカウンター、進捗バーを表示する。情報密度が高く、タスクの状態を一目で把握可能。

主なユースケース: バグトラッカー、イシュー管理システム、スプリントボード、サポートチケット管理、開発タスクの優先度・進捗の可視化

## ImagesKanbanBoardsCard

説明: ECサイトの注文管理向けカンバンカード。注文番号と購入アイテム名を表示し、商品のサムネイル画像をグリッド形式で表示。残りアイテム数は「+15」形式で表示。リンクとして機能し詳細ページへ遷移可能。

主なユースケース: 注文ステータス管理ボード、出荷準備・発送状況の追跡、商品ピッキング管理、返品・交換処理の進捗管理

## ListDescriptionWithIconsKanbanBoardsCard

説明: プロジェクト管理向けのカンバンカード。タイトルとアイコン付きの詳細リスト（合計時間、日数、プロジェクトコスト、担当者）を縦に並べて表示。担当者は重なり合うアバターグループで表示し、残り人数は「+4」形式で表示。コメント数・添付ファイル数のカウンターも含む。

主なユースケース: プロジェクト進捗管理、クライアント案件のカンバン管理、チームタスクボード、工数・コスト管理を含むプロジェクト追跡
