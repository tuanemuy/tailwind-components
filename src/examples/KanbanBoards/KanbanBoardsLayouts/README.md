# Kanban Boards Layouts

Kanban Boards Layouts provide an organized, visual way to manage tasks and projects, enhancing workflow and productivity.

## KanbanCRMTasksLayout

説明: CRM（顧客関係管理）向けのフルページカンバンボードレイアウト。複数のステータスカラム（Prospecting等）を横並びに配置し、水平スクロールで移動可能。各カラムはタイトル・件数表示・カラー設定ドロップダウンを備え、カード追加機能も含む。ダークモード対応。

主なユースケース: 営業パイプラインの可視化、リード管理、商談ステージ管理、顧客ステータスの追跡

## KanbanDashboardProjectsLayout

説明: ダッシュボード内に組み込むプロジェクト管理用カンバンボード。ステータスラベルをドロップダウンで動的に変更可能で、カラー選択機能（複数色から選択）を備える。コンパクトな設計でダッシュボードの一部として配置しやすい。

主なユースケース: プロジェクト管理ダッシュボード、タスクステータスの可視化、チーム内ワークフロー管理、アジャイルボード

## FullHeightKanbanE-CommercePurchaseOrdersLayout

説明: Eコマースの購入注文管理向けフルハイトカンバンボード。画面の縦幅いっぱいに表示され、各カラムは個別に縦スクロール可能。「Arrived」「Ready for pickup」等の注文ステータスカラムを持ち、件数表示と新規タスク追加ボタンを備える。

主なユースケース: 注文ステータス管理（到着待ち・ピックアップ準備完了・配送中等）、倉庫管理ボード、フルフィルメント進捗管理、大量注文の視覚的追跡

## KanbanWorkspaceProjectsLayout

説明: チーム・ワークスペース向けのプロジェクト管理カンバンボード。ステータスカラムごとにタイトル編集・カラー設定のドロップダウンを備え、カードの追加・管理が可能。水平スクロールで複数カラムを表示。

主なユースケース: チームプロジェクト管理、ワークスペース内のタスク整理、スプリント管理、部門横断プロジェクトの進捗追跡

## KanbanE-CommercePurchaseOrdersLayout

説明: Eコマースの購入注文管理向けカンバンボード。FullHeightバージョンと同様の機能を持つが、ページ内の一部として配置する通常のスクロール方式。各カラムは背景付きのカードコンテナで、注文ステータス（Arrived, Ready for pickup等）ごとにグループ化。

主なユースケース: 注文管理ページ内のステータスボード、ピックアップ・配送管理、ECサイト管理画面の注文フロー可視化
